# 计算机入门学习

从零开始，真正理解你的电脑。

一本面向零基础读者的开源计算机入门书，使用 [mdBook](https://github.com/rust-lang/mdBook) 构建。

## 在线阅读

[GitHub Pages 链接](https://newwenu.github.io/learn-from-computer/)

## 本地构建

```bash
mdbook serve
```

然后在浏览器打开 `http://localhost:3000`。
