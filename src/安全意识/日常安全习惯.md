# 日常安全习惯：保护电脑和账号的好做法

> 阅读前提：你已了解常见的网络骗术（参见《识别常见骗术》）。本文不讲紧急应对，而是讲日常使用中应该养成的安全习惯——防患于未然。

---

## 一、账号密码安全

**同一个密码用遍所有网站，一个泄露就等于全部泄露**

### 为什么密码这么重要

你用的每一个网站都有可能被攻击。如果某个网站的数据库被泄露（这比你想的更常见），你的账号和密码就会暴露。如果你在其他网站也用了同一套密码，别人就可以拿它去尝试登录你的邮箱、社交账号、支付平台。

### 什么才算好密码

| 不好的做法 | 好的做法 |
|-----------|---------|
| 123456、password 等简单密码 | 密码长度至少 8 位 |
| 用生日、手机号当密码 | 不要包含容易被猜到的个人信息 |
| 一个密码用遍所有网站 | 每个重要平台用不同的密码 |
| 纯数字或纯字母 | 混合使用大小写字母、数字和符号 |

### 实际可行的做法

对大多数人来说，要求记住几十个不同密码是不现实的。以下是三个按重要性排序的做法，先从第一条开始：

**1. 核心账号用不同密码**

至少做到以下三类用不同密码：邮箱、微信/QQ、支付平台（支付宝/银行卡）。即使其他小网站的密码泄露了，这三个最重要的账号仍然是安全的。

**2. 用密码管理器来帮忙**

密码管理器是一个专门帮你生成和保存密码的程序。你只需要记住一个"主密码"，其他密码交给它管理。日常使用中，浏览器会提示你保存密码，手机系统也有内置密码管理功能。搜索"密码管理器推荐"可以了解更多选择。

**3. 开启双重验证**

微信、支付宝、苹果账号、Google 账号等都支持双重验证。开启后，即使别人知道了你的密码，没有你的手机也登不上。在账号的安全设置里找"双重验证"或"两步验证"即可开启。

> 自行探索：打开微信 → 我 → 设置 → 账号与安全 → 密码与安全，看看有没有可以开启的安全设置。其他账号也类似，在"安全设置"里找一找。

## 二、从官方渠道获取软件

**搜索引擎的第一个结果，不一定是官网**

### 为什么不能随便下载

你在搜索引擎搜"微信"，排在第一位的可能是广告，而不是真正的微信官网。这些广告位经常被第三方下载站购买，点击后下载的安装包会捆绑多余软件，甚至在极少情况下可能包含恶意程序。

### 怎么找到真正的官网

| 软件类型 | 官网特征 |
|---------|---------|
| 国内大厂产品 | 域名通常是 `产品名.公司名.com`，如微信是 `weixin.qq.com`，百度网盘是 `pan.baidu.com` |
| 国外常用软件 | 域名通常是 `软件名.com` 或 `软件名.org`，如 `7-zip.org`、`vlc.org` |
| 不确定时 | 搜索"软件名 + 官网"，看多数结果指向哪个地址 |

### 最安全的选择

- **优先使用应用商店**：Microsoft Store 和 Mac App Store 里的软件经过审核，不会捆绑多余软件，也方便更新和卸载
- **官网下载**：确认域名正确后再下载
- **安装时多留个心眼**：每一步都看看有没有额外勾选的选项，不需要的取消勾选

## 三、系统更新与日常习惯

**更新提醒不是来烦你的，是来保护你的**

### 为什么更新很重要

操作系统和软件的更新不只是"加功能"。大部分更新包含安全修复——修补已知的漏洞。不更新的电脑就像一个不锁门的房子：病毒和恶意软件可以利用这些已知漏洞入侵你的系统。

### 几个最重要的日常习惯

**1. 保持系统更新**

Windows 每月推送安全更新，修复新发现的漏洞。更新确实会打扰你——正工作时突然提示重启，正打游戏时开始下载，烦是正常的。

但安全更新是目前防范病毒最有效的手段之一。可以在"设置 → Windows 更新 → 高级选项 → 活跃时段"中设置你通常用电脑的时间段，系统就不会在那段时间内自动重启。看到更新提示时，自己选一个方便的时间再重启就好。更新前记得保存打开的文档。

**2. 离开时锁屏**

按 **Win+L** 一键锁屏。在办公室、学校或合租环境尤其重要——防止别人在你不在时使用你的电脑。

**3. 不插来历不明的 U 盘**

有些恶意软件通过 U 盘传播（比如在打印店、公共电脑上插过的 U 盘）。不要随意打开别人给的 U 盘，也不要在陌生电脑上插你的 U 盘。

**4. 卸载不需要的软件**

电脑里留着一堆用不上的软件，除了占地方，有些还会在后台自动运行、弹窗或拖慢开机速度。在"设置 → 应用 → 已安装的应用"中检查，不需要的直接卸载。

**5. 重要文件备份**

硬件会坏，也可能中病毒导致文件打不开。重要的文件至少保留两份：一份在电脑上，一份在 U 盘或云盘上。不需要频繁备份，但重要的作业、照片、文档值得多存一份。

---

**总结：一个好习惯胜过十个补救措施。**

1. **重要的账号用不同的密码**
2. **只从官网或应用商店下载软件**
3. **让系统保持更新**
