# 防患于未然：日常怎么保护电脑和账号

> 阅读前提：无。本文讲日常使用中应该养成的安全习惯。建议配合《看穿那些套路》一起阅读，但不必强制按顺序。

---

## 一、账号密码安全

**同一个密码用遍所有网站，一个泄露就等于全部泄露**

### 密码基本原则

- 不用 123456、password 等简单密码
- 不用纯生日、纯手机号当密码
- 长度至少 12 位
- 建议用一句与你无关的话当密码，或用密码管理器生成随机密码
- 一个密码不要用在所有网站
- 核心账号（邮箱、微信/QQ、支付平台）用不同密码

### 记不住怎么办

不用追求每个密码都复杂到记不住。先从最重要的一条做起：**邮箱、微信/QQ、支付平台这三类账号用不同的密码**。即使其他小网站泄露了，最重要的账号还是安全的。

如果你愿意更进一步：

- **让浏览器帮你记**：浏览器自带的密码保存功能很方便，也能生成复杂密码。在登录网站时，浏览器通常会提示"是否保存密码"，选择保存即可。注意：在公共电脑（如图书馆、网吧）上不要保存密码
- **开启双重验证**：微信、支付宝、苹果账号等都支持，开启后登录时除了密码还需要手机验证码，安全性会高很多

> 打开微信 → 我 → 设置 → 账号与安全，看看能开启哪些安全设置。

## 二、从官方渠道获取软件

**搜索引擎的第一个结果，不一定是官网**

### 为什么不能随便下载

你在搜索引擎搜"微信"，排在第一位的可能是广告，而不是真正的微信官网。这些广告位经常被第三方下载站购买，点击后下载的安装包会捆绑多余软件，甚至在极少情况下可能包含恶意程序。

### 怎么找到真正的官网

| 软件类型     | 官网特征                                                                             |
| ------------ | ------------------------------------------------------------------------------------ |
| 国内大厂产品 | 域名通常是 `产品名.公司名.com`，如微信是 `weixin.qq.com`，百度网盘是 `pan.baidu.com` |
| 国外常用软件 | 域名通常是 `软件名.com` 或 `软件名.org`，如 `7-zip.org`、`videolan.org`                   |
| 不确定时     | 搜索"软件名 + 官网"，看多数结果指向哪个地址                                          |

### 安装时多留个心眼

- **优先使用应用商店**：Microsoft Store 里的软件经过审核，基本不会捆绑多余软件，也方便更新和卸载
- **官网下载**：确认域名正确后再下载
- **每一步都看看**：安装向导里有没有额外勾选的选项，不需要的软件取消勾选

## 三、保持系统更新

**更新提醒不是来烦你的，是来保护你的**

### 为什么更新很重要

更新不只是"加功能"，大部分更新是安全修复——修补已知漏洞。不更新的电脑就像不锁门，病毒可以利用这些漏洞入侵。

### 怎么设置

Windows 定期推送安全更新（通常每月一次）。更新确实会打扰你，但它是防范病毒最有效的手段之一。

在"设置 → Windows 更新 → 高级选项"中设置"使用时段"，避免在你用电脑时自动重启 `[查]`。看到更新提示时，选一个方便的时间重启就好。更新前记得保存打开的文档。

## 四、离开电脑时锁屏

**按 Win+L 一键锁屏**

在办公室、学校或合租环境尤其重要——防止别人在你不在时使用你的电脑。即使只是去倒杯水，养成锁屏习惯也不亏。

设置方法：在"设置"里搜索"登录选项"，可以设置唤醒时是否需要密码。如果想让电脑自动锁屏，在"设置"里搜索"屏幕超时"或"睡眠"，设置一段时间无操作后自动关闭屏幕并锁定。

---

## 五、U 盘使用注意

**核心原则：来历不明的 U 盘，不要插**

U 盘传播病毒最常见的方式：里面藏有恶意文件，双击运行后中招。Windows 自带的 Microsoft Defender 有实时保护功能，打开文件时会自动检查，但建议插入陌生 U 盘后还是手动扫描一遍更稳妥。

但除了文件病毒，U 盘本身也可能被动过手脚。所以核心原则是：

- **陌生人给的 U 盘、路边捡的 U 盘，不要插**
- 工作/学习场景中交换的 U 盘，插入后让 Windows 扫描一遍再用
- 如果 U 盘里有你不认识的奇怪文件（尤其是 `.exe` 结尾的），不要双击运行

打开"文件资源管理器"，右键点击 U 盘 →"使用 Microsoft Defender 扫描"，手动扫一遍更安心。

## 六、重要文件定期备份

**硬件会坏，文件可能丢**

重要的文件至少保留两份：一份在电脑上，一份在 U 盘或云盘上。不需要频繁备份，但重要的作业、照片、文档值得多存一份。

**简单备份做法：**

- 使用 OneDrive、百度网盘等云盘自动同步重要文件夹，这是最省心的方式
- 或者每隔一段时间把"文档""图片"文件夹复制到 U 盘或移动硬盘
- 核心原则：重要的文件至少存两个不同的地方（比如电脑+网盘），这样即使电脑坏了，文件还在

## 七、警惕第三方"系统优化"软件

**Windows 自带的工具已经足够好用**

很多第三方"优化软件"的常见套路是：先吓唬你说"发现 500 个垃圾文件""内存占用 90% 需要清理"，然后诱导你安装更多软件。它们往往自带弹窗广告、捆绑其他程序，有些还会误删系统文件。

如果你确实需要清理或优化，优先用系统自带工具：

**电脑提示空间不足？**
→ 用系统自动清理功能（存储感知）或手动运行磁盘清理

**开机太慢？**
→ 按 `Ctrl+Shift+Esc` 打开任务管理器，切换到"启动"选项卡，禁用不需要开机启动的程序

**电脑变慢、卡顿？**
→ 按 `Ctrl+Shift+Esc` 打开任务管理器，切换到"进程"选项卡，看哪个程序占用了大量 CPU 或内存

**网络有问题？**
→ 按 `Ctrl+Shift+Esc` 打开任务管理器，切换到"性能"选项卡查看网络状态 `[查]`

如果电脑确实卡顿，先用任务管理器排查原因，而不是寄希望于"一键优化"。

## 八、浏览器与隐私设置

**上网时留一点心眼**

### 几个值得检查的浏览器设置

1. **密码保存**

   在自己常用的私人电脑上，让浏览器保存密码是方便的，也足够安全。如果你在公共电脑（如图书馆、网吧）上使用，记得不要保存密码，使用完后清除浏览数据。

2. **定期清理浏览数据**

   浏览器会记录你访问过的网站、填过的表单、保存的 Cookie（网站留在你电脑上的一些小数据，比如记住登录状态用的）。在"设置 → 隐私和安全 → 清除浏览数据"中，可以定期清理。不需要每天清，但几个月清理一次是合理的。

3. **注意网站权限**

   有些网站会请求"发送通知""获取位置""使用摄像头/麦克风"等权限。不需要的权限可以拒绝。你可以在浏览器设置中查看和管理已授予的权限，把不需要的关掉。

## 九、安装软件时的弹窗（UAC）

**看到"你要允许此应用对你的设备进行更改吗？"时，先停一秒**

安装软件、修改系统设置时，Windows 会弹出一个背景变暗的窗口，问你"是"还是"否"。这个弹窗叫 **UAC（用户账户控制）**，是 Windows 的安全防线——它是一个授权机制。

### 什么时候点"是"

- 你刚刚主动点了某个软件的"安装"或"运行"
- 你正在修改系统设置（如添加用户、修改网络配置）

### 什么时候点"否"

- 你没做任何操作，弹窗突然自己跳出来
- 弹窗出现时你感到意外——明明没点安装，却冒出权限请求
- 你只是在浏览网页，没点任何下载或安装按钮

**核心原则**：看到弹窗先想一下，**你刚才是不是在安装软件或者修改系统设置？**

- **是**——比如刚点了安装包，或者正在调整系统设置——那弹窗是正常流程，点是就行
- **不是**——你只是在浏览网页、打字、看视频，什么都没做，弹窗就跳出来了——这比较可疑，建议点否

你只需要结合自己正在做的事来判断。点错了也不必担心，点"否"只是这次不给权限，程序不会坏，系统也不会坏。如果不确定，先点"否"，再去搜索一下弹窗上显示的程序名是什么。


想了解更多 UAC 背后的原理，可以阅读[《权限与账户进阶》](../扩展知识/权限与账户进阶.md)。

## 总结：八个好习惯

1. **重要的账号用不同的密码**
2. **只从官网或应用商店下载软件**
3. **让系统保持更新**
4. **离开电脑时锁屏**
5. **不插来历不明的 U 盘**
6. **重要文件定期备份**
7. **浏览器密码和权限按需管理**
8. **UAC 弹窗时先确认是不是正常的操作**

安全习惯刚开始会觉得麻烦，但养成之后几乎不增加操作负担——就像出门锁门一样自然。这些习惯的目的不是让你变成安全专家，而是让常见的骗术和攻击对你无效。

> 读到这里，你已经具备了保护自己和电脑的基本意识——这比大多数人起步时都要好了。
