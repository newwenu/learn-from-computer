# 识别常见骗术：那些在屏幕上骗你的东西

> 本文帮你识别日常使用中最常遇到的骗术，不讲安全理论，只讲怎么判断真假。

---

## 一、虚假弹窗与病毒警报

**你电脑里没有病毒，那个弹窗才是问题**

上网时突然弹出一个窗口，背景变暗，上面显示"发现 5 个安全威胁！立即清理！"，还有一个进度条在"扫描"你的电脑。这是最常见的网络骗术之一，它利用的是恐慌心理——让你觉得电脑出了严重问题，急于解决而失去判断力。

### 怎么判断是假的

这些弹窗有一个共同点：它们出现在**浏览器里**。真正的系统警告——比如 Windows 安全中心的提示——不会以浏览器弹窗的形式出现。你在看网页时突然跳出来的"病毒警报"，不用怀疑，就是假的。

如果弹窗铺满整个屏幕、看不到地址栏，按 **Esc** 或 **F11** 可以退出全屏模式——地址栏和标签页会重新出现，你就知道自己还在浏览器里，没有被"入侵"。

### 常见变种

- 浏览器弹出"您的 Chrome 浏览器已感染 3 个病毒"——实际上是网页广告，Chrome 自己不会在浏览器里弹这种窗
- 全屏播放视频时突然弹出"内存不足！立即清理"——视频网站常见广告位
- 假装是 Windows 系统界面，写着"Windows Defender 发现威胁"——假的，Windows 安全提示不会出现在浏览器窗口里

### 安全做法

1. **不要点击弹窗上的任何按钮**——包括"取消""关闭""×"。有些假关闭按钮点了反而会下载东西
2. 按 **Ctrl+W** 关闭当前标签页，或按 **Alt+F4** 关闭整个浏览器窗口——弹窗只是网页，关掉窗口就没了

一个简单的判断标准：**如果你没有主动打开某个程序，但突然弹出一个窗口让你做某事，先假设它是假的。**

## 二、钓鱼链接与虚假消息

**点进去之前，它看起来完全正常**

你收到一条短信："【XX银行】您的账户出现异常登录，请立即验证：http://xxxx.com"，或者"您的快递已退回，点击链接查询详情"。点进去是一个看起来跟官网一模一样的页面，让你输入账号密码。

### 怎么判断是假的

正规的银行和支付平台**不会通过短信里的链接**让你登录、输入密码或验证码。凡是发来链接让你点的，先当成假的。号码显示为"10086""95555"也不代表真的——号码可以伪造。

### 常见变种

- "您的 QQ 账号存在违规行为，点击链接验证"——假冒腾讯安全中心
- "您的微信支付已冻结，点击恢复"——假冒微信团队
- "ETC 已过期，点击链接续期"——针对车主的诈骗短信
- "您的订单已退款，点击链接填写信息"——假冒电商平台
- 收到同事或好友发来的"这个文件你帮我看一下"的链接——账号被盗后发送的钓鱼链接

### 安全做法

1. **不要点开短信或聊天软件里发来的链接**，尤其是在它让你输入密码或验证码的情况下
2. 需要登录某个网站时，**自己打开浏览器手动输入网址**，或者用收藏夹里的地址
3. 检查网址：看域名是否正确。`taobao.com` 是真正的淘宝，`taobao-shop.com` 不是
4. 如果不确定，**通过官方渠道核实**——打客服电话，从官网找号码，而不是短信里给的号码

## 三、虚假下载与软件陷阱

**那个绿色的"下载"按钮，不一定是下载按钮**

你想下载一个软件，在搜索引擎搜到后进入一个下载站。页面上有好几个"下载"按钮，颜色不同、位置不同。下载后运行安装程序，发现桌面多了好几个不认识的软件。

### 怎么判断是假的

正规软件的官网下载入口简单明了，不会让你在多个"下载"按钮里猜哪个是真的。页面布满颜色鲜艳的"下载"按钮、写着"高速下载""安全下载"——这些特征本身就说明这个网站不可靠。

### 骗术原理

下载站靠广告和捆绑软件赚钱。它们把真正的下载链接藏得很深，把广告按钮做得很大很显眼。你下载的不是你要的软件，而是一个"下载器"——它会在安装过程中再给你装上浏览器主页、压缩软件、杀毒软件等捆绑物。

### 安全做法

1. **只从官方渠道下载**：在搜索结果里看域名。微信的官网是 `weixin.qq.com`，不是 `weixindownload.com`
2. **不确定官网地址时**：搜索"软件名 + 官网"，多数结果指向的地址通常是正确的
3. **安装时看一眼**：每一步都注意有没有额外勾选的选项，不需要的软件取消勾选
4. **优先使用应用商店**：Microsoft Store、Mac App Store 里的软件经过审核，比第三方下载站安全得多

---

**总结：不点不请自来的弹窗、链接和按钮。让人紧张的东西，先假设它是假的。**
