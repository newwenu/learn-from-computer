# C 盘清理

## 基本概念

- **虚拟内存（pagefile.sys）**：硬盘上划出的空间用作虚拟内存，删除会导致系统无虚拟内存
- **休眠（hiberfil.sys）**：休眠文件，大小约等于内存容量
- **存储感知**：Windows 内置自动清理功能

## 存储感知（自动清理）

1. 打开"设置>系统>存储"
2. 开启"存储感知"
3. 点击"配置存储感知或立即运行"
4. 选择清理选项（临时文件、回收站、下载文件夹等）
5. 点击"立即运行"

### 可配置项

- 运行存储感知频率：每天/每周/每月/磁盘空间不足时
- 删除临时文件：开/关
- 删除回收站文件：1/14/30/60 天
- 删除下载文件夹文件：开/关

## 磁盘清理（手动深度清理）

1. 在搜索框输入"磁盘清理"并打开
2. 选择要清理的驱动器（C 盘）
3. 勾选要清理的文件类型
4. 点击"确定">"删除文件"

### 清理系统文件

| 文件类型 | 说明 |
|----------|------|
| 临时文件 | 程序运行产生的临时数据 |
| 回收站 | 已删除但未清空的文件 |
| Windows 更新清理 | 旧版更新缓存 |
| 传递优化文件 | 更新分发缓存 |
| 缩略图 | 文件预览缓存 |
| 以前的 Windows 安装 | 旧版系统文件（删除后无法回退旧版本） |
| 下载文件夹 | 会清空整个下载文件夹 |

## 手动删除会导致异常的文件夹

- `Windows`（含 System32、WinSxS）
- `Program Files`、`Program Files (x86)`
- `AppData`

## 系统文件说明

| 文件 | 作用 | 大小 | 处理方式 |
|------|------|------|----------|
| pagefile.sys | 虚拟内存 | 几GB到几十GB | 可移到其他盘，删除会导致系统无虚拟内存 |
| hiberfil.sys | 休眠文件 | 约等于内存容量 | 关闭休眠：管理员命令行输入 `powercfg /hibernate off` |
| swapfile.sys | UWP 应用交换文件 | 几百MB到几GB | 不可删除 |
| WinSxS | 组件存储（系统组件的多个版本） | 数GB到十几GB | 不可手动删除，可用 `Dism /Online /Cleanup-Image /StartComponentCleanup` 清理 |

## AppData 文件夹

路径：`C:\Users\用户名\AppData`（默认隐藏）

| 子文件夹 | 内容 |
|----------|------|
| `AppData\Local` | 软件本地数据（缓存、日志等），部分可清理 |
| `AppData\Local\Temp` | 临时文件，可清理 |
| `AppData\Roaming` | 软件漫游数据（配置文件、个人设置等），随用户账号同步 |
| `AppData\LocalLow` | 低权限软件数据 |

打开方式：`Win+R`→输入 `%appdata%` 打开 Roaming 文件夹

## 临时文件位置

| 路径 | 内容 |
|------|------|
| `C:\Windows\Temp` | 系统临时文件 |
| `C:\Users\用户名\AppData\Local\Temp` | 用户临时文件 |
| `C:\Windows\SoftwareDistribution\Download` | Windows 更新下载缓存 |
