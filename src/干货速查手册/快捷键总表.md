# 快捷键总表

## 系统操作

| 快捷键 | 作用 |
|--------|------|
| Win+E | 打开文件资源管理器 |
| Win+D | 显示桌面（再按一次恢复） |
| Win+L | 锁屏 |
| Win+R | 打开运行对话框 |
| Win+I | 打开设置 |
| Win+Tab | 打开任务视图（切换窗口或虚拟桌面） |
| Alt+F4 | 关闭当前窗口 |
| Ctrl+Shift+Esc | 打开任务管理器 |
| Win+Print Screen | 全屏截图并自动保存到"图片>屏幕截图"文件夹 |
| Win+. / Win+; | 打开表情符号面板 |
| Win+V | 打开剪贴板历史 |

## 文件管理

| 快捷键 | 作用 |
|--------|------|
| Ctrl+C | 复制 |
| Ctrl+X | 剪切 |
| Ctrl+V | 粘贴 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+S | 保存 |
| Ctrl+A | 全选 |
| Ctrl+F | 查找 |
| F2 | 重命名 |
| F5 | 刷新 |
| Delete | 删除到回收站（可恢复） |
| Shift+Delete | 永久删除（不可恢复） |

## 文件资源管理器

| 快捷键 | 作用 |
|--------|------|
| Ctrl+Shift+N | 新建文件夹 |
| Alt+↑ | 返回上一级文件夹 |
| Alt+← / Alt+→ | 后退 / 前进 |
| Ctrl+T | 新建标签页（Win11） |
| Ctrl+W | 关闭当前标签页（Win11） |
| F4 | 地址栏变为可输入状态 |

## 窗口管理

| 快捷键 | 作用 |
|--------|------|
| Alt+Tab | 切换窗口（按住 Alt 多按 Tab 在窗口间切换） |
| Alt+Shift+Tab | 反方向切换窗口 |
| Win+← | 窗口左半屏 |
| Win+→ | 窗口右半屏 |
| Win+↑ | 窗口最大化 |
| Win+↓ | 窗口最小化 |
| Win+Shift+←/→ | 移动窗口到其他显示器 |
| Win+Z | 打开分屏布局（Win11） |

## 虚拟桌面

| 快捷键 | 作用 |
|--------|------|
| Win+Ctrl+D | 创建新虚拟桌面 |
| Win+Ctrl+F4 | 关闭当前虚拟桌面（窗口移到上一个桌面） |
| Win+Ctrl+← | 切换到上一个虚拟桌面 |
| Win+Ctrl+→ | 切换到下一个虚拟桌面 |

## 浏览器

| 快捷键 | 作用 |
|--------|------|
| Ctrl+T | 新建标签页 |
| Ctrl+W | 关闭当前标签页 |
| Ctrl+Shift+T | 恢复关闭的标签页 |
| Ctrl+L | 选中地址栏 |
| Ctrl+Tab | 切换标签页 |
| Ctrl+1~8 | 切换到第 1-8 个标签页 |
| Ctrl+9 | 切换到最后一个标签页 |

## 截图

| 快捷键 | 作用 |
|--------|------|
| Win+Shift+S | 打开截图工具栏（矩形/窗口/全屏） |
| Alt+Print Screen | 截取当前窗口（保存到剪贴板） |
| Print Screen | Win11 打开截图工具栏 |

## 显示

| 快捷键 | 作用 |
|--------|------|
| Win+P | 打开投影设置（切换显示器模式） |
| Win+K | 打开投屏面板（无线投屏） |

## 输入法

| 快捷键 | 作用 |
|--------|------|
| Win+空格 | 切换输入法 |
| Shift | 在当前输入法里切换中英文 |

## 其他

| 快捷键 | 作用 |
|--------|------|
| Ctrl+滚轮 | 缩放（网页或文件内容） |
| Win+H | 语音输入（听写） |
| Win+G | 打开 Xbox Game Bar |
| Win+Alt+R | Game Bar 开始/停止录制 |
