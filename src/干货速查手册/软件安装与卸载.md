# 软件安装与卸载

## 基本概念

- **安装程序**：安装软件到电脑的程序，常见格式 `.exe`、`.msi`
- **UAC（用户账户控制）**：Windows 安全机制，程序需要系统权限时弹出确认窗口
- **32 位 vs 64 位**：64 位系统可运行 32 位软件，反之不行
- **注册表**：Windows 存储系统和软件配置的数据库，卸载时应清除对应条目

### .exe 与 .msi 区别

- **.exe**：安装流程由软件自定义
- **.msi**：Windows 标准安装包，支持静默安装

## 安装软件

### 安装步骤

1. 双击安装程序
2. 按 UAC 弹窗提示点击"是"
3. 选择安装路径（默认 `C:\Program Files` 或 `C:\Program Files (x86)`）
4. 点击"安装"，等待完成

### 安装路径

- 默认路径：`C:\Program Files`（64 位程序）或 `C:\Program Files (x86)`（32 位程序）
- 自定义路径：安装时将路径中的 `C:` 改为其他盘符
- 部分软件默认安装到 `C:\Users\用户名\AppData`（免管理员权限安装位置）

### 32 位还是 64 位

- 右键"开始"按钮→"系统"→看"系统类型"
- 64 位操作系统 → 选 64 位（x64）
- 32 位操作系统 → 只能选 32 位（x86）
- 64 位操作系统可运行 32 位软件，32 位操作系统不能运行 64 位软件

### 便携版软件

部分软件提供"便携版"（Portable），无需安装，解压后运行 `.exe` 文件：
- 不写入注册表，不在"设置→应用"中显示
- 卸载时删除文件夹
- 配置文件通常保存在软件所在文件夹内

## 卸载软件

1. 打开"设置>应用>已安装的应用"
2. 找到要卸载的软件，点击"三个点>卸载"
3. 按提示完成卸载

> **注意**：直接删除软件文件夹会残留配置文件和系统注册表记录；删除桌面快捷方式 ≠ 卸载软件
