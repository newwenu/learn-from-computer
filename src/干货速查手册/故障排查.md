# 故障排查

## 基本概念

- **UAC（用户账户控制）**：Windows 安全机制，程序需要系统权限时弹出确认窗口
- **蓝屏（BSOD）**：Windows 遇到严重错误时显示蓝色错误页面，通常显示错误代码，重启后可能恢复正常
- **驱动程序**：让操作系统识别和控制硬件的软件

## 程序卡死

1. 按 `Ctrl+Shift+Esc` 打开任务管理器
2. 找到卡死的程序，右键"结束任务"
3. 重新打开程序

> **注意**：结束系统进程可能导致系统不稳定

## 黑屏风扇转

1. 检查显示器电源灯是否亮着
2. 检查视频线（HDMI/VGA/DP）是否插紧
3. 按显示器菜单键，手动切换输入源（HDMI1/HDMI2）
4. 重启电脑

## Wi-Fi 连着打不开网页

1. 判断是所有设备都这样还是只有当前设备
2. 检查微信或 QQ 能否登录
   - 能登录微信但打不开网页 → DNS 问题，修改 DNS（参考"网络.md"）
   - 微信也无法登录 → 网络本身有问题，重启路由器
3. 重启路由器和电脑
4. 检查网线是否插紧

## 软件打不开

1. 右键程序>"属性>兼容性>以兼容模式运行"
2. 右键程序>"以管理员身份运行"
3. 检查软件是否支持当前系统类型（32 位软件可在 64 位系统运行，反之不行）
4. 搜索错误提示
5. 卸载重装

## 电脑突然变慢

1. 按 `Ctrl+Shift+Esc` 打开任务管理器
2. 查看"进程"选项卡，点击 CPU 列头排序
3. 找到占用率高的程序
4. 如果"磁盘"列显示 100% 且持续不降 → 重启

## 蓝屏

1. 重启电脑
2. 记录蓝屏代码（如 0x0000007B）或扫描二维码，搜索解决方案
3. 卸载最近安装的软件或驱动后测试

## USB 设备没反应

1. 换个 USB 接口（台式机试试机箱背面接口）
2. 确认插紧（Type-A 接口有方向，插不进时翻转重试）
3. 换一台电脑试试（排除设备本身问题）

## 设备管理器排查

打开方式：`Win+X`→设备管理器

- **黄色感叹号**：驱动有问题 → 右键设备→"更新驱动程序"→"自动搜索驱动程序"
- **没有感叹号但不工作**：可能是隐私权限问题（摄像头/麦克风 → 设置→隐私和安全性）
- **设备没出现在列表里**：系统没检测到，检查物理连接

## 安全模式

**进入方法**：
1. 按住 `Shift` 键，点击"开始→电源→重启"
2. 选择"疑难解答→高级选项→启动设置→重启"
3. 重启后按 `4` 进入安全模式，按 `5` 进入带网络的安全模式

## 系统还原

**打开方式**：`Win+R`→`sysdm.cpl`→"系统保护"选项卡→"系统还原"

**创建还原点**：同一窗口→点击"创建"

## 事件查看器

打开方式：`Win+R`→`eventvwr.msc`

| 日志位置 | 内容 |
|----------|------|
| Windows 日志→系统 | 硬件错误、驱动错误、系统崩溃 |
| Windows 日志→应用程序 | 软件崩溃、应用程序错误 |

查看方法：点击对应日志→按"级别"列筛选"错误"和"关键"→双击条目查看详情
