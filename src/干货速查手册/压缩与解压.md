# 压缩与解压

## 基本概念

- **压缩**：压缩后需解压才能使用原文件
- **压缩率**：压缩后体积与原体积的比值，压缩率越高文件越小，但压缩/解压耗时越长

## 格式区别

| 格式 | 压缩率 | 速度 | 支持系统 |
|------|--------|------|----------|
| .zip | 中等 | 快 | Windows/macOS 内置基本支持，Linux 通常需安装 unzip |
| .7z | 高 | 中等 | Win11 24H2+ 内置基本解压支持，其他需 7-Zip |
| .rar | 高 | 中等 | WinRAR 专有格式，需 WinRAR 或 7-Zip 解压 |
| .tar.gz | 中等 | 快 | Linux/macOS 常见，Windows 需 7-Zip |
| .tar.xz | 高 | 慢 | Linux 软件包常用，Windows 需 7-Zip |

## 创建压缩文件

右键文件或文件夹→压缩到 ZIP 文件，或使用压缩工具选择格式和压缩级别。

## 7-Zip 右键选项

- **提取到当前文件夹**：解压到压缩包所在位置
- **提取到"文件夹名"**：创建同名文件夹并解压到其中
- **提取文件**：选择解压位置和选项

## 创建压缩包时的选项

右键文件→7-Zip→"添加到压缩包"：
- **压缩格式**：.zip 或 .7z
- **压缩等级**：存储/最快/快速/正常/最大/极限
- **加密/密码**：在右侧"加密"区域输入密码

## 分卷压缩包

文件名带编号的压缩包（如 `资料.001`、`资料.002`）是分卷压缩。解压方法：把所有分卷放在同一文件夹，右键第一个分卷（编号最小的）选择解压。

**常见问题**：只下载了部分分卷会提示"缺少分卷"。

## 压缩包损坏

- 重新下载
- 让对方重新发送
- 部分压缩软件有修复功能
