# 截图与录屏

## 基本概念

- **剪贴板**：系统临时存储复制/截图内容的区域，`Win+V` 可查看剪贴板历史
- **截图工具栏**：`Win+Shift+S` 打开的系统截图界面，支持矩形、任意形状、窗口、全屏四种模式

## 截图

| 快捷键 | 作用 | 保存位置 |
|--------|------|----------|
| `Win+Shift+S` | 打开截图工具栏（矩形/任意形状/窗口/全屏） | 剪贴板+自动保存到"图片>屏幕截图" |
| `Win+Print Screen` | 全屏截图 | 自动保存到"图片>屏幕截图"文件夹 |
| `Alt+Print Screen` | 截取当前窗口 | 剪贴板，需粘贴保存 |
| `Print Screen` | Win11：打开截图工具栏；Win10：截取全屏到剪贴板 | 见左 |

**截图工具栏模式**：矩形截图、任意形状截图、窗口截图、全屏截图

**延迟截图**：截图工具中点击"延迟"按钮，设置 3 秒或 5 秒后再截图，用于截取右键菜单等点击后会消失的画面

**截图工具（独立应用）**：Win11 内置"截图工具"应用（Snipping Tool），搜索"截图工具"可打开，支持矩形/任意形状/窗口/全屏截图和录屏，可设置延迟截图。

## 录屏

### 截图工具栏录屏

1. 按 `Win+Shift+S` 打开截图工具栏
2. 点击"录屏"按钮进入录制模式，或按 `Win+Shift+R`
3. 屏幕变暗，用鼠标拖选要录制的区域
4. 点击顶部工具栏的录制按钮开始
5. 录制完成后点击停止按钮
6. 视频自动保存到"视频>屏幕录制"文件夹

**录制选项**：
- 麦克风：录制前在工具栏上点击麦克风图标开启/关闭
- 系统声音：默认录制

### Xbox Game Bar

1. 按 `Win+G` 打开 Xbox Game Bar
2. 点击"录制"按钮开始录制，或按 `Win+Alt+R` 开始
3. 录制完成后按 `Win+Alt+R` 停止
4. 视频自动保存到"视频>捕获"文件夹（部分版本叫"摄像"）

**录制选项**：
- 麦克风：默认不录制，在录制小部件里点麦克风图标开启
- 系统声音：默认录制
- 局限：部分场景不支持录制桌面，需先打开一个应用窗口
