# 操作系统对比

## Windows

| 项目 | 内容 |
|------|------|
| 用户文件夹 | `C:\Users\用户名` |
| 路径分隔符 | `\`（反斜杠） |
| 环境变量分隔符 | `;`（分号） |
| 文件名大小写 | 不区分 |
| 文件系统 | NTFS、exFAT、FAT32 |
| 安装包格式 | `.exe`、`.msi` |
| 默认安装路径 | `C:\Program Files`（64 位）、`C:\Program Files (x86)`（32 位） |
| 包管理器 | winget、choco |
| 卸载方式 | 设置→应用→卸载 |
| 默认 Shell | PowerShell、cmd |
| 管理员权限 | 右键→以管理员身份运行 |
| 复制/粘贴/撤销 | `Ctrl+C/V/Z` |
| 截图 | `Win+Shift+S` |
| 锁屏 | `Win+L` |
| 任务管理器 | `Ctrl+Shift+Esc` |
| 窗口控制按钮 | 右上角（─ □ ✕） |

## macOS

| 项目 | 内容 |
|------|------|
| 用户文件夹 | `/Users/用户名` |
| 路径分隔符 | `/`（正斜杠） |
| 环境变量分隔符 | `:`（冒号） |
| 文件名大小写 | 不区分 |
| 文件系统 | APFS、Mac OS Extended |
| 安装包格式 | `.dmg` |
| 默认安装路径 | `/Applications` |
| 包管理器 | Homebrew |
| 卸载方式 | 拖入废纸篓或使用卸载工具 |
| 默认 Shell | zsh |
| 管理员权限 | `sudo` |
| 复制/粘贴/撤销 | `Cmd+C/V/Z` |
| 截图 | `Cmd+Shift+3`（全屏）/ `Cmd+Shift+4`（选区） |
| 锁屏 | `Ctrl+Cmd+Q` |
| 强制退出 | `Cmd+Option+Esc` |
| 窗口控制按钮 | 左上角（红 黄 绿） |

## Linux

| 项目 | 内容 |
|------|------|
| 用户文件夹 | `/home/用户名` |
| 路径分隔符 | `/`（正斜杠） |
| 环境变量分隔符 | `:`（冒号） |
| 文件名大小写 | 区分 |
| 文件系统 | ext4 |
| 安装包格式 | `.deb`（Debian/Ubuntu）、`.rpm`（Fedora/RedHat）、`.flatpak` |
| 默认安装路径 | `/usr/bin`、`/opt` |
| 包管理器 | apt（Debian/Ubuntu）、yum/dnf（Fedora）、pacman（Arch） |
| 卸载方式 | 包管理器卸载（如 `apt remove`） |
| 默认 Shell | bash |
| 管理员权限 | `sudo` |
| 复制/粘贴/撤销 | `Ctrl+C/V/Z` |
| 截图 | `PrtSc`（全屏）/ `Shift+PrtSc`（选区） |
| 锁屏 | `Ctrl+Alt+L` |
| 强制退出 | `xkill` 或 `kill` 命令 |
| 桌面环境 | GNOME、KDE、XFCE 等（不同发行版默认不同） |

## 命令对照

| 操作 | Windows | macOS / Linux |
|------|---------|---------------|
| 列出文件 | `dir` | `ls` |
| 复制文件 | `copy` | `cp` |
| 移动/重命名 | `move`、`ren` | `mv` |
| 删除文件 | `del` | `rm` |
| 查看文件内容 | `type` | `cat` |
| 搜索字符串 | `findstr` | `grep` |
| 创建文件夹 | `mkdir` | `mkdir` |
| 切换目录 | `cd` | `cd` |
| 显示当前路径 | `cd`（无参数） | `pwd` |
| 清屏 | `cls` | `clear` |
