# 文件与磁盘

## 基本概念

- **路径**：文件在电脑中的位置描述，如 `C:\Users\用户名\Documents\报告.docx`
- **扩展名**：文件名中最后一个 `.` 后面的部分，标识文件类型，如 `.txt` 是文本文件、`.exe` 是可执行文件
- **文件系统**：操作系统管理文件存储的方式，不同文件系统对单个文件大小、权限等有不同限制
- **分区表**：记录磁盘如何划分为多个分区的数据结构，决定了磁盘最大容量、分区数量和启动方式

## 路径格式

| 系统 | 用户文件夹路径 | 中文显示名对照 |
|------|--------------|--------------|
| Windows | `C:\Users\用户名` | `Users`→用户、`Documents`→文档、`Desktop`→桌面、`Downloads`→下载、`Pictures`→图片 |
| macOS | `/Users/用户名` | `Documents`→文稿、`Desktop`→桌面、`Downloads`→下载、`Pictures`→图片 |
| Linux | `/home/用户名` | — |

## 常见扩展名含义

| 扩展名 | 类型 |
|--------|------|
| .txt | 文本文件 |
| .docx | Word 文档 |
| .xlsx | Excel 表格 |
| .pptx | PowerPoint 演示文稿 |
| .pdf | PDF 文件 |
| .jpg/.png | 图片文件 |
| .mp3/.mp4 | 音视频文件 |
| .exe | Windows 可执行文件 |
| .zip/.7z/.rar | 压缩文件 |
| .dmg | macOS 安装包 |
| .msi | Windows 安装包 |

## C 盘根目录文件夹

| 文件夹 | 内容 | 注意 |
|--------|------|------|
| `Program Files` | 64 位程序 | 手动删除会导致程序无法运行，需通过"设置→应用"卸载 |
| `Program Files (x86)` | 32 位程序 | 同上 |
| `Users` | 用户个人文件（文档、下载、桌面等） | — |
| `Windows` | 操作系统核心文件 | 手动修改或删除会导致系统异常 |

## 文件系统选择

| 文件系统 | 最大单个文件 | 权限支持 | 跨平台 |
|----------|--------------|----------|--------|
| NTFS | 16EB | 支持 | Windows 内置读写，macOS 默认只读，Linux 需 ntfs-3g |
| FAT32 | 4GB | 不支持 | 全平台支持 |
| exFAT | 16EB | 不支持 | 全平台支持 |
| APFS | 8EB | 支持 | macOS 内置，Windows 需第三方驱动，Linux 只读 |
| ext4 | 16TB | 支持 | Linux 内置，Windows 需第三方驱动，macOS 需第三方驱动 |

## 分区表

| 分区表 | 最大容量 | 最大分区数 | 容错性 | 启动方式 |
|--------|----------|----------|--------|----------|
| MBR | 2TB | 4 个主分区 | 无 | BIOS |
| GPT | 9.4ZB | 128 个分区 | 有冗余备份 | UEFI |

## 隐藏文件与文件属性

### 显示隐藏文件

1. 打开文件资源管理器（`Win+E`）
2. 点击顶部"查看"→勾选"隐藏的项目"（Win10）或"查看"→"显示"→"隐藏的项目"（Win11）

### 文件属性

右键文件→"属性"→"常规"选项卡底部：

- **只读**：文件只能读取，不能修改或删除内容
- **隐藏**：文件在资源管理器中默认不显示

### 文件扩展名显示

1. 打开文件资源管理器（`Win+E`）
2. "查看"→勾选"文件扩展名"（Win10）或"查看"→"显示"→"文件扩展名"（Win11）

## 回收站

- **恢复文件**：双击桌面回收站→右键选中文件→"还原"
- **彻底删除**：右键选中文件→"永久删除"（或清空整个回收站）
> **注意**：U 盘、移动硬盘上的文件按 Delete 通常直接永久删除，不会进回收站

## 磁盘管理

打开方式：右键开始菜单（Win+X）→磁盘管理，或 `Win+R`→`diskmgmt.msc`

| 操作 | 方法 |
|------|------|
| 新建分区 | 右键未分配区域→"新建简单卷" |
| 格式化分区 | 右键分区→"格式化"（会清空所有文件） |
| 缩小分区 | 右键分区→"收缩卷" |
| 扩大分区 | 右键分区→"扩展卷"（需右侧紧邻未分配空间） |
| 更改盘符 | 右键分区→"更改驱动器号和路径" |
