# 连接外设

## 基本概念

- **蓝牙**：短距离无线通信技术，有效范围约 10 米，用于连接耳机、鼠标、键盘等
- **Miracast**：无线显示协议，通过 Wi-Fi Direct 将屏幕画面投射到支持该协议的显示设备
- **MTP（媒体传输协议）**：手机连电脑时选择的文件传输模式，允许电脑访问手机存储

## USB 版本与速度

| 版本 | 最大速度 | 接口形态 |
|------|----------|----------|
| USB 2.0 | 480Mbps | Type-A |
| USB 3.2 Gen 1（原 USB 3.0） | 5Gbps | Type-A / Type-C |
| USB 3.2 Gen 2（原 USB 3.1） | 10Gbps | Type-A / Type-C |
| USB 3.2 Gen 2×2 | 20Gbps | Type-C |
| USB4 | 40Gbps | Type-C |
| USB4 v2 | 80Gbps | Type-C |

识别方法：接口内部蓝色或红色塑料片通常为 USB 3.x，黑色为 USB 2.0

### Type-C 功能标识

| 标识 | 功能 |
|------|------|
| 闪电图标 | 仅充电 |
| SS（SuperSpeed） | USB 3.x 数据传输 |
| SS + 闪电 | USB 3.x 数据 + 充电 |
| D（DisplayPort） | 视频输出 |
| Thunderbolt（闪电箭头） | Thunderbolt 3/4（数据+视频+充电） |

无任何标识的 Type-C 口需查阅设备说明书确认功能。

### Thunderbolt 版本

| 版本 | 速度 | 视频输出 | 充电 |
|------|------|----------|------|
| Thunderbolt 3 | 40Gbps | 支持 | 支持（最高 100W） |
| Thunderbolt 4 | 40Gbps | 支持（双 4K 或单 8K） | 支持（最高 100W） |
| Thunderbolt 5 | 80Gbps（带宽增强模式 120Gbps） | 支持（双 8K） | 支持（最高 240W） |

## 蓝牙配对

1. 打开"设置>蓝牙和设备>蓝牙"，开启蓝牙
2. 打开蓝牙设备的配对模式（耳机通常长按电源键几秒直到指示灯闪烁，鼠标一般底部有配对按钮）
3. 在电脑上点击"添加设备>蓝牙"
4. 找到要配对的设备，点击连接
5. 确认配对码（如有），点击"连接"

### 已配对设备管理

- **断开但保留**：列表中点击设备→"断开"
- **彻底删除**：列表中点击设备→"删除设备"（需重新配对才能再用）
- **自动重连失败**：确认双方蓝牙都开着、距离 10 米以内，或删除后重新配对

### 蓝牙连不上排查

1. 确认电脑蓝牙已打开
2. 确认设备电量充足且已开机
3. 确认设备处于可被发现状态
4. 删除配对记录，重新配对
5. 台式机无内置蓝牙模块需外接 USB 蓝牙适配器

## 外接显示器

1. 用视频线（HDMI/VGA/DP）连接电脑和显示器
2. 打开显示器电源，切换到对应输入源
3. 按 `Win+P` 选择投影模式：
   - **仅电脑屏幕**：仅电脑显示
   - **复制**：电脑和显示器显示相同内容
   - **扩展**：电脑和显示器显示不同内容
   - **仅第二屏幕**：仅显示器显示

### 连接线接口

| 接口 | 说明 |
|------|------|
| HDMI | 传画面和声音 |
| DisplayPort（DP） | 传画面和声音，支持高刷新率 |
| VGA | 模拟信号接口，只传画面不传声音 |
| Type-C | 部分支持视频输出（不是所有 Type-C 口都能输出视频） |

**HDMI 版本**

| 版本 | 最大分辨率@刷新率 |
|------|-------------------|
| HDMI 1.4 | 4K@30Hz |
| HDMI 2.0 | 4K@60Hz |
| HDMI 2.1 | 4K@120Hz / 8K@60Hz |

**DP 版本**

| 版本 | 最大带宽 |
|------|----------|
| DP 1.2 | 17.28Gbps（4K@60Hz） |
| DP 1.4 | 25.92Gbps（4K@120Hz / 8K@60Hz） |
| DP 2.1 | UHBR10: 40Gbps / UHBR13.5: 54Gbps / UHBR20: 80Gbps |

### 扩展模式调整排列

1. 桌面空白处右键>"显示设置"
2. 拖动屏幕方块，让位置和实际摆放一致
3. 点击"应用"

**设置主屏幕**：显示设置中点击某个屏幕→勾选"使其成为主显示器"

**调整分辨率与缩放**：显示设置中可调整缩放比例和分辨率

### 合盖使用外接显示器

1. 按 `Win+R` 输入 `control` 打开控制面板
2. "硬件和声音">"电源选项"> 左侧"选择关闭盖子的功能"
3. 把"关闭盖子时"改为"不采取任何操作"
4. 点击"保存修改"

### 无线投屏

1. 确保笔记本和显示设备在同一 Wi-Fi 下
2. 按 `Win+K` 打开投屏面板
3. 点击要投屏的设备名称
4. 选择"复制"或"扩展"模式

**技术限制**：延迟 50-200ms、需同一 Wi-Fi、接收设备需支持 Miracast 协议

## 手机连电脑

1. 用数据线连接手机和电脑
2. 手机上选择"传输文件"或"传输媒体"（有的手机叫"文件传输"或"MTP 模式"）
3. 打开"此电脑"，找到手机名称
4. 双击打开手机存储，可复制文件

**照片位置**：通常在 `DCIM` 或 `Pictures` 文件夹

## U 盘弹出

1. 点击任务栏右下角的"安全删除硬件并弹出媒体"图标
2. 点击要弹出的 U 盘名称
3. 提示"安全地移除硬件"后，拔出 U 盘
4. 弹出失败：关掉所有访问 U 盘的窗口和程序，再尝试

## 有线音频设备

- 3.5mm 音频接口（圆形小孔，旁边有耳机图标）：插上自动识别
- 没有接口的轻薄本：用 Type-C 转 3.5mm 转接头，或蓝牙耳机
- 插了耳机但声音从扬声器出来：检查"设置→系统→声音"输出设备

## 扩展坞

- 接口类型要跟电脑匹配（USB-A 或 Type-C）
- 需要外接显示器的，确认扩展坞支持视频输出
