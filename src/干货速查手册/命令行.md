# 命令行

## 基本概念

- **命令行**：通过输入文本命令操作电脑
- **CMD（命令提示符）**：Windows 自带命令行程序
- **PowerShell**：Windows 新一代命令行程序
- **管道（|）**：前一个命令的输出作为后一个命令的输入
- **重定向（>、>>）**：将命令输出保存到文件，`>` 覆盖写入，`>>` 追加写入
- **通配符**：代替文件名中的部分内容，`*` 匹配任意多个字符，`?` 匹配单个字符

## 打开方式

### 命令提示符（CMD）

1. 按 `Win+R` 打开运行对话框
2. 输入 `cmd`，回车

### PowerShell

1. 按 `Win+X`，选择"Windows PowerShell"或"Windows 终端"

### 管理员模式

1. 搜索"cmd"或"PowerShell"
2. 右键"以管理员身份运行"
3. 按 UAC 弹窗提示点击"是"

## 常用命令

### 文件管理

| 命令 | 作用 | 示例 |
|------|------|------|
| `dir` | 列出当前目录下的文件和文件夹 | `dir`、`dir *.txt` |
| `cd` | 进入指定目录 | `cd Documents` |
| `cd ..` | 返回上级目录 | `cd ..` |
| `cd \` | 回到根目录 | `cd \` |
| `mkdir` | 创建新文件夹 | `mkdir 新建文件夹` |
| `rmdir` | 删除空文件夹 | `rmdir 空文件夹` |
| `del` | 删除文件 | `del *.tmp` |
| `copy` | 复制文件 | `copy file.txt D:\backup\` |
| `move` | 移动文件 | `move file.txt D:\backup\` |
| `ren` | 重命名 | `ren old.txt new.txt` |

### 网络命令

| 命令 | 作用 | 示例 |
|------|------|------|
| `ipconfig` | 查看网络配置 | `ipconfig`、`ipconfig /all`、`ipconfig /flushdns` |
| `ping` | 测试连通性 | `ping baidu.com`、`ping -t baidu.com` |
| `tracert` | 追踪网络路径 | `tracert baidu.com` |
| `netstat` | 查看网络连接 | `netstat -ano` |

### 系统命令

| 命令 | 作用 | 示例 |
|------|------|------|
| `tasklist` | 列出所有运行的进程 | `tasklist` |
| `taskkill` | 结束进程 | `taskkill /im chrome.exe /f` |
| `sfc /scannow` | 扫描并修复系统文件 | `sfc /scannow`（需管理员权限） |
| `chkdsk` | 检查磁盘错误 | `chkdsk C: /f`（需重启） |
| `systeminfo` | 查看系统信息 | `systeminfo` |
| `cls` | 清屏 | `cls` |
| `exit` | 退出命令行 | `exit` |
| `where` | 查找可执行文件路径 | `where python` |
| `type` | 显示文本文件内容 | `type readme.txt` |
| `findstr` | 在文件中搜索字符串 | `findstr "error" log.txt` |
| `tree` | 以树形结构显示目录 | `tree` |
| `diskpart` | 磁盘分区管理 | 需管理员权限，交互式操作 |

## CMD 与 PowerShell 差异

| 命令 | CMD 行为 | PowerShell 行为 |
|------|----------|-----------------|
| `dir` | 列出文件（DOS 内置命令） | 列出文件（实际是 `Get-ChildItem` 的别名） |
| `cd` | 切换目录（DOS 内置命令） | 切换目录（实际是 `Set-Location` 的别名） |
| `del` | 删除文件 | 删除文件（实际是 `Remove-Item` 的别名） |
| `copy` | 复制文件 | 复制文件（实际是 `Copy-Item` 的别名） |
| `move` | 移动文件 | 移动文件（实际是 `Move-Item` 的别名） |
| `mkdir` | 创建文件夹 | 创建文件夹（实际是 `New-Item -ItemType Directory` 的别名） |
| `type` | 显示文件内容 | 显示文件内容（实际是 `Get-Content` 的别名） |
| `findstr` | 搜索字符串 | 搜索字符串（PowerShell 中也可用 `Select-String`，输出对象格式） |
| `cls` | 清屏 | 清屏（实际是 `Clear-Host` 的别名） |
| `tasklist` | 列出进程 | 列出进程（PowerShell 中也可用 `Get-Process`） |
| `taskkill` | 结束进程 | 结束进程（PowerShell 中也可用 `Stop-Process`） |

### CMD 可用但 PowerShell 需替代的命令

| CMD 命令 | PowerShell 替代 | 说明 |
|----------|----------------|------|
| `where` | `Get-Command` 或 `where.exe` | PowerShell 中 `where` 是 `Where-Object` 的别名，需加 `.exe` 后缀才能调用系统命令 |
| `findstr` | `Select-String` | `findstr` 在 PowerShell 中仍可用，`Select-String` 输出对象格式 |
| `set` | `Get-ChildItem Env:` | `set` 在 PowerShell 中用于变量赋值，查看环境变量需用 `Get-ChildItem Env:` |
| `chcp` | `[Console]::OutputEncoding` | 修改编码页的方式不同 |
| `title` | `$host.UI.RawUI.WindowTitle` | 修改窗口标题的方式不同 |
| `echo` | `Write-Output` | `echo` 在 PowerShell 中是 `Write-Output` 的别名，基本用法相同；但 `echo.`（输出空行）在 PowerShell 中无效，需用 `Write-Output ""` |
| `echo off` | 无直接替代 | CMD 批处理中关闭命令回显，PowerShell 中不适用 |

**加 `.exe` 后缀**：PowerShell 中部分 CMD 命令被同名别名覆盖，加 `.exe` 可强制调用系统原生命令，如 `where.exe python`、`findstr.exe "error" log.txt`。

### 环境变量格式

| 操作 | CMD | PowerShell |
|------|-----|------------|
| 读取环境变量 | `%PATH%` | `$env:PATH` |
| 设置环境变量（临时） | `set MYVAR=hello` | `$env:MYVAR = "hello"` |
| 追加环境变量 | `set PATH=%PATH%;C:\new` | `$env:PATH += ";C:\new"` |
| 查看所有环境变量 | `set` | `Get-ChildItem Env:` |

**永久修改环境变量**：CMD 和 PowerShell 临时修改都只对当前窗口生效。永久修改需通过"系统属性→环境变量"。

**CMD 与 PowerShell 区别**：
- PowerShell 中 `>` 和 `>>` 是重定向操作符，与 CMD 行为一致
- PowerShell 支持 `|` 管道传递对象，例如 `Get-Process | Where-Object {$_.CPU -gt 100}`
- PowerShell 命令参数用 `-` 前缀，CMD 用 `/` 前缀
- PowerShell 不区分命令大小写，CMD 也不区分

## 高级技巧

### 管道（|）

```cmd
type log.txt | findstr "error"
tasklist | findstr "chrome"
```

### 重定向（>、>>）

```cmd
systeminfo > system.txt
tracert baidu.com >> trace.txt
```

### 通配符

- `*`：匹配任意字符（`*.txt`、`file*`）
- `?`：匹配单个字符（`file?.txt`）

### 自动补全

- `Tab`：自动补全文件名或命令
- `↑` `↓`：查看历史命令

### 中止命令

- `Ctrl+C`：中止当前正在运行的命令

### 路径中有空格

```cmd
cd "Program Files"
```

### 复制粘贴

- **复制**：选中文字，按回车（或右键一下）
- **粘贴**：在光标位置右键一下
- 新版 Windows 终端也支持 `Ctrl+Shift+C` 复制、`Ctrl+Shift+V` 粘贴
