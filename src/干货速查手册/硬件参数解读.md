# 硬件参数解读

## 基本概念

- **CPU（中央处理器）**：电脑的核心运算芯片
- **GPU（图形处理器）**：处理图形渲染的芯片，集成显卡内置于 CPU，独立显卡为独立芯片
- **TDP（热设计功耗）**：芯片最大发热量，单位瓦特（W）
- **DDR**：内存的技术标准代数，DDR4 和 DDR5 互不兼容，需主板和 CPU 同时支持对应版本
- **VRAM（显存）**：显卡专用存储芯片
- **PPI（像素密度）**：每英寸像素数量
- **FPS（帧率）**：每秒输出的画面帧数
- **NVMe**：固态硬盘传输协议，通过 PCIe 通道传输数据
- **SATA**：硬盘传输接口，速度上限约 600MB/s

## CPU

### 型号解读

**Intel 型号**：`i5-13500H` → `i5` 定位 / `13` 代 / `500` 型号 / `H` 后缀
**Intel 新旧命名对照**：
- 旧：`i3/i5/i7/i9` → 新：`Core 3/5/7`
- 新：`Core Ultra 3/5/7/9`（移动处理器，带 **NPU（神经网络处理器）**）
**AMD 型号**：`R5-7600X` → `R5` 定位 / `7` 代 / `600` 型号 / `X` 后缀

### 后缀含义

**Intel**

| 后缀 | 含义 |
|------|------|
| H | 高性能移动版 |
| U | 低功耗移动版 |
| P | 高性能低功耗版 |
| K | 可超频台式机版 |
| F | 无核显台式机版 |
| G | 含特定级别核显的移动版（G1/G4/G7） |
| XE | 极致性能台式机版 |

**AMD**

| 后缀 | 含义 |
|------|------|
| X | 高频版 |
| XT | 更高频版 |
| G | 含核显版 |
| HX | 高性能移动版（可超频） |
| HS | 高性能低功耗移动版 |
| U | 低功耗移动版 |

## 内存

### DDR 版本与双通道

**DDR 版本**：DDR4 vs DDR5（DDR4 和 DDR5 互不兼容，需主板和 CPU 同时支持对应版本）
**双通道**：`2×8GB` 性能 > `1×16GB`

### 常见内存频率

| DDR 版本 | 常见频率 |
|----------|----------|
| DDR4 | 2400MHz、2666MHz、3200MHz、3600MHz |
| DDR5 | 4800MHz、5200MHz、5600MHz、6000MHz、6400MHz |

### 笔记本内存

- 可更换：部分游戏本和商务本
- 板载焊死：轻薄本常见

## 硬盘

### 类型对比

| 类型 | 速度 | 结构 |
|------|------|------|
| HDD（机械硬盘） | 100-200MB/s | 有机械部件，怕震动 |
| SSD（固态硬盘） | 500MB/s 以上 | 无机械部件 |

### SSD 接口

| 接口 | 速度 |
|------|------|
| SATA | 500MB/s 左右 |
| NVMe（M.2） | 3000-7000MB/s |

NVMe SSD 的速度取决于 PCIe 版本：

| PCIe 版本 | ×4 通道理论带宽 | 实际顺序读写约 |
|-----------|----------------|---------------|
| PCIe 3.0 | ~4GB/s | 3000-3500MB/s |
| PCIe 4.0 | ~8GB/s | 5000-7500MB/s |
| PCIe 5.0 | ~16GB/s | 10000-12000MB/s |

## 显卡

### 核心区别

**集成显卡**：集成在 CPU 中，与 CPU 共享内存作为显存
**独立显卡**：独立芯片，自带显存（VRAM）

### 型号系列

- NVIDIA：RTX 50 系列（5060/5070/5080）、RTX 40 系列（4060/4070）
- AMD：RX 9070、RX 7000 系列（7600/7700/7800）

### 显存（VRAM）

显存不足时会出现画面卡顿、纹理模糊、闪退等现象。

| 容量 | 技术说明 |
|------|----------|
| 4GB | 可容纳 1080p 级别纹理资源 |
| 8GB | 可容纳 1080p 高品质纹理资源 |
| 12GB | 可容纳 2K 级别纹理资源 |
| 16GB | 可容纳 4K 级别纹理资源或 AI 模型 |

查看显存：`Win+R`→`dxdiag`→"显示"选项卡→"显存"字段

### 笔记本显卡 vs 桌面版

笔记本显卡为移动版，TDP 低于桌面版，同型号性能低于桌面版。

## 显示器

### 分辨率

| 名称 | 分辨率 | 像素总数 | 长宽比 |
|------|--------|----------|--------|
| 1080p / FHD | 1920×1080 | 约 207 万 | 16:9 |
| 2K / QHD | 2560×1440 | 约 369 万 | 16:9 |
| 4K / UHD | 3840×2160 | 约 829 万 | 16:9 |

27 寸屏幕 PPI 参考：1080p 约 82，2K 约 109，4K 约 163

设置分辨率：桌面空白处右键→"显示设置"→"显示分辨率"

### 刷新率

| 刷新率 | 说明 |
|--------|------|
| 60Hz | 默认值 |
| 120Hz | 每秒刷新 120 次 |
| 144Hz | 每秒刷新 144 次 |
| 240Hz | 每秒刷新 240 次 |

GPU 帧率低于刷新率时，显示器实际按 GPU 帧率显示

设置刷新率：桌面空白处右键→"显示设置"→"高级显示设置"→"选择刷新率"

### 面板类型

| 面板 | 色域覆盖 | 可视角度 | 响应时间 | 对比度 |
|------|----------|----------|----------|--------|
| IPS | 广色域 | 178° | 4-5ms | 1000:1 |
| TN | 色域窄 | 170° | 1ms | 1000:1 |
| VA | 广色域 | 178° | 4-8ms | 3000:1+ |

### HDR

HDR（高动态范围）：亮部和暗部同时显示更多细节。显示器标注的 HDR 等级：

| 等级 | 峰值亮度 | 说明 |
|------|----------|------|
| HDR400 | 400nit | 最低 HDR 认证等级 |
| HDR600 | 600nit | 中等级别 |
| HDR1000 | 1000nit | 高等级别 |

开启 HDR：桌面空白处右键→"显示设置"→"使用 HDR"

### 色域

色域：显示器能呈现的颜色范围

| 标准 | 覆盖范围 |
|------|----------|
| sRGB | 互联网和大多数软件的默认色彩空间 |
| DCI-P3 | 电影行业标准，比 sRGB 覆盖更广 |
| Adobe RGB | 印刷行业标准，绿色覆盖比 sRGB 更广 |

## 查看配置的工具

| 想看什么 | 用什么 | 怎么打开 |
|----------|--------|----------|
| CPU 型号、内存大小、显卡型号 | 任务管理器→性能 | `Ctrl+Shift+Esc` |
| 整机硬件信息（图形化） | 系统信息 | 按 Win 键→输入"系统信息"→回车 |
| Windows 版本、系统类型 | 设置→关于 | 右键开始→系统→关于 |
| 只看 Windows 版本号 | winver | `Win+R`→`winver` |
| 硬盘剩余空间 | 此电脑 | `Win+E` |
| 显卡驱动版本、显存、DX 版本 | DirectX 诊断工具 | `Win+R`→`dxdiag` |
| 设备是否正常、驱动状态 | 设备管理器 | `Win+X`→设备管理器 |

## 笔记本 vs 台式机

| 硬件 | 笔记本 | 台式机 |
|------|--------|--------|
| CPU | 移动版（H/U/P/HX 等后缀），TDP 9-55W | 台式机版（K/F/T 等后缀或无后缀），TDP 35-125W+ |
| 显卡 | 移动版，性能低于同型号桌面版 | 桌面版 |
| 内存 | 部分可更换，多数轻薄本板载 | 可更换，支持大容量 |
| 硬盘 | 多为 M.2 NVMe SSD | 支持多种接口，可装多块硬盘 |
| 升级空间 | 受机身限制 | 机箱空间大，扩展槽多 |
| 便携性 | 可移动 | 固定使用 |
| 散热 | 受机身尺寸限制 | 机箱空间大，散热条件好 |
