# 日常使用：熟悉基本操作后的进阶技能

> 阅读前提：你已经了解了 PC 的硬件组成和基本操作（参见《认识PC》和《基本操作》）。

## 一、文件管理——最重要的基本技能

电脑里的一切——文档、照片、音乐、软件——都以"文件"的形式存在。文件存放在"文件夹"（也称目录）里。管理好文件，才能在使用电脑时不手忙脚乱。

### 打开文件资源管理器

按 Win+E，或双击桌面上的"此电脑"图标，就打开了文件资源管理器。这是你管理文件的"总部"。

### 盘符（驱动器）的概念——Windows 下的划分方式

一块物理硬盘可以划分成多个"分区"，每个分区在 Windows 里被分配一个盘符（如 C:、D:）。

在"此电脑"里，你会看到：

- **C 盘** — 系统盘，Windows 和大部分软件默认安装在这里。**不要手动删除 C 盘里的系统文件和程序文件**
- **其他盘**（如果有 D 盘、E 盘等）— 可以放个人文件，减轻 C 盘压力
- **U 盘/移动硬盘** — 插上后出现。Windows 默认开启"快速删除"模式，文件复制/移动完成后可以直接拔出。如果不确定是否还在传输，等几秒或看任务栏有没有"安全删除硬件"图标，点一下再拔更稳妥

C 盘空间太满会影响装软件和系统更新。如果有其他盘，建议把个人文件放其他盘；如果只有 C 盘（很多 SSD 只分一个区），注意留出足够空间即可。想调整分区可以阅读[《磁盘分区基础》](../深入存储知识/磁盘分区.md)。

### 文件与文件夹

- **文件夹**：可以理解为"收纳盒"，里面可以放文件和子文件夹。在文件资源管理器中它以黄色图标显示。
- **文件**：每一个文件都有名字，名字后面通常还有一个扩展名。扩展名（文件名最后一个小数点后面的部分）表示文件的类型，比如：
  - .txt — 纯文本文件，记事本打开
  - .docx — Word 文档
  - .xlsx — Excel 表格
  - .pptx — PowerPoint 演示文稿
  - .pdf — 电子文档（跨平台显示格式，不能直接编辑）
  - .jpg / .png — 图片文件
  - .mp3 — 音频文件
  - .mp4 — 视频文件
  - .exe — 可执行程序（安装包或软件本身）

默认情况下，Windows 隐藏了已知文件类型的扩展名（所以你看到的可能是"报告"而不是"报告.docx"）。建议让扩展名显示出来：

**Windows 10：** 打开任意文件夹 → 顶部菜单栏点击"查看" → 勾选"文件扩展名"
**Windows 11：** 打开任意文件夹 → 顶部菜单栏点击"查看" →"显示" → 勾选"文件扩展名" [查]

显示后你能清楚知道每个文件是什么类型。后续学习压缩文件时，也需要靠扩展名来识别 .zip、.7z、.rar 等格式。

### 基本操作

**新建文件或文件夹：**

1. 在你想要创建的位置（比如 D 盘、桌面上），右键空白处
2. 选择"新建" → 选"文件夹"或你要创建的文件类型（文本文档、Word 文档等）
3. 给文件夹/文件取个有意义的名字，按 Enter 确认

> **试一试：** 现在在桌面空白处右键 → 新建 → 文件夹，给它取个名字，试试看。

**重命名：**

- **最简单的方法**：选中文件/文件夹 → 等约1秒 → 再次点击文件名（不要点图标，也不要点得太快变成双击），文件名会变成可编辑状态 → 输入新名字 → 回车
- 或者：选中文件/文件夹 → 按 F2 → 输入新名字 → 回车
- 每次重命名只改变名字，不影响文件内容
- **批量重命名**：选中多个文件 → 按 F2 → 输入新名字 → 回车，系统会自动给每个文件加上 (1)、(2) 编号

**选中多个文件：**

- **框选（最直观，不需要按键盘）**：按住鼠标左键不放，在文件区域拖动出一个虚线框，框住的文件都会被选中。适合选中一片相邻的文件。
- 连续选中：点击第一个 → 按住 Shift → 点击最后一个（选中两者之间的全部文件）
- 不连续选中：按住 Ctrl → 逐个点击要选的文件
- 全选：Ctrl+A

**复制/移动文件：**

- **纯鼠标方式（不需要记快捷键）**：
  - 复制：右键选中文件 → "复制" → 到目标位置 → 右键空白处 → "粘贴"
  - 移动：右键选中文件 → "剪切" → 到目标位置 → 右键空白处 → "粘贴"
- 快捷键方式：
  - 复制：选中 → Ctrl+C → 到目标位置 → Ctrl+V
  - 移动：选中 → Ctrl+X → 到目标位置 → Ctrl+V
- 也可以用鼠标拖拽：同盘拖拽是移动，异盘拖拽是复制。

**删除文件：**

- **最简单的方法**：右键选中文件 → "删除" → 文件进入回收站（可恢复）
- 或者：选中 → Delete 键 → 文件进入回收站（可恢复）
- 彻底删除（跳过回收站，很难恢复）：选中 → Shift+Delete
- 如果想恢复回收站里的文件：双击桌面回收站 → 右键选中文件 → "还原"

### 给文件/文件夹命名的好习惯

好的命名能让你快速找到需要的文件。建议：

- 名字要有意义，能反映文件内容
- 包含日期信息（如"2026-05-21_学期总结_v2.docx"）
- 序号或版本号（v1、v2、final、最终版——最后建议加日期避免混淆）
- 避免用"新建文档(1).docx"、"111.jpg"这种默认名字

一种参考的文件组织方式（在 D 盘或"文档"文件夹内）：

```text
D:\
├── 学习
│   ├── 课程笔记
│   ├── 作业
│   └── 电子书
├── 工作
│   ├── 项目A
│   └── 项目B
├── 个人
│   ├── 照片
│   └── 重要文件
└── 下载（可以设置浏览器下载默认路径到这里）
```

## 二、C 盘里有什么（简单了解）

C 盘是系统盘，Windows 操作系统和大部分软件默认安装在这里。打开"此电脑"双击 C 盘，你会看到几个文件夹：

| 文件夹            | 用途                             | 能不能动                        |
| ----------------- | -------------------------------- | ------------------------------- |
| **Program Files** | 安装的软件放在这里               | 不建议手动删，通过"设置→应用"卸载更干净 |
| **Users（用户）** | 你的个人文件（文档、下载、桌面） | 可以管理自己的文件              |
| **Windows**       | 操作系统本身                     | 不建议手动修改，系统运行全靠它  |
| **回收站**        | 删除的文件临时存放处             | 可以清空释放空间                |

你的个人文件实际存在 `C:\Users\你的用户名` 里面，包括桌面、文档、下载、图片等文件夹。C 盘空间太满会影响装软件和系统更新，建议：

- 个人文件（照片、视频、下载）尽量放 D 盘（如果有的话）
- 定期清理"下载"文件夹
- 桌面尽量避免堆太多文件（桌面文件也在 C 盘）

想了解更多细节，可以阅读[《Windows 存储进阶》](../深入存储知识/Windows%20存储进阶.md)。

## 三、C 盘满了怎么办

如果 C 盘空间变红（剩余空间不足），用系统自带的工具清理就够了。

### 存储感知（推荐）

设置 → 系统 → 存储 → 存储感知 → 开启"自动清理"。

可以设置每月自动清理临时文件和回收站里超过一定时间的文件。

### 磁盘清理

适合不喜欢"自动"、想自己看一眼再决定删什么的人。

1. 在搜索框输入"磁盘清理"并打开
2. 选择 C 盘，等待扫描
3. 勾选可以安全删除的项目：临时文件、回收站、Windows 更新清理
4. 点击"确定"删除

**不建议做的事：**

- 避免手动删除 Windows 或 Program Files 里的文件
- 尽量避免安装第三方"清理大师"类软件（容易弹广告、可能误删文件）

（更多清理细节参见上文提到的《Windows 存储进阶》。）

### 回收站操作

按 Delete 键删除的本地硬盘文件，通常会被移到回收站（如果文件太大超出回收站容量，则会直接删除）。

- **恢复文件**：双击桌面回收站 → 右键选中文件 → "还原"
- **彻底删除**：右键选中文件 → "永久删除"（或清空整个回收站）
- **注意**：U 盘、移动硬盘上的文件按 Delete 通常直接永久删除，不会进回收站（不同于本地硬盘）

### C 盘空间建议

- 留出足够的空间给系统运转。如果 C 盘剩余空间低于 **15-20GB**，容易遇到更新失败或系统卡顿
- 安装大型软件前看一眼 C 盘剩余空间。如果只有一个 C 盘且空间充足，直接装 C 盘没问题；如果 C 盘空间紧张，可以考虑装到其他盘（如果有）
- 桌面尽量避免堆太多文件——桌面的文件实际存在 `C:\Users\你的用户名\Desktop`，堆满桌面等于堆满 C 盘

（pagefile.sys、hiberfil.sys 等系统文件的说明和深度清理方法，参见上文提到的《Windows 存储进阶》。）

## 四、连接网络

### 连接 Wi-Fi

1. 点击右下角托盘区的网络图标（像扇形或小地球）
2. 在弹出面板里找到你的 Wi-Fi 名称，点击它
3. 勾选"自动连接"（下次自动连，不用再输密码）
4. 点击"连接"，输入 Wi-Fi 密码
5. 连接成功后，网络图标会变成实心的扇形

### 有线网络

插上网线通常会自动获取 IP 地址并联网（家用网络一般不需要额外设置）。如果插了网线但连不上，重启一下电脑或检查网线是否插紧了。公司/学校网络可能需要额外认证或配置。想了解这个过程的技术原理，可以搜索"DHCP"自行探索。

### 连接后打不开网页怎么办

看网络图标，如果有黄色感叹号，说明连上了但无法上网。按这个顺序排查：

1. **重启路由器**（拔掉电源等 10 秒再插上）— 能解决绝大多数问题
2. 重启电脑
3. 打开"设置"→"网络和 Internet"→"网络疑难解答"，让 Windows 自动修复
4. 以上都不行，联系网络服务商

想了解更多网络原理和排查方法，可以阅读[《理解你的网络》](../理解你的网络.md)。

## 五、安装与卸载软件

### 安装软件的安全途径

安装软件的最佳方式是从**软件的官方网站**下载。用搜索引擎找官网时要小心——不同搜索引擎的结果排序不一样，有些前排是广告或仿冒网站，点进去下载的可能不是你要的软件，而是"XX管家""XX助手"之类的捆绑程序。有些搜索引擎会把官网放在第一位，但广告和仿冒链接也常常混在结果里，需要仔细辨别。

#### 识别官网的三个技巧

**1. 看网址的主域名**

官网的网址通常包含软件或厂商的名字。例如：
- WPS 官网：`wps.cn`
- QQ/微信：`qq.com`
- 7-Zip：`7-zip.org`
- Chrome：`google.com/chrome`

如果网址是"XX下载站""XX软件园"，那就是第三方网站，不是官网。

**2. 警惕"高仿"名称**

有些网站会在软件名后面加字来混淆，比如"Steam管家""Office助手""WPS破解版"——这些都不是官网。

**3. 搜索引擎广告位**

搜索结果里标注"广告"的，基本都是第三方下载站。真正的官网通常在广告下面，或者标注"官网"字样。

> 不确定的时候，可以问 AI"XX软件官网是什么"，让它给你地址，你自己核对一下域名。

#### 第三方下载站怎么用

有时候确实找不到官网，只能用第三方下载站。记住一条：**避开所有写着"高速下载""极速下载""安全下载"的按钮**，只点"普通下载"或"本地下载"。那些"高速下载"下下来的往往是"下载器"，会给你装一堆不需要的软件。

#### 32位还是64位

下载时如果让你选"32位"或"64位"，按这个规则：
- 右键"开始"按钮 →"系统" → 看"系统类型"
- 显示"64位操作系统" → 选64位（x64）
- 显示"32位操作系统" → 只能选32位（x86）

现在大部分电脑都是64位，选64位一般没错。

### 安装流程（以 7-Zip 为例）

7-Zip 是一款免费的压缩软件，官网 `7-zip.org` 简洁好认，国内可以直接访问。我们以它为例演示安装过程：

1. 在浏览器地址栏直接输入 `7-zip.org`，进入官网下载安装包（通常是一个 .exe 文件）
2. 双击运行安装包——系统可能会弹出一个窗口，背景变暗，写着 **"你要允许此应用对你的设备进行更改吗？"**，下面有"是"和"否"两个按钮。这个弹窗叫 **UAC**（用户账户控制），是 Windows 的安全确认。如果是你主动安装软件，点"是"；如果突然弹出来你没做任何操作，点"否"。
3. 安装向导窗口弹出，每一步都看清楚再点"下一步"
4. **特别注意勾选框**：某些安装包会在某一步问你要不要装"XX浏览器主页"或"XX软件管家"，**取消勾选**这些无关选项再继续
5. 安装完成，桌面会出现快捷方式，或者开始菜单里能找到

> 如果你只是想装一个浏览器，Windows 自带的 **Microsoft Edge** 对大多数用户来说已经够用，不需要额外下载。Edge 的图标是蓝色波浪，在开始菜单里就能找到。如果你有特殊需求（如特定扩展、跨设备同步习惯），再考虑其他浏览器。

### 安装路径怎么选

软件默认装在 C 盘（通常是 `C:\Program Files` 或 `C:\Program Files (x86)`）。如果你的 C 盘空间不大（比如小于 300GB），可以在安装时把路径里的 `C:` 改成 `D:`，让软件装到 D 盘。

**改路径的方法**：安装过程中有个"浏览"或"自定义安装"的选项，点进去把 `C:\` 改成 `D:\` 即可。保持后面的文件夹结构不变，便于以后管理。

**注意**：如果软件默认装到 `C:\Users\你的用户名\AppData`，这种是"免权限"安装，改到其他盘可能需要你自己新建一个文件夹（比如 `D:\UserApps`）来存放。

### Microsoft Store 应用

Windows 自带的应用商店（Microsoft Store）里的应用，像手机装应用一样一键安装，安全干净、自动更新。不过 Store 的应用种类没有传统 exe 安装包多，有些常用软件找不到。

### 卸载软件

**推荐方式**（一般干净彻底）：

1. 打开"设置"→"应用"→"已安装的应用"
2. 搜索要卸载的软件
3. 点击软件右侧的三个点，选择"卸载"
4. 按提示完成

**不建议直接删除软件所在的文件夹**，那样会残留配置文件和系统记录，既占空间又让系统变杂乱。想了解"注册表"的技术细节，可以自行搜索探索。

**有些软件自带卸载程序**，在开始菜单的软件条目下能找到"卸载 XXX"的快捷方式。

### 注意"全家桶"

一些软件安装时会在你不注意的情况下"顺便"装上其他软件（比如装了 A 之后，发现桌面上多了 B 和 C）。**安装时每一步都仔细看**，取消勾选你不需要的附加软件。

### 软件一定要花钱吗

很多专业软件需要购买（如 Photoshop、Office），价格不便宜。对于初学者：

- **优先用免费替代品**：比如用 GIMP 代替 Photoshop，用 LibreOffice 代替 Office，用 VLC 代替付费播放器
- **开源软件**：很多优秀的软件是开源的（代码公开，免费使用），如 7-Zip、VLC、Firefox
- **学生优惠**：如果你是在校学生，很多软件有教育版或学生折扣
- **试用版**：大部分收费软件有免费试用期，可以先试试再决定

> 网上有很多"破解版"软件，但使用它们有安全风险（可能被植入病毒）和法律风险（用于商业用途可能被告）。建议尽量使用正版或免费替代品。

## 六、账户与安全设置

### 创建你的微软账户

如果你没有微软账户，建议注册一个（outlook.com 邮箱或用其他邮箱注册，也可在手机上访问 account.microsoft.com 提前创建）。微软账户的好处：

- 忘记 PIN 码时可以在线重置
- 重装系统后登录账户可以恢复一些设置
- 免费的 OneDrive 网盘（文件存到网上后，换电脑也能访问）

如果你不想用微软账户登录，可以创建本地账户（仅限此电脑，忘记密码时无法在线重置）。

### 设置安全的密码/PIN

PIN 只在这台电脑上有效，即使别人知道了你的 PIN，也无法用来登录其他设备。建议：

- PIN 至少 6 位，建议避免使用生日、重复数字（111111）、连续数字（123456）等易被猜到的组合
- 设置"要求登录"：离开座位时按 Win+L 锁定，别人无法使用你的电脑

### 免费杀毒够用吗

Windows 10/11 自带的 Microsoft Defender 对大多数普通用户已经足够。它会在后台自动更新病毒库、扫描下载的文件、检查 U 盘。**一般情况下不需要额外安装第三方杀毒软件**，很多"免费杀毒"软件反而会拖慢电脑、弹广告、捆绑其他软件，并不比 Defender 更好。如果你有特殊安全需求（如企业环境、指定安全软件要求），则按实际情况选择。

## 七、养成这四个好习惯（长期受益）

1. **随时 Ctrl+S** — 编辑文档、写代码、做表格的时候，养成几秒就按一下 Ctrl+S 的习惯。很多崩溃发生在你忘记保存的时候。
2. **文件不乱放桌面** — 桌面只是中转站，不是仓库。用完的文件移动到合适的文件夹里。桌面塞满会拖慢系统，而且找东西很痛苦。
3. **不明链接不乱点** — 浏览器弹出"你的电脑有危险，立即扫描"的广告、邮件里有"你中了奖"的链接、QQ 里陌生人发来的文件——保持警惕。
4. **遇到问题先搜索** — 电脑出了小问题（软件报错、连不上网、某个功能不会用），把错误提示复制到手机百度或问 AI，大概率有人遇到过并给出了解决方案。养成这个习惯后，能帮你解决大部分小问题。
