# 日常使用：熟悉基本操作后的进阶技能

> 阅读前提：你已经了解了 PC 的硬件组成和基本操作（参见《认识PC》和《基本操作》）。

## 一、文件管理——最重要的基本技能

电脑里的一切——文档、照片、音乐、软件——都以"文件"的形式存在。文件存放在"文件夹"（也称目录）里。管理好文件，才能在使用电脑时不手忙脚乱。

### 打开文件资源管理器

按 Win+E，或双击桌面上的"此电脑"图标，就打开了文件资源管理器。这是你管理文件的"总部"。

### 盘符（驱动器）的概念——Windows 下的划分方式

一块物理硬盘可以划分成多个"分区"（就像把一个大柜子隔成几个独立小柜），每个分区在 Windows 里被分配一个盘符（字母加冒号，如 C:、D:）。品牌笔记本出厂时已经预装好 Windows 并做好了分区设置，具体因品牌和配置而异：有些只有一个 C 盘，有些则分成了 C 盘和 D 盘等多个分区。台式机如果是自己组装或请人安装的，分区方案在装系统时由装机者决定。如果你觉得分区不合理或需要调整（比如 C 盘空间不够、想分出更多盘），可以在搜索引擎或 AI 工具中搜索"Windows 磁盘管理 调整分区"了解操作步骤——分区操作有风险，请务必先备份重要数据。

在"此电脑"里，你会看到几个图标：

- **C 盘**（本地磁盘 C）— 系统盘，Windows 操作系统和大部分软件默认安装在这里。**不要随意删除或移动 C 盘里的文件和文件夹**，很多是系统运行必需的。
- **D 盘、E 盘**（如果硬盘分了多个区）— 有些用户会把个人文件（文档、照片、下载文件）放在这些盘。
- **有 USB 图标的是 U 盘或移动硬盘**，插上后会出现。用完直接拔就行（Windows 默认已设为快速删除模式，传完文件等几秒拔出即可）。如果不放心，右键托盘区 USB 图标选"弹出"更保险。

C 盘空间太满会影响装软件和系统更新。如果除了 C 盘还有其他盘（如 D 盘），建议把个人文件放在那里以分担 C 盘压力；如果只有一个 C 盘，注意留出足够的可用空间即可。想深入了解分区，可以搜索相关教程，或阅读[《磁盘分区》](./深入存储知识/磁盘分区.md)。


### 文件与文件夹

- **文件夹**：可以理解为"收纳盒"，里面可以放文件和子文件夹。在文件资源管理器中它以黄色图标显示。
- **文件**：每一个文件都有名字和扩展名。扩展名（文件名最后一个小数点后面的部分）表示文件的类型，比如：
  - .txt — 纯文本文件，记事本打开
  - .docx — Word 文档
  - .xlsx — Excel 表格
  - .pptx — PowerPoint 演示文稿
  - .pdf — 电子文档（跨平台显示格式，不能直接编辑）
  - .jpg / .png — 图片文件
  - .mp3 — 音频文件
  - .mp4 — 视频文件
  - .exe — 可执行程序（安装包或软件本身）

默认情况下，Windows 隐藏了已知文件类型的扩展名（所以你看到的可能是"报告"而不是"报告.docx"）。建议让扩展名显示出来，不同 Windows 版本的操作入口略有差异，可以搜索"Windows 显示文件扩展名"找到对应版本的方法。显示后你能清楚知道每个文件是什么类型。

### 基本操作

**新建文件或文件夹：**

1. 在你想要创建的位置（比如 D 盘、桌面上），右键空白处
2. 选择"新建" → 选"文件夹"或你要创建的文件类型（文本文档、Word 文档等）
3. 给文件夹/文件取个有意义的名字，按 Enter 确认

**重命名：**

- 选中文件/文件夹 → 按 F2 → 输入新名字 → 回车
- 每次重命名只改变名字，不影响文件内容

**选中多个文件：**

- 连续选中：点击第一个 → 按住 Shift → 点击最后一个（选中两者之间的全部文件）
- 不连续选中：按住 Ctrl → 逐个点击要选的文件
- 全选：Ctrl+A

**复制/移动文件：**

- 复制（原位置保留，新位置多一份拷贝）：选中 → Ctrl+C → 到目标位置 → Ctrl+V
- 移动（原位置消失，整体搬到新位置）：选中 → Ctrl+X → 到目标位置 → Ctrl+V
- 也可以用鼠标拖拽：同盘拖拽是移动，异盘拖拽是复制。记不住可以搜索"Windows 拖拽复制移动"确认。

**删除文件：**

- 选中 → Delete 键 → 文件进入回收站（可恢复）
- 选中 → Shift+Delete → 彻底删除（跳过回收站，很难恢复）
- 如果想恢复回收站里的文件：双击桌面回收站 → 右键选中文件 → "还原"

### 给文件/文件夹命名的好习惯

好的命名能让你快速找到需要的文件。建议：

- 名字要有意义，能反映文件内容
- 包含日期信息（如"2026-05-21_学期总结_v2.docx"）
- 序号或版本号（v1、v2、final、最终版——最后建议加日期避免混淆）
- 避免用"新建文档(1).docx"、"111.jpg"这种默认名字

一种参考的文件组织方式（在 D 盘或"文档"文件夹内）：

```text
D:\
├── 学习
│   ├── 课程笔记
│   ├── 作业
│   └── 电子书
├── 工作
│   ├── 项目A
│   └── 项目B
├── 个人
│   ├── 照片
│   └── 重要文件
└── 下载（可以设置浏览器下载默认路径到这里）
```

## 二、连接网络

### 连接 Wi-Fi

1. 点击右下角托盘区的网络图标（像扇形或小地球）
2. 在弹出面板里找到你的 Wi-Fi 名称，点击它
3. 勾选"自动连接"（下次自动连，不用再输密码）
4. 点击"连接"，输入 Wi-Fi 密码
5. 连接成功后，网络图标会变成实心的扇形

### 有线网络

插上网线通常会自动获取 IP 地址并联网（这个过程叫 DHCP，感兴趣可以搜索了解），不需要任何设置。如果插了网线但连不上，重启一下电脑或检查网线是否插紧了。

### 连接后打不开网页怎么办

先确认是否真的连上了：看网络图标，如果有个黄色感叹号，说明连上了但无法访问互联网。按这个顺序排查：

1. 打开"设置"→"网络和 Internet"→"高级网络设置"→"网络疑难解答"，让 Windows 自动检测修复
2. 忘记该 Wi-Fi，重新连接并输密码
3. 重启路由器（拔掉电源等 10 秒再插上）
4. 重启电脑
5. 如果仍不行，打开"设置"→"网络和 Internet"→"高级网络设置"→"网络重置"（此操作会删除并重装所有网卡驱动和网络设置）
6. 以上都不行，可能是宽带本身的问题，联系网络服务商

请注意：第 3 步（重启路由器）能解决绝大多数家庭网络问题。

## 三、安装与卸载软件

### 安装软件的安全途径

安装软件的最佳方式是从**软件的官方网站**下载，尽量避免从第三方下载站下载——很多第三方下载站会捆绑不需要的软件甚至夹带病毒。如何识别官网？

- 搜索引擎搜出来的结果，官方网站通常标注"官网"字样；也可以在 AI 工具中搜索软件名称，请它提供官网地址并核对后再访问
- 看网址，官网通常就是软件名字.com 或 .org（如 qq.com、google.com/chrome）

常见软件的官网（举例，非推荐）：

- 浏览器：google.cn/chrome、mozilla.org/firefox
- 压缩软件：7-zip.org（免费纯净无广告）
- 播放器：vlc.org
- 输入法：搜狗输入法 (ime.sogou.com)、百度输入法 (shurufa.baidu.com)

### 安装流程（以 Chrome 为例）

1. 官网下载安装包（通常是一个 .exe 文件）
2. 双击运行安装包——系统可能会弹出"你要允许此应用对你的设备进行更改吗？"的确认窗口，这是因为安装程序需要管理员权限才能向系统目录写入文件，详见《用户账户控制与权限》
3. 安装向导窗口弹出，多数情况下直接点"下一步"或"安装"即可
4. 注意每一步：某些安装包会在最后一步问你要不要装"XX 浏览器主页"或"XX 软件管家"，取消勾选这些无关选项
5. 安装完成，桌面会出现快捷方式，或者开始菜单里能找到

### Microsoft Store 应用

Windows 自带的应用商店（Microsoft Store）里的应用，像手机装应用一样一键安装，安全干净、自动更新。不过 Store 的应用种类没有传统 exe 安装包多。

### 卸载软件

**推荐方式**（一般干净彻底，有时需要管理员权限，详见《用户账户控制与权限》）：

1. 打开"设置"→"应用"→"已安装的应用"
2. 搜索要卸载的软件
3. 点击软件右侧的三个点，选择"卸载"
4. 按提示完成

**不建议直接删除软件所在的文件夹**，那样会残留配置文件和注册表信息（注册表是 Windows 用来记录系统和软件设置的大型数据库，感兴趣可以搜索了解），既占空间又让系统变杂乱。

**有些软件自带卸载程序**，在开始菜单的软件条目下能找到"卸载 XXX"的快捷方式。

### 注意"全家桶"

一些软件安装时会在你不注意的情况下"顺便"装上其他软件（比如装了 A 之后，发现桌面上多了 B 和 C）。**安装时每一步都仔细看**，取消勾选你不需要的附加软件。

## 四、账户与安全设置

### 创建你的微软账户

如果你没有微软账户，建议注册一个（outlook.com 邮箱或用其他邮箱注册，也可在手机上访问 account.microsoft.com 提前创建）。微软账户的好处：

- 忘记 PIN 码时可以在线重置
- 重装系统后登录账户可以恢复一些设置
- 免费的 OneDrive 云存储空间（在线网盘，文件存到云端后可在其他设备上访问）

如果你不想用微软账户登录，可以创建本地账户（仅限此电脑，忘记密码时无法在线重置）。

### 设置安全的密码/PIN

PIN 是 Windows Hello 的一部分（Windows Hello 是 Windows 的登录验证体系，包含指纹、人脸识别等，感兴趣可以搜索了解），只在这台电脑上有效。建议：

- PIN 至少 6 位，建议避免使用生日、重复数字（111111）、连续数字（123456）等易被猜到的组合
- 设置"要求登录"：离开座位时按 Win+L 锁定，别人无法使用你的电脑

### 免费杀毒够用吗

Windows 10/11 自带的 Microsoft Defender 已经足够好。它会在后台自动更新病毒库、扫描下载的文件、检查 U 盘。**你不需要安装第三方杀毒软件**，很多"免费杀毒"软件反而会拖慢电脑、弹广告、捆绑其他软件，并不比 Defender 更好。

## 五、养成这四个好习惯（长期受益）

1. **随时 Ctrl+S** — 编辑文档、写代码、做表格的时候，养成几秒就按一下 Ctrl+S 的习惯。很多崩溃发生在你忘记保存的时候。
2. **文件不乱放桌面** — 桌面只是中转站，不是仓库。用完的文件移动到合适的文件夹里。桌面塞满会拖慢系统，而且找东西很痛苦。
3. **不明链接不乱点** — 浏览器弹出"你的电脑有危险，立即扫描"的广告、邮件里有"你中了奖"的链接、QQ 里陌生人发来的文件——保持警惕。
4. **遇到问题先搜索** — 电脑出了小问题（软件报错、连不上网、某个功能不会用），把错误提示复制到搜索引擎或 AI 工具里搜索，大概率有人遇到过并给出了解决方案。这是每个电脑用户都应该掌握的技能。
