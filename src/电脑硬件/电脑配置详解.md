# 电脑配置详解：想深入了解时再读

> 阅读前提：你已经了解了 PC 的基本组成（参见《认识PC》）。本文是《认识PC》的延伸。如果你只是日常使用电脑，知道 CPU、内存、硬盘是什么就够了，不需要读这篇。当你想升级硬件、买新电脑、或者玩游戏需要了解配置时，再回来看。

## 补充说明：从《认识 PC》精简过来的内容

以下内容在主线中为了控制篇幅被精简了，在这里完整保留。

### CPU 指令是什么

CPU 的工作就是"从内存里读指令，然后执行它"。一条指令就是一件很简单的事，比如"把这两个数加起来"、"把这个数存到某个位置"、"如果这个数是零，就跳转到另一段指令去执行"。CPU 一秒钟能执行几十亿条这样的简单指令，靠这种极快的速度完成复杂任务。

### 硬盘详细对比

**机械硬盘（HDD）**：

- 里面有一个或多个旋转的圆形盘片，磁头在盘片表面读写数据，原理类似老式唱片机
- **优点**：容量大、价格便宜（1TB 约 200-300 元）
- **缺点**：速度慢（读写时需要等待盘片转到正确位置，这个时间叫"寻道时间"，大约几毫秒到十几毫秒），而且怕震动（磁头可能刮伤盘片）
- 适合存放不经常访问的大文件，比如电影、照片备份

**固态硬盘（SSD）**：

- 里面是闪存芯片，没有任何机械运动部件，原理和 U 盘类似但快得多
- **优点**：速度极快（比机械硬盘快 10-50 倍），安静、抗震
- **缺点**：同容量下比机械硬盘贵

### 显卡详细说明

**集成显卡**：集成在 CPU 内部，和 CPU 共享内存。性能够用但不强，适合办公、看视频、轻度游戏。

**独立显卡**：

- 台式机中是一块单独的板卡，可插拔更换
- 笔记本中则直接焊在主板上，出厂后无法升级
- 有自己的 GPU 芯片和显存（专门给显卡用的内存），性能远强于集成显卡
- 适合玩游戏、3D 建模、视频剪辑、AI 训练

目前独立显卡的两个主要品牌是 NVIDIA（GeForce RTX 系列）和 AMD（Radeon RX 系列）。

## 一、CPU 的更多细节

### 频率与核心数

CPU 的性能由多个因素决定，频率和核心数只是其中两个。

**频率**：CPU 内部的时钟速度，单位 GHz。频率越高，单核性能通常越强。但不同架构的 CPU 不能直接比频率——Intel 和 AMD 的设计不同，同样的频率下实际速度也不一样。

**核心数**：一个核心就像一个人干活。4 核能同时处理 4 件事，8 核能同时处理 8 件事。但很多事情没法拆成多个部分并行处理，所以核心数翻倍不等于速度翻倍。

**超线程**：多数现代 CPU 支持超线程，让一个物理核心模拟出两个逻辑核心。操作系统可以像调度两个独立核心一样调度它们，但两个逻辑核心会共享物理核心的部分资源。任务管理器里看到的"逻辑处理器"数通常是核心数的两倍（如果支持超线程）。

### 选购建议

- **轻度使用**（上网、办公、看视频）：i3/R3 或 i5/R5 足够（Intel 新机型对应 Core 3/5）
- **中度使用**（多标签浏览器、轻度图片编辑）：i5/R5 是合理选择（Intel 新机型对应 Core 5/7）
- **重度使用**（视频剪辑、3D 建模、编程编译）：i7/R7 或更高（Intel 新机型对应 Core 7/9 或 Core Ultra）
- **游戏**：多数游戏更吃单核性能，i5/R5 搭配好显卡比 i7/R7 配弱显卡更划算

## 二、内存的更多细节

### DDR4 与 DDR5

目前主流内存规格是 DDR4 和 DDR5：

- **DDR4**：成熟、便宜、兼容性好。3200 MHz 是常见频率。
- **DDR5**：更新、更快、更省电，但需要主板和 CPU 都支持。DDR5 已成为新电脑的主流选择，价格已与 DDR4 接近甚至在某些规格上更低。

对于大多数人，DDR4 完全够用。DDR5 的优势在重度生产力场景（大型视频剪辑、虚拟机）中更明显。

### 内存通道

主板支持双通道时，插两根内存条比插一根同容量的性能更好（带宽翻倍）。所以 2×8GB 通常比 1×16GB 更好。

## 三、主板的更多细节

### 尺寸规格

台式机主板有几种常见尺寸：

| 尺寸 | 大小 | 特点 |
|------|------|------|
| ATX | 30.5×24.4 cm | 标准大板，扩展插槽多 |
| Micro-ATX | 24.4×24.4 cm | 小板，够用，适合普通机箱 |
| ITX | 17×17 cm | 迷你板，适合小机箱，扩展性有限 |

笔记本主板没有统一标准，按机型定制，无法自由更换。

### 芯片组

主板芯片组决定了支持哪些 CPU、多少 USB 接口、能不能超频等。买主板时要确认芯片组和你的 CPU 匹配。

## 四、显卡的更多细节

### 集成显卡 vs 独立显卡

- **集成显卡**：集成在 CPU 内部，和 CPU 共享内存。性能够用，功耗低。Intel 的核显、AMD 的 APU 都属于这类。
- **独立显卡**：有自己的 GPU 芯片和显存（VRAM），性能远强于集成显卡，但功耗和发热也更大。

### 显存（VRAM）

显存是显卡专用的内存，存放正在处理的图像数据：

- **4GB**：轻度游戏、办公够用
- **8GB**：1080p 分辨率的主流游戏
- **12-16GB**：2K/4K 分辨率、大型 3A 游戏、AI 训练

显存不够时，游戏会卡顿或无法开启高画质设置。

### 显卡品牌与系列

- **NVIDIA**：GeForce RTX 系列（如 RTX 5060、5070、5080，上一代 4060、4070），数字越大通常越强
- **AMD**：Radeon RX 系列（如 RX 9070，上一代 RX 7600、7700、7800）

同代产品中，型号数字越大性能越强，价格也越高。

## 五、电源的更多细节

### 功率计算

电源功率需要大于整台电脑的最大功耗，还要留 20-30% 余量。

常见配置的参考：

| 配置 | 建议功率 |
|------|---------|
| 办公核显 | 300-400W |
| 主流游戏（独显） | 550-650W |
| 高端游戏/生产力 | 750-850W |

### 80 Plus 认证

电源的 80 Plus 认证表示转换效率：

- 白牌 / 铜牌 / 银牌 / 金牌 / 白金 / 钛金
- 等级越高，电能转换效率越高，发热越少，越省电

大多数人选铜牌或金牌就够了。

## 六、散热器的更多细节

### 风冷 vs 水冷

- **风冷**：金属散热片+风扇，简单可靠，维护方便。主流产品 100-300 元。
- **水冷**：液体循环散热，散热能力更强，噪音可能更低，但有漏液风险（虽然概率很低）。分一体式（出厂装好的，200-1000 元）和分体式（自己组装，极高端）。

大多数人风冷完全够用。水冷主要为了颜值或极端超频。

---

## 七、显示器的更多细节

### 刷新率

- **60Hz**：标准，日常够用
- **120Hz/144Hz**：游戏更流畅，滚动网页也更顺滑
- **240Hz 以上**：电竞需求，大多数人感知不明显

### 响应时间

像素从一种颜色切换到另一种颜色所需的时间。越低越好：

- **5ms 以上**：普通显示器，日常够用
- **1-3ms**：游戏显示器，减少拖影

### 面板类型

| 类型 | 特点 | 适合 |
|------|------|------|
| IPS | 色彩准、可视角度广 | 设计、日常 |
| TN | 响应快、便宜 | 竞技电竞 |
| VA | 对比度高、黑色纯 | 观影、单机游戏 |

## 八、键盘的更多细节

### 机械键盘轴体

机械键盘的核心是"轴体"，不同轴体手感不同：

| 轴体 | 特点 | 声音 |
|------|------|------|
| 红轴 | 线性、轻、无段落感 | 较小 |
| 茶轴 | 轻微段落感、万能 | 中等 |
| 青轴 | 强段落感、清脆 | 大 |
| 黑轴 | 线性、重、无段落感 | 较小 |

新手建议从红轴或茶轴开始。

## 九、查看配置的进阶工具

### DirectX 诊断工具（dxdiag）

当你需要确认显卡的**具体型号、显存大小、驱动版本**时，dxdiag 比任务管理器更详细。

**打开方法：** 按 `Win+R` → 输入 `dxdiag` → 回车 → 在弹出的窗口中点击"是"（检查驱动程序签名）。

**主要看这两个选项卡：**

**系统选项卡**（打开时默认显示）：

- 操作系统名称和版本
- 处理器（CPU 型号和频率）
- 内存（RAM 总容量）
- DirectX 版本（如 DirectX 12）

**显示选项卡**（如果有独立显卡，会有多个显示选项卡）：

- 显卡名称和制造商
- 显存（显示内存）
- 当前显示模式（分辨率、刷新率）
- 驱动程序版本和日期

这个工具信息最全，适合装机后确认所有硬件都能正常识别，或排查游戏、图形相关问题时查看。

> dxdiag 的"输入"选项卡可以看到键盘、鼠标、手柄信息，"声音"选项卡可以看到声卡信息，日常使用较少用到。

### 系统信息（图形化查看硬件信息）

按 `Win` 键打开开始菜单，直接输入"系统信息"，回车打开。这是一个图形化工具，左侧有分类树，右侧显示详细信息。

**适合查看：**
- 操作系统名称和版本
- 处理器（CPU 型号）
- 已安装的物理内存（RAM 总量）
- BIOS 模式（UEFI 或传统）
- 主板制造商和型号

比任务管理器信息更全，又比命令行工具直观，是查看整机配置的首选图形化工具。

### winver（快速看 Windows 版本）

按 `Win+R` → 输入 `winver` → 回车。弹出一个简洁的窗口显示 Windows 版本和操作系统版本号，比"关于"页面更直接。

### systeminfo 命令

按 `Win+R` → 输入 `cmd` → 回车 → 在黑色窗口中输入 `systeminfo`。

会列出操作系统的详细安装信息、网络配置、已安装的补丁列表等。找不到图形界面入口时，可以用它来补充。

### PowerShell 的 Get-ComputerInfo

在 PowerShell 中输入 `Get-ComputerInfo` 会输出极其详尽的系统信息（比 systeminfo 更全）。还可以只查某项，比如 `Get-ComputerInfo -Property "Os*"` 只查操作系统相关信息。

### BIOS/UEFI 中查看硬件

开机时按特定键（Del、F2、F10 等，不同主板不同）可以进入系统固件设置界面，里面也能看到 CPU、内存、硬盘信息。

**注意：只查看，不要随意修改任何设置，否则可能导致系统无法正常启动。**

### 第三方工具

如果 Windows 自带工具给不了你想要的细节，可以尝试：

- **CPU-Z**：看 CPU、主板、内存的详细参数（如内存通道数、核心频率）
- **GPU-Z**：看显卡的详细参数（如核心频率、温度）
- **Speccy**：看整机配置概览

**建议先用熟 Windows 自带工具，再尝试第三方工具。**

> 不管用哪种方法，遇到看不懂的参数项，最简单的做法就是：把那个词复制到搜索引擎或 AI 工具里搜索。比如看到"逻辑处理器"不明白，搜索"逻辑处理器是什么意思"，比死记硬背强得多。

## 十、查看工具速查表

| 你想看什么 | 用哪个工具 | 怎么打开 |
|-----------|-----------|---------|
| CPU 型号、内存大小、显卡型号（快速看） | 任务管理器 → 性能 | `Ctrl+Shift+Esc` |
| 整机硬件信息（图形化） | 系统信息 | 按 `Win` 键 → 输入"系统信息" → 回车 |
| Windows 版本、系统类型（32/64 位） | 设置 → 关于 | 右键开始 → 系统 → 关于 |
| 只看 Windows 版本号 | winver | `Win+R` → `winver` |
| 硬盘还剩多少空间 | 此电脑 | `Win+E` |
| 显卡驱动版本、显存大小、DX 版本 | DirectX 诊断工具 | `Win+R` → `dxdiag` |
| 操作系统详细安装信息、补丁列表 | systeminfo | `Win+R` → `cmd` → `systeminfo` |
| 极详尽的系统信息 | Get-ComputerInfo | PowerShell 中输入 |
| CPU/主板/内存详细参数 | CPU-Z | 第三方工具 |
| 显卡详细参数 | GPU-Z | 第三方工具 |

## 十一、什么时候需要看这篇

| 场景 | 需要了解什么 |
|------|-------------|
| 买新电脑 | CPU 型号含义、内存容量、显卡性能级别 |
| 升级硬件 | 主板支持什么 CPU、内存插槽有几个、电源功率够不够 |
