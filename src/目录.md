## 完整路线（12 章）

| | 章节 | 内容 |
|:-:|------|------|
| **1** | [认识 PC](./认识PC.md) | 硬件组成（CPU、内存、硬盘、主板、显卡、电源、散热器）与外设 |
| **2** | [基本操作](./基本操作.md) | 开关机、桌面与窗口操作、鼠标键盘快捷键、中文输入 |
| **3** | [日常使用](./日常使用.md) | 文件管理、网络连接、软件安装卸载、账户安全与好习惯 |
| **4** | [浏览器入门](./简单操作/浏览器入门.md) | 标签页管理、地址栏搜索、下载、书签、隐私模式、扩展 |
| **5** | [看穿那些套路](./安全意识/识别常见骗术.md) | 虚假弹窗、钓鱼链接、虚假下载站、社交工程 |
| **6** | [防患于未然](./安全意识/日常安全习惯.md) | 密码策略、官方渠道下载、系统更新、备份 |
| **7** | [理解你的操作系统](./理解你的操作系统.md) | 操作系统原理、任务管理器、文件系统、进程/服务/驱动 |
| **8** | [理解你的网络](./理解你的网络.md) | IP 地址与端口、DNS 解析、路由器、带宽与延迟 |
| **9** | [命令行入门](./简单操作/命令行入门.md) | 命令提示符打开方式、常用命令、管理员身份运行 |
| **10** | [常见故障排查思路](./简单操作/常见故障排查思路.md) | 程序无响应、通用排查原则、什么时候该找人 |
| **11** | [查看电脑配置](./查看电脑配置.md) | 用任务管理器、dxdiag 等工具查看硬件 |
| **12** | [Windows 存储位置](./深入存储知识/Windows%20存储位置.md) | C 盘目录结构、用户文件夹、AppData、磁盘清理 |
