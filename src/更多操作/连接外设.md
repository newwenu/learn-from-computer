# 连接外设：从 U 盘到显示器

> 阅读前提：你已经了解了 PC 的基本硬件组成和基本操作（参见[《认识 PC》](../认识PC.md)和[《基本操作》](../基本操作.md)），并知道怎么在文件资源管理器中查看和管理文件（参见[《日常使用》](../日常使用.md)）。本文讲的是各种外设怎么连上电脑、连不上怎么排查。

---

## 一、电脑怎么认识一个新设备

你在电脑上用的每一样东西——键盘、鼠标、U 盘、耳机、显示器——都是**外设**（外部设备）。它们不是电脑主机的一部分，而是通过线缆或无线信号连到电脑上，让电脑能接收你的输入或把结果呈现给你。

笔记本自带了屏幕、键盘、触控板、扬声器、摄像头和麦克风，开盖即用。但当你想插个 U 盘传文件、连个蓝牙耳机听歌、或者外接一台大显示器办公时，就需要了解"外设怎么连上电脑"这件事。

大多数外设连接的背后，遵循同一个基本流程：

1. **物理连接**——通过 USB 线、HDMI 线插入接口，或者通过蓝牙无线配对
2. **系统识别**——Windows 检测到有新设备接入，尝试加载对应的驱动程序
3. **正常使用**——驱动加载成功后，设备就可以用了
4. **出问题排查**——设备没反应时，用设备管理器查看原因

其中第 2 步是关键：**驱动程序**是让操作系统与硬件设备通信的软件（就像硬件的翻译官）。有些设备的驱动 Windows 自带，插上就能用；有些需要额外安装驱动才能正常工作（比如某些打印机、手写板）。关于驱动的更多解释，可以参见[《理解你的操作系统》](../理解你的操作系统.md)的"为什么装了 Windows 还要装驱动"一节。

下面按连接方式分类讲解。

---

## 二、USB 设备——插上就能用的最多数

USB 是目前最常见的外设接口。U 盘、鼠标、键盘、摄像头、移动硬盘，大多数都通过 USB 连接。

### 插上后发生了什么

把 USB 设备插入电脑的 USB 接口后，Windows 会自动检测并尝试加载驱动。大多数常见设备（U 盘、鼠标、键盘）的驱动 Windows 自带，这个过程通常只需要几秒钟。如果听到系统发出"设备已连接"的提示音，说明识别成功了。你会看到：

- **U 盘**：任务栏右下角弹出提示，文件资源管理器里出现新的盘符
- **读卡器**：一种把 SD 卡（相机常用）或 TF 卡（也叫 microSD 卡，手机常用）转成 USB 接口的小设备。使用方式跟 U 盘一样：把卡插入读卡器，再把读卡器插到电脑上
- **鼠标**：插上后几秒内就能移动光标
- **键盘**：插上后直接可以打字

### 手机连接电脑

用 USB 数据线把手机连到电脑，可以传输照片、备份文件。但跟 U 盘不同，手机连上电脑后**不会自动变成一个盘符**——你需要在手机上做一步确认：

1. 用数据线连接手机和电脑
2. 手机屏幕上会弹出一个提示，问你是否允许通过 USB 访问数据。选择**"传输文件"**或**"传输媒体"**（有的手机叫"文件传输"或"MTP 模式"，即媒体传输协议，是一种让电脑访问手机文件的方式）。如果选成了"仅充电"，电脑上是看不到手机文件的
3. 电脑上打开文件资源管理器，此时会出现手机的名字（如"Galaxy S23"或"Redmi Note"），点击进入就能看到手机里的文件

如果插上后电脑没反应，先检查手机屏幕上有没有弹出确认提示——有些手机的弹窗会被系统自动关闭或被你无意中划掉了。拔掉数据线重新插一次，留意手机屏幕。

> 不同品牌的手机，手机上的文件管理应用里显示的文件夹结构可能不同。照片通常在 `DCIM` 或 `Pictures` 文件夹里。

### "安全删除硬件"什么时候用

在文件资源管理器中，U 盘或移动硬盘旁边有一个弹出图标（向上的小三角下面一条横线）。点击它，等提示"安全移除硬件"后再拔出设备。

这样做是为了确保所有数据已经写入完毕。虽然 Windows 默认允许直接拔出 U 盘（不需要先弹出），但更稳妥的做法还是先点击弹出再拔——尤其是当你刚往 U 盘里复制完文件、不确定是否写完的时候。如果弹出时提示"设备正在使用中"，关掉所有可能正在访问 U 盘的程序（比如你打开的 Word 文档、图片查看器等），再试一次。

### USB 常见问题

**插上没反应？** 按这个顺序排查：

1. **换个 USB 接口试试**——台式机前面板的 USB 接口有时供电不足，试试机箱背面的接口；笔记本则换另一个 USB 接口
2. **确认插紧了**——USB 接口有正反方向，Type-A 接口（常见的长方形 USB 口）插不进去时翻过来再试——硬插可能损坏接口
3. **换一台电脑试试**——如果在别的电脑上也没反应，可能是设备本身的问题

**台式机前面板 vs 后面板：** 台式机机箱正面的 USB 接口（前面板）通过内部线缆连接到主板，供电和稳定性可能不如背面的接口（后面板，直接焊在主板上）。如果移动硬盘插前面板识别不到或频繁断开，插到后面板试试。

### 有线音频设备

部分台式机和旧款笔记本有 3.5mm 音频接口（圆形小孔，旁边通常有个耳机图标），可以插有线耳机或音箱。插上后 Windows 会自动识别，声音默认从耳机/音箱输出。

**注意：** 现在很多轻薄本已经取消了 3.5mm 接口。如果你的笔记本没有这个接口，可以用 Type-C 转 3.5mm 的转接头（几块钱），或者直接用蓝牙耳机。

如果插了耳机但声音还是从扬声器出来，在"设置 → 系统 → 声音"中检查输出设备是否选对了。

### 扩展坞：接口不够时的解决方案

很多轻薄本只有一两个 USB 接口，不够用。**扩展坞**（也叫 USB Hub）可以把一个接口扩展成多个——插上扩展坞后，你可以同时连接 U 盘、鼠标、键盘，甚至外接显示器。

选购扩展坞时注意两点：
- 接口类型要跟你的电脑匹配（常见的有 USB-A 和 Type-C 两种插头）
- 如果你需要通过扩展坞外接显示器，确认扩展坞支持视频输出（不是所有扩展坞都支持）

## 三、蓝牙设备——无线连接

蓝牙是一种短距离无线通信技术，适合连接耳机、鼠标、键盘、音箱等不需要插线的设备。

### 打开蓝牙

蓝牙开关在两个地方可以找到：

- **设置**：按 `Win+I` 打开设置（也可以点击开始菜单→"设置"图标）→"蓝牙和其他设备"→ 确认蓝牙已打开
- **任务栏快捷方式**：点击任务栏右下角的网络/音量图标区域，在弹出的面板中找到蓝牙图标，点击打开或关闭 `[查]`

### 配对设备

蓝牙设备第一次连接时需要**配对**——让电脑和设备互相认识。配对之后，以后只要两者都开着蓝牙，就会自动连接。

1. 确认蓝牙设备已开启且处于**可被发现**状态（不同设备操作不同：耳机通常是长按电源键几秒直到指示灯闪烁，鼠标一般是底部有个配对按钮。具体看设备说明书）
2. 在电脑上打开"设置 → 蓝牙和其他设备"，点击"添加设备"
3. 选择"蓝牙"，等几秒，列表里会出现附近可发现的蓝牙设备
4. 点击你要连接的设备名称，按提示确认配对（有些设备会显示一个配对码，确认跟设备上显示的一致后点击"连接"）
5. 显示"已连接"就成功了

> **试一试：** 如果你手边有蓝牙耳机或蓝牙鼠标，现在试试把它连到电脑上。打开"设置 → 蓝牙和其他设备"，按上面的步骤操作。

### 已配对设备的管理

配对过的设备会出现在"设置 → 蓝牙和其他设备"的列表中。下次打开两者蓝牙时，通常会自动连接。

- **想断开但不想删除**：在列表中点击设备 →"断开"。下次想用的时候再点"连接"
- **想彻底删除**：在列表中点击设备 →"删除设备"。删除后如果还想用，需要重新配对
- **自动重连没生效**：确认电脑和设备的蓝牙都开着，距离不要太远（通常 10 米以内）。如果还是连不上，删除后重新配对

### 蓝牙连不上的排查

1. 确认电脑蓝牙已打开（设置 → 蓝牙和其他设备）
2. 确认设备电量充足且已开机
3. 确认设备处于可被发现状态（有些设备开机后几秒内才能被发现，错过了可以重新操作进入配对模式——通常关掉再开，或长按配对键）
4. 删除配对记录，重新配对
5. 如果电脑完全没有蓝牙功能（有些台式机没有蓝牙模块），可以买一个 USB 蓝牙适配器（几十元），插到 USB 接口上就能用

## 四、外接显示器与投影——需要额外配置的外设

前面讲的 USB 和蓝牙设备，插上或配对后基本就能直接用了。但外接显示器和投影仪不一样——它们连上线之后，还需要告诉 Windows 你想怎么用这块屏幕。

### 4.1 Win+P：四种显示模式

按下 `Win+P`（也可以在桌面空白处右键→"显示设置"中找到多显示器选项），屏幕右侧会弹出四个选项：

| 模式 | 效果 | 适用场景 |
| ---- | ---- | -------- |
| 仅电脑屏幕 | 只有笔记本屏幕亮，外接设备不显示 | 只需使用笔记本自带屏幕时主动选用 |
| 复制 | 两个屏幕显示完全相同的内容 | 做演示、接投影仪 |
| 扩展 | 两个屏幕合起来形成一个更大的桌面 | 日常办公：一边写文档一边看资料 |
| 仅第二屏幕 | 笔记本屏幕关闭，只用外接显示器 | 外接大屏办公，或笔记本屏幕已损坏 |

**操作建议：** 插上线后按 `Win+P`，选"扩展"或"复制"。如果没反应，等几秒再按一次。部分投影仪或显示器需要手动切换输入源（用显示器上的按键选 HDMI 或 VGA）。

> **试一试：** 如果你手边有外接显示器或投影仪，试试按 `Win+P` 切换不同模式，观察两个屏幕的变化。

### 4.2 扩展模式下调整排列

选"扩展"后，两个屏幕各显示桌面的一部分。默认情况下，外接显示器的排列位置可能和实际摆放不符——比如笔记本在左边、外接显示器在右边，但系统把外接显示器排到了左边。这时鼠标移到屏幕边缘时会从"错误的一边"出去。

**调整方法：**

1. 桌面空白处右键 →"显示设置"
2. 你会看到两个标了数字的方块，每个代表一个屏幕。数字 1 通常对应笔记本主屏幕
3. 用鼠标拖动方块，让它们的位置和你的实际摆放一致（笔记本在左、显示器在右，就把方块 1 拖到左边、方块 2 拖到右边）
4. 点击"应用"确认

**设置主屏幕：** 在显示设置中点击某个屏幕 → 勾选"使其成为主显示器"。主显示器会显示任务栏的开始按钮和系统托盘，新打开的程序默认出现在主显示器上。

**调整分辨率与缩放：** 如果外接显示器显示模糊或图标太大：

1. 在显示设置中点击对应的屏幕方块
2. 先调整"缩放"比例（高分屏笔记本外接普通显示器时，系统可能误用 150% 或 200% 缩放导致图标偏大，可尝试改为 100%）
3. 再向下滚动到"显示分辨率"，选择系统推荐的（标注了"推荐"）分辨率

### 4.3 合盖使用外接显示器

把笔记本接到显示器后，如果一合上盖子电脑就睡眠了，外接显示器也黑了。这是因为合盖的默认行为是"睡眠"，需要手动修改。

按 `Win+R` 打开"运行"对话框（也可以右键点击"开始"按钮→"运行"），输入 `control` 回车打开控制面板（也可以在开始菜单搜索"控制面板"打开）→"硬件和声音"→"电源选项"→ 左侧"选择关闭盖子的功能" `[查]`

在弹出的窗口中，把"关闭盖子"的行为改为"不采取任何操作"，然后点击"保存修改"。

> 修改电源设置属于系统级操作，看不懂步骤时先停手，搜索图文教程确认后再操作。
>
> 改成"不采取任何操作"后，合上盖子不会睡眠。如果把笔记本放进包里，它可能会持续发热并消耗电量。建议只在固定工位使用外接显示器时采用此设置，移动前把合盖行为改回"睡眠"。



### 4.4 连接线接口说明

除了显示模式，连接显示器的线材类型也需要了解。外接显示器需要一根视频线连接笔记本和显示器。常见的接口类型：

| 接口 | 说明 | 常见于 |
| ---- | ---- | ------ |
| HDMI | 最常见的视频接口，传输画面和声音 | 笔记本、显示器、投影仪、电视 |
| DisplayPort（DP） | 性能和 HDMI 相当，部分显示器只有 DP | 台式机显卡、高端显示器 |
| VGA | 老式接口，只传画面不传声音 | 旧投影仪、旧显示器 |
| Type-C | 部分支持视频输出（不是所有 Type-C 口都能输出视频，购买前确认） | 轻薄本、新显示器 |

如果不确定自己的笔记本有什么接口，在搜索引擎或 AI 工具中搜索你的笔记本型号 + "接口"即可。买线或转接头时确认两端的接口类型匹配。

### 4.5 无线投屏

如果你的投影仪或电视支持无线投屏，可以不用插线，直接把笔记本画面投过去。

1. 确保笔记本和显示设备（电视/投影仪）连接在同一个 Wi-Fi 网络下
2. 按 `Win+K` 打开投屏面板（也可以打开"设置 → 蓝牙和其他设备 → 添加设备"，选择"无线显示器或扩展坞" `[查]`）
3. 系统会搜索附近可用的无线显示设备，点击你要投屏的设备名称
4. 如果电视/投影仪上弹出确认提示，点击"允许"或"接受"
5. 连接成功后，选择"复制"或"扩展"模式（和 `Win+P` 的逻辑一样）

**无线投屏的限制：**

- **延迟**：无线传输有延迟，不适合打游戏或播放对同步要求高的内容，演示 PPT、看视频基本够用
- **同一 Wi-Fi**：笔记本和显示设备需要在同一个局域网内
- **兼容性**：部分老旧投影仪不支持无线投屏；部分电视虽然支持，但需要先在电视设置里打开"屏幕镜像"或"无线显示"功能
- **稳定性**：Wi-Fi 信号不好时，画面可能卡顿或断开

如果 `Win+K` 搜不到设备，先检查电视/投影仪的无线显示功能是否开启，或重启两台设备再试。

### 4.6 外接显示器常见问题

#### 检测不到外接显示器

插上线、按了 `Win+P`，但外接显示器或投影仪始终黑屏、提示"无信号"——按以下顺序排查：

1. **检查线材是否插紧**——两端都确认插牢：笔记本端的接口和显示器/投影仪端的接口。稍微拔出来重新插一次，听到"咔哒"声或感觉到阻力确认插紧即可
   > 如果是 Type-C 转 HDMI/DP 的转接线或扩展坞，确认转接头也插紧了。有些转接头比较松，动一下就会断开。
2. **确认接口支持视频输出**——不是所有 Type-C 口都能输出画面。搜索你的笔记本型号 + "Type-C 视频输出"确认。如果不支持，你需要买一根从笔记本其他视频接口（如 HDMI）出的线
3. **手动切换显示器的输入源**——显示器或投影仪通常有多个接口（HDMI 1、HDMI 2、DP、VGA），需要用机身按键或遥控器手动选择当前接入的接口。如果显示器提示"无信号"，先按显示器上的菜单键，找到"输入源"或"信号源"，切换到对应的接口
4. **更新显卡驱动**——如果以上步骤都试过还是不行，可能是显卡驱动有问题。更新驱动的方法：在设备管理器中展开"显示适配器"→ 右键你的显卡 →"更新驱动程序"→ 选择"自动搜索驱动程序"。如果自动搜索找不到，建议去笔记本品牌官网下载对应型号的驱动，或找人帮忙 `[查]`

> 更新显卡驱动属于系统级操作，看不懂步骤时先停手，让懂的人帮忙或搜索图文教程跟着做。

#### 画面闪烁或间歇黑屏

插线后画面一直闪烁或间歇性黑屏，通常是线材或接口接触不良。换一根线试试，或换个接口。

#### 屏幕方向旋转了

有时候外接显示器的画面突然横过来了，或者倒过来了——这通常是不小心按到了显卡驱动的旋转快捷键（如 `Ctrl+Alt+方向键`）。

**解决办法：** 桌面空白处右键 →"显示设置"→ 找到"显示方向"（在"缩放与布局"下方）→ 从"纵向""横向（翻转）"等选项中选回"横向"。

## 五、设备管理器——排查外设问题的通用工具

当你遇到"设备插上了但没反应""设备管理器里有黄色感叹号"这类问题时，**设备管理器**是最常用的排查工具。它列出了电脑上所有硬件设备的状态——相当于所有设备的"户口本"。

### 在哪打开

右键点击"开始"按钮（Windows 图标）→ 搜索"设备管理器" `[查]`，或者按 `Win+X`（也可以右键点击"开始"按钮）然后选择"设备管理器"。

### 黄色感叹号意味着什么

如果某个设备旁边有**黄色三角感叹号**，通常表示驱动程序有问题——可能是驱动缺失、损坏、或不兼容。这是设备不能正常工作的最常见原因。

**处理方法：** 右键点击带感叹号的设备 →"更新驱动程序"→ 选择"自动搜索驱动程序"。Windows 会在本地和 Windows Update 中搜索合适的驱动。

如果自动搜索找不到，可以尝试：
- 去设备厂商官网下载驱动（搜索"设备型号 + 驱动"）
- 去笔记本品牌官网下载（如果你用的是笔记本，品牌官网通常有整套驱动）
- 右键设备 →"卸载设备"，然后重启电脑，Windows 会重新尝试安装驱动

> 更新驱动属于系统级操作，看不懂步骤时先停手，让懂的人帮忙或搜索图文教程跟着做。

### 用设备管理器排查的通用思路

不管是什么外设出了问题，用设备管理器排查的思路是一样的：

1. **打开设备管理器**，找到出问题的设备类别（比如"声音、视频和游戏控制器"里找耳机，"通用串行总线控制器"里找 USB 设备）
2. **看有没有黄色感叹号**——有，说明驱动有问题，按上面的方法更新驱动
3. **没有黄色感叹号但设备还是不工作**——说明驱动没问题，问题可能在别处：
   - 摄像头或麦克风用不了 → 可能是隐私权限没开（见下方）
   - 耳机没声音 → 可能是输出设备选错了（在"设置 → 系统 → 声音"中检查输出设备）
   - 蓝牙设备连不上 → 回到第三章的排查步骤
4. **设备根本没出现在列表里**——说明系统没有检测到这个设备。检查物理连接（换接口、换线、重新配对），确认设备已开机

### 摄像头或麦克风用不了

这是"设备管理器里看着正常但设备不工作"的典型例子。如果摄像头和麦克风在设备管理器里没有黄色感叹号，但某个应用里用不了，可能是**隐私权限**的问题：

1. 打开"设置 → 隐私和安全性"（Windows 10 叫"隐私"）`[查]`
2. 找到"摄像头"或"麦克风"
3. 确认"允许应用访问摄像头/麦克风"已打开
4. 确认你正在使用的应用在列表中也被允许了

## 六、打印机与扫描仪

前面讲的 USB 设备、蓝牙设备、外接显示器，大多属于连上就能直接使用的类型。打印机和扫描仪稍有不同——它们也是外设，但涉及一些额外的步骤和概念。

### 6.1 打印机和其他外设的区别

| 对比项 | 普通 USB 外设（如 U 盘、鼠标） | 打印机 |
| ---- | ---- | ---- |
| 驱动来源 | Windows 自带，插上就用 | 通常需要安装品牌专用驱动 |
| 配置复杂度 | 几乎为零 | 可能涉及网络配置、打印首选项 |
| 问题排查 | 换接口、重启即可 | 可能有队列卡住、脱机状态等问题 |
| 标准化程度 | 高（USB 协议统一） | 低（各品牌差异大） |

不过这张表只是说明打印机比 U 盘多了几个环节，基础打印操作本身并不复杂。

### 6.2 打印队列：点了打印后发生了什么

你在 Word 里点了 `Ctrl+P`（也可以点击左上角"文件"→"打印"）再点"打印"，文档不会立刻从打印机里出来。中间经过了一个叫**打印队列**（Print Spooler，也叫打印后台处理程序）的环节。

流程是这样的：

1. 你点击"打印"后，Word 等软件把打印任务交给 Windows
2. Windows 把任务放进打印队列，排队等待
3. 当打印机空闲时，Windows 把任务逐页发给打印机
4. 打印机开始打印

为什么要有队列？因为打印机打印的速度远慢于电脑生成打印任务的速度。如果没有队列，每次打印都得等着打印机打完上一页才能发下一页，电脑会一直等着没法做别的事。有了队列，电脑可以立刻返回去做别的事，打印在后台慢慢完成——跟你在餐厅点完餐后拿号等候，不用一直站在柜台前等一个道理。

**队列卡住了怎么办：** 有时候你会遇到"点了打印但打印机没反应"的情况，可能是队列卡死了。最简单的办法是重启电脑（队列会随系统重启而重置）。如果想更精确地操作：打开"设置"→"蓝牙和其他设备"→"打印机和扫描仪" `[查]`，找到你的打印机，点击"打开队列"，可以看到当前排队的任务，选择取消卡住的任务即可。

### 6.3 连接打印机

#### USB 直连（推荐初次使用）

USB 直连通常比 Wi-Fi 更稳定，适合初次连接打印机：

1. 用 USB 线连接打印机和电脑
2. Windows 自动检测到新设备
3. 如果 Windows 自带有这款打印机的驱动，会自动安装完成
4. 如果没有，Windows 会提示你去哪里下载（或自动联网搜索驱动）
5. 安装完成后，在"设置"→"蓝牙和其他设备"→"打印机和扫描仪" `[查]` 中应该能看到这台打印机

手动安装打印机驱动时会弹出用户账户控制（UAC）窗口（背景变暗，问你是否允许），如果是你自己主动连的打印机，点"是"即可。UAC 的原理详见[《用户账户控制与权限》](../权限知识/用户账户控制与权限.md)。

#### Wi-Fi 网络打印机

如果你不想插线，很多新款打印机也支持 Wi-Fi 连接，不用插线。基本流程：

1. 先让打印机自己连上 Wi-Fi（每种打印机的操作不同，通常是在打印机屏幕上设置，或通过手机 App 配置。具体请查阅打印机说明书或搜索"型号 + Wi-Fi 配置"）
2. 确认打印机和电脑在**同一个 Wi-Fi 网络**下
3. 电脑端："设置"→"蓝牙和其他设备"→"打印机和扫描仪" `[查]` →"添加设备"
4. Windows 会搜索同一网络下的打印机，找到后点击连接

网络打印机的配置因品牌和型号差异较大。如果自动搜索找不到，建议搜索"你的打印机型号 + Win10/Win11 安装教程"跟着图文步骤操作。看不懂的步骤先停手，搜索确认后再继续。

#### Microsoft Print to PDF（虚拟打印机）

Windows 自带一个特殊的"打印机"，它不输出到纸张，而是生成 PDF 文件：

- 在任意程序的打印对话框中，打印机列表里选择"Microsoft Print to PDF"
- 点击打印后，会让你选择保存位置和文件名
- 生成的 `.pdf` 文件可以用浏览器或 PDF 阅读器打开

用途包括：电子存档、发送给别人（对方无需安装你用的软件就能查看）、提交电子版作业。

> **试一试：** 打开记事本写几行字，按 `Ctrl+P`（也可以点击"文件"→"打印"），在打印机列表中选择"Microsoft Print to PDF"，点击打印——你会得到一个 PDF 文件。

### 6.4 扫描文件

如果你的打印机是一体机（打印+扫描+复印），通常安装驱动时会一并安装扫描软件。

**用 Windows 自带工具扫描：**

1. 开始菜单搜索"传真和扫描"并打开
2. 顶部菜单选择"新建"→"扫描"
3. 点击"扫描"，完成后保存为图片或 PDF

**用打印机自带的扫描软件：**

各品牌的扫描软件界面不同，但基本功能类似：预览→调整区域→扫描→保存。部分一体机的配套软件支持把图片中的文字转成可编辑文本。

如果找不到扫描软件，先确认打印机驱动是否完整安装。有时基础驱动只包含打印功能，需要去品牌官网下载完整驱动才能使用扫描功能。

### 6.5 打印机常见问题

| 问题 | 可能原因 | 做法 |
| ---- | ---- | ---- |
| 点了打印没反应 | 默认打印机选错 / 打印机脱机 | 打印对话框检查打印机名称；检查打印机电源和网络 |
| 打印卡在队列不动 | 队列卡死 | 取消所有任务，重启电脑（最简单的办法） |
| 显示"打印机脱机" | 打印机休眠或断网 | 检查打印机电源按钮；网络打印机确认 Wi-Fi 连接 |
| 打印出来是乱码 | 驱动型号不对 | 卸载当前驱动，重新安装正确型号的驱动 |
| 只打印一半 | 驱动问题或文档异常 | 更新驱动；尝试打印其他文档 |
| 网络打印机搜不到 | 不在同一网络 / 防火墙拦截 | 确认电脑和打印机在同一 Wi-Fi 下；临时关闭防火墙试试 |
| 打印颜色异常 | 墨水/碳粉耗尽 | 检查墨水/碳粉余量，及时更换 |

打印机的品牌（HP、Canon、Epson、Brother 等）和型号非常多，具体驱动的下载地址和安装步骤各不相同。遇到上述表格解决不了的问题，建议搜索"打印机型号 + Win10/Win11 + 问题描述"查找针对性教程。

## 七、常见问题速查

| 现象 | 可能原因 | 做法 |
| ---- | ---- | ---- |
| U 盘插上没反应 | 接口松动或供电不足 | 换个 USB 接口；台式机试试后面板 |
| 手机连电脑没反应 | 手机端未确认传输模式 | 拔掉重插，留意手机弹窗，选"传输文件" |
| 插了耳机声音还是从扬声器出来 | 输出设备选错了 | 设置 → 系统 → 声音 → 切换输出设备 |
| 蓝牙搜不到设备 | 蓝牙没开或设备没进配对模式 | 检查两端，确认设备可被发现 |
| 蓝牙配对后连不上 | 配对记录异常 | 删除配对记录，重新配对 |
| 外接显示器没画面 | 未切换显示模式 | 按 `Win+P` 选"扩展"或"复制" |
| 外接显示器画面模糊 | 分辨率或缩放不符 | 显示设置中选"推荐"分辨率 |
| 鼠标移到屏幕边缘过不去 | 排列位置不匹配 | 显示设置里拖动屏幕方块调整 |
| 合盖后外接显示器黑屏 | 合盖行为设为"睡眠" | 电源设置中改为"不采取任何操作" |
| 插线后画面闪烁或间歇黑屏 | 线材或接口接触不良 | 换一根线试试，或换个接口 |
| 检测不到外接显示器 | 线材松动、接口不支持视频输出、驱动问题 | 参见 4.6"检测不到外接显示器" |
| 外接显示器画面横过来/倒过来 | 误触了旋转快捷键 | 显示设置 → 显示方向 → 选回"横向" |
| 设备管理器有黄色感叹号 | 驱动缺失或损坏 | 右键设备 → 更新驱动程序 |
| 摄像头/麦克风用不了 | 隐私权限未授予 | 设置 → 隐私 → 允许应用访问 |
| `Win+K` 搜不到电视/投影仪 | 设备未开启无线显示或不在同一 Wi-Fi | 检查电视/投影仪设置，确认同一网络 |

---

这一章的核心思路：**外设连接的本质是"物理连接 → 系统识别 → 配置使用 → 排查问题"**。USB 设备通常即插即用，蓝牙设备需要先配对，外接显示器需要额外选择显示模式，打印机则需要安装专用驱动并理解打印队列的概念。每种外设的常见问题在各自的章节里就能找到答案；如果遇到没见过的设备，打开设备管理器，按"看感叹号 → 更新驱动 → 检查权限"的通用思路排查即可。
