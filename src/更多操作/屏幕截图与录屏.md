# 屏幕截图与录屏：把屏幕上的画面保存下来

> 阅读前提：你已经掌握了基本操作（参见《基本操作》），知道键盘快捷键怎么按。

---

## 一、什么时候用到截图和录屏

截图和录屏是日常使用频率很高的技能。常见场景：

- 看到一段有用的信息，想保存下来但不方便复制文字
- 软件报错了，把错误界面截下来发给别人帮忙看
- 想把游戏里的精彩时刻录下来
- 做教程或演示时，录下操作步骤给别人看

Windows 自带了截图和录屏工具，不需要安装任何第三方软件。

---

## 二、截图：四种方式对应不同需求

### 1. 全屏截图——Print Screen

按下键盘上的 `Print Screen` 键（有的键盘上标的是 `PrtScn` 或 `PrtSc`），Win11 默认会打开截图工具栏，与 `Win+Shift+S` 效果相同。Win10 默认是复制全屏到剪贴板。

如果按了没反应，可前往「设置 → 辅助功能 → 键盘」开启「使用 Print Screen 键打开屏幕截图」。

### 2. 直接截全屏并保存为文件——Win+Print Screen

按 `Win+Print Screen`，屏幕会闪一下（变暗再恢复），截图会自动保存到 **`此电脑 → 图片 → 屏幕截图`** 文件夹里，文件按"屏幕截图 (1).png""屏幕截图 (2).png"依次编号。

这种方式最适合需要保存截图留底的场景——截完直接变成文件，不需要手动粘贴保存。笔记本电脑可能需要同时按 `Fn` 键才能触发（因为 `Print Screen` 可能和别的功能共用按键）。

### 3. 截取任意区域——Win+Shift+S（截图工具）

按 `Win+Shift+S`（或 `Print Screen`，效果相同），屏幕顶部会出现截图工具栏，提供以下截图功能：

| 功能 | 作用 |
| --- | --- |
| 矩形截图 | 按住鼠标拖出一个矩形区域，最常见 |
| 任意形状截图 | 手绘一个不规则形状来截图 |
| 窗口截图 | 点击某个窗口，自动截取该窗口内容 |
| 全屏截图 | 截取整个屏幕 |

选好区域后，截图自动复制到剪贴板。**Win11 默认同时自动保存到 `图片\屏幕截图` 文件夹**；Win10 早期版本只复制到剪贴板，需要点击右下角弹出的通知才能保存为文件。点击通知或直接点击工具栏的编辑按钮，可以对截图进行标注、裁剪等操作。

这是最灵活的截图方式，日常使用频率最高。记不住这几种也没关系，默认就是矩形截图，拖选区域即可。

### 4. 只截当前窗口——Alt+Print Screen

按 `Alt+Print Screen`，只把当前激活的窗口画面复制到剪贴板，背景不会被截进去。适合只需要某个窗口内容、不想手动裁剪的场景。

## 三、截好的图去哪了

| 按键 | 去哪了 | 自动保存文件？ |
| --- | --- | --- |
| Print Screen | Win11 打开截图工具栏；Win10 复制全屏到剪贴板 | Win11 自动保存；Win10 不自动保存 |
| Win+Print Screen | 图片\屏幕截图 | 是 |
| Win+Shift+S | 剪贴板（Win11 同时自动保存） | Win11 是；Win10 早期版本否 |
| Alt+Print Screen | 剪贴板 | 否 |

如果你的需求不止这些——比如需要截取右键菜单打开时的画面（按下快捷键时菜单会消失），可以试试截图工具栏中的"延迟"功能，设置为 3 秒或 5 秒后再截图。要截取超过一页的网页，搜索"滚动截图"可以找到相关工具或方案。

## 四、录屏

### 1. 截图工具栏录屏

在截图工具栏中点击"录屏"按钮（圆形图标），即可切换到屏幕录制模式。直接按 `Win+Shift+R` 也可进入录屏模式（此快捷键在部分版本中可能已合并或不存在，以实际表现为准）。

#### 截图工具录制步骤

- 打开截图工具栏（按 `Win+Shift+S` 或 `Print Screen`），点击"录屏"按钮进入录制模式，或直接按 `Win+Shift+R`
- 屏幕会变暗，顶部出现录制工具栏
- 用鼠标拖选要录制的区域（可以选整个屏幕，也可以只选局部）
- 点击顶部工具栏的"新建"按钮开始录制
- 录制过程中屏幕右上角会出现浮动控制条，显示录制时长和暂停/停止按钮
- 点击停止按钮结束录制
- 录制完成后会自动弹出预览窗口，可以查看、编辑或保存视频
- 视频默认保存到 `C:\Users\你的用户名\视频\屏幕录制` 文件夹（具体名称和位置可能因版本而异）

#### 录制时的细节

- **录制麦克风**：在开始录制前，工具栏上有麦克风图标，点击可以开启或关闭麦克风录音
- **录制系统声音**：默认会录制系统声音（电脑播放的声音）
- **录制格式**：MP4 视频文件，大多数播放器和剪辑软件都支持
- **优点**：可以录制桌面、文件资源管理器等任意区域；操作简单直观；不需要将程序标记为游戏

#### 截图工具局限

- 缺少独立的音量混合器（无法分别调节系统音和麦克风音量的比例）
- 功能相对基础，没有直播推流等高级功能

### 2. Xbox Game Bar（Win+G）

按 `Win+G` 打开 Xbox Game Bar（游戏栏），它是 Windows 11 自带的另一个录屏工具。如未响应，可前往 `设置 → 游戏 → Xbox Game Bar` 确认功能已开启。

#### Game Bar 录制步骤

1. 打开你想录制的程序窗口
2. 按 `Win+G` 打开 Game Bar
3. 在顶部或底部的小部件栏里找到**录制**按钮，点击开始录制
4. 录制过程中屏幕边缘会出现一个小工具栏，显示录制时长和停止按钮
5. 按 `Win+Alt+R` 也可以直接开始或停止录制，比点按钮更快
6. 录制完成后，视频保存到 **`C:\Users\你的用户名\视频\捕获`**（部分版本可能叫"摄像"，具体随版本有所差异，也可以自行设置）

#### 录制时的一些细节

- **录制麦克风**：Game Bar 默认不录制麦克风声音。在录制小部件里可以点麦克风图标开启，这样你的讲解声也会被录进去
- **录制系统声音**：默认会录。可以在 Game Bar 的音量混合器里调整游戏音量和麦克风音量的比例
- **录制时长**：单次录制可在设置中调整，同时也受硬盘剩余空间限制
- **录制格式**：MP4 视频文件，大多数播放器和剪辑软件都支持
- **Win11 变化**：部分场景支持录制桌面，但如果提示"无法录制 Windows 桌面"，属于正常限制，可先打开一个应用程序窗口再录制

#### Game Bar 局限

- 录制过程中不能切换录制目标（如果要录另一个程序，需要停止后重新开始）
- 在部分旧硬件或未安装显卡驱动的电脑上可能无法正常工作

如果需要录制桌面、录制多窗口操作、或者需要更复杂的剪辑功能，可以在搜索引擎或 AI 工具中搜索"免费录屏软件推荐"，找适合你的第三方工具。日常快速录个操作演示给朋友看，Game Bar 够用了。

## 五、总结：什么时候用哪个

| 你想做什么 | 用什么 | 快捷键 |
| --- | --- | --- |
| 截全屏并粘贴给别人 | Print Screen | `Print Screen` |
| 截全屏并保存为文件 | Win+Print Screen | `Win+Print Screen` |
| 截屏幕的任意区域 | 截图工具 | `Win+Shift+S`（或 `Print Screen`） |
| 只截当前窗口 | Alt+Print Screen | `Alt+Print Screen` |
| 对截图做标注或裁剪 | 截图工具 / Print Screen | 截完点通知进入编辑，或直接点顶部标注按钮 |
| 录制屏幕操作 | Game Bar | `Win+G` 或 `Win+Alt+R` |
| 录制屏幕局部区域 | 截图工具录屏 | `Win+Shift+R`（打开后选区域再点录制） |

这些工具都是 Windows 自带的，不需要下载任何额外软件。下次遇到"这个怎么截下来给别人看"的场景，直接上手用就行。

---

> 自行探索：在截图工具中点击"延迟"按钮可以设置截图延迟，适合截取菜单、提示框等点击后会消失的画面。Game Bar 的设置里可以修改录制快捷键、视频帧率和质量等参数。有兴趣的话可以打开看看。
