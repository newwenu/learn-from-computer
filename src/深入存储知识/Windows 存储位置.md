# Windows 存储位置：C 盘里的文件都放在哪

> 阅读前提：你已经了解了文件路径和盘符的概念（参见《理解你的操作系统》的「文件路径与文件系统」部分），并且知道 C 盘是系统盘。

---

## 一、C 盘根目录下的关键文件夹

打开 C 盘（按 Win+E → 左侧点击"此电脑" → 双击 C 盘），你会看到几个文件夹。下面说明每个文件夹放的是什么。

> **看不到某些文件夹？** 部分系统文件夹默认是隐藏的（不在文件管理器中显示，以免误操作）。如果下面提到的文件夹你找不到，在文件管理器顶部点击"查看" → 勾选"隐藏的项目"即可显示。这不影响系统安全，只是 Windows 默认把它们藏起来了。

### Program Files 和 Program Files (x86)

存放你安装的应用程序。你在安装软件时选择的安装位置，默认就是这里。

- **Program Files** — 64 位程序安装在这里
- **Program Files (x86)** — 32 位程序安装在这里（详见《理解你的操作系统》的「32 位和 64 位」部分）

大多数情况下你不需要直接进这两个文件夹。卸载软件应该通过"设置 → 应用 → 已安装的应用"来操作，不要直接删除这里的文件夹（《日常使用》的「卸载软件」部分有说明）。

### Windows

存放操作系统本身的核心文件。**不要修改或删除这个文件夹里的任何内容**，系统运行全靠它。

具体来说，Windows 文件夹里包括：

- `System32` — 系统核心文件、驱动程序、命令行工具
- `Temp` — 系统和部分程序存放临时文件的地方，下文会讲到
- `WinSxS` — 系统组件存储，体积很大但不要手动删

### Users（用户）

存放所有用户的个人数据。每个用户一个子文件夹，名字就是你的用户名。下文详细说明。

### ProgramData

存放应用程序的共享数据和设置（不区分用户，对所有登录用户生效的配置）。默认是隐藏的（需要在文件管理器"查看"中勾选"隐藏的项目"才能看到）。不需要手动操作。

---

## 二、你的用户文件夹（Users）

进入 `C:\Users\你的用户名`，你会看到系统为你创建的几个专用文件夹。这是你个人文件的"总部"。

| 文件夹 | 路径名 | 用途 |
| ------ | ------ | ---- |
| **桌面** | Desktop | 桌面上所有文件和快捷方式 |
| **文档** | Documents | 文档、学习资料的默认保存位置 |
| **下载** | Downloads | 浏览器下载的文件默认存到这里 |
| **图片** | Pictures | 图片和照片 |
| **视频** | Videos | 视频文件 |
| **音乐** | Music | 音频文件 |

在文件管理器左侧点击对应的图标（桌面、文档、下载等），可以快速进入这些文件夹，不需要记路径。

> **建议**：定期整理"下载"文件夹，里面的文件最容易堆积。下载过的安装包、压缩包，用完就可以删掉。

### AppData — 程序存放设置的地方

这是一个**隐藏文件夹**，存放每个程序的专属设置和缓存数据。它分为三个子文件夹：

- **Roaming** — 会跟随微软账户同步到其他电脑的程序设置（换电脑登录同一账户设置还在）
- **Local** — 只属于这台电脑的程序设置和缓存（换电脑就不在了）
- **LocalLow** — 权限受限的程序设置，例如浏览器等需要高安全性的程序（权限越低越安全——程序能做破坏的事就越少。这涉及 Windows 完整性级别的概念，有兴趣可自行了解"Windows 完整性级别"）

在文件管理器地址栏输入 `%appdata%` 可以快速打开 Roaming 文件夹，输入 `%localappdata%` 打开 Local 文件夹。

日常使用建议：**不要动它**。AppData 里的文件体积可能很大（浏览器缓存、聊天软件的历史文件），但手动删除可能引发程序异常。如果你需要在 AppData 里清理空间，最好是去对应程序的设置里操作（比如在浏览器设置里清除缓存）。

## 三、临时文件夹

Windows 和很多程序在工作时会产生临时文件——解压缩的中间文件、软件安装时的缓存、系统更新时留下的碎片。正常情况下程序用完就会自动删除，但偶尔会有残留。

有两个主要的临时文件夹：

- `C:\Windows\Temp` — 系统临时文件（需要有管理员权限才能清理）
- `C:\Users\你的用户名\AppData\Local\Temp` — 当前用户的临时文件（`%temp%` 是它的快捷路径，按 Win+R 输入 `%temp%` 回车即可打开）

这里的文件理论上可以全部选中删除（有些正在使用的文件会提示"无法删除"，跳过就行），但不需要特意去找——下文讲的系统自带清理工具会处理它们。

## 四、回收站

按 Delete 键删除的本地硬盘文件默认不会真正消失，而是被移到了回收站。回收站占用的是 C 盘空间（每个分区有自己的回收站，实际是分区根目录下名为 `$Recycle.Bin` 的隐藏文件夹），默认大小是 C 盘总容量的 5%-10%。当回收站里的文件总大小超过这个上限时，最旧的文件会被永久删除。

注意：U 盘、移动硬盘、网络位置等**可移动或远程存储上的文件**，按 Delete 会直接永久删除，不会进入回收站。

### 清空回收站

右键桌面回收站图标 → "清空回收站"。文件会彻底删除，无法从回收站恢复。

### 单独恢复或删除

双击回收站打开：

- 恢复某个文件：右键 → "还原"
- 彻底删除某个文件：右键 → "永久删除"

### 调整回收站大小

右键回收站 → "属性" → 在「自定义大小」中修改最大值。如果你 C 盘空间紧张，可以把比例调小一些。

## 五、系统清理

当 C 盘空间不足时（通常是红色条提示），以下两种系统自带的工具足够解决问题。

### 存储感知（推荐）

Windows 10/11 自带的自动清理功能。

**打开方式：** 设置 → 系统 → 存储 → 存储感知

**可以做的事：**

- 开启"自动清理"开关
- 点击"配置存储感知或立即运行"，设置运行频率（建议设为"每月"）
- 选择要清理的内容：临时文件、回收站文件（超过多少天未删除的自动清掉）、下载文件夹（超过多少天未打开的自动清掉）

### 磁盘清理

适合不喜欢"自动"、想自己看一眼再决定删什么的人。

**打开方式：**

1. 在搜索框输入"磁盘清理"并打开
2. 选择要清理的磁盘（通常是 C 盘）
3. 等待扫描完成后，你会看到一个清单，每一项都有说明

**可以安全勾选的项目：**

| 项目 | 说明 |
| ------ | ---- |
| 临时 Internet 文件 | 浏览器缓存的网页文件，删了不影响使用 |
| 回收站 | 清空回收站 |
| 临时文件 | 程序和系统产生的临时文件 |
| Windows 更新清理 | 更新后保留的旧版本文件（通常能释放几个G） |
| 传递优化文件 | Windows 更新下载过程中暂存的缓存文件 |
| 缩略图 | 文件夹和图片的缩略图缓存，删了自动重建 |

**不那么安全、建议保留的项目：**

- 以前的 Windows 安装 —— 如果你确定不想回退到旧版本，可以删（但不能反悔）
- 下载 —— 勾选它会清空你的下载文件夹

勾选完成后点击"确定"→"删除文件"。清单里的每一项都是系统判定可以安全删除的内容，勾选前确认你明白要删的是什么即可。

## 六、整理习惯与注意事项

### C 盘空间建议

- 留出足够的空间给系统运转：Windows 更新、虚拟内存、休眠文件、临时文件等都需要空间。如果 C 盘剩余空间低于 **15-20GB**，容易遇到更新失败或系统卡顿。256G 的 C 盘留 25-30G 是比较稳妥的范围
- 安装大型软件（游戏、设计工具、开发环境）前看一眼 C 盘剩余空间。如果空间充裕（几十 G），装 C 盘没问题；如果 C 盘本身不大，可以考虑装到其他盘。真正持续增长的个人文件（照片、视频、下载文件）养成习惯放到 D 盘更合理
- 《日常使用》提到了文件不要堆桌面——桌面的文件实际存在 `C:\Users\你的用户名\Desktop`，堆满桌面等于堆满 C 盘

### 什么情况不需要清理

- C 盘还有几十 G 空余时，不必为了"优化"而频繁清理
- 电脑卡顿但 C 盘还有很多空间——卡顿通常不是垃圾文件导致的

### 不建议做的事

| 操作 | 为什么不建议 |
| ------ | ------------ |
| 安装"优化大师""垃圾清理"类软件 | 弹广告、捆绑其他软件、可能误删文件。系统自带工具完全够用 |
| 手动删除 Windows 或 Program Files 里的文件 | 可能导致系统或程序无法正常运行 |
| 清理 AppData 里的文件夹 | 可能导致程序设置丢失或异常，而且通常腾不了多少空间 |

---

## 七、总结

| 位置 | 用途 | 需要手动管吗 |
| ------ | ---- | ------------ |
| Program Files | 已安装的程序 | 通过"设置→应用"卸载 |
| Users\你的用户名 | 你的个人文件 | 定期整理下载文件夹即可 |
| AppData | 程序设置和缓存 | 不要动 |
| 回收站 | 已删除文件的暂存 | 需要时清空 |
| Temp | 临时文件 | 存储感知或磁盘清理会自动处理 |
| Windows | 操作系统文件 | 不要动 |
