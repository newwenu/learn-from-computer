# 基本操作：拿到一台新电脑后怎么上手

> 阅读前提：你已经了解了 PC 有哪些部件（参见《认识PC》）。本文以一台预装 Windows 的笔记本为例展开，大部分操作也适用于台式机。

## 一、开机与关机

### 开机

找到电源键。台式机通常在机箱前面板（正面），笔记本通常在键盘上方或机身侧面边缘。按一下，听到风扇转动、屏幕亮起，说明开机了。

> **第一次开机按了没反应？** 部分笔记本出厂有"运输保护"机制，电池处于休眠状态，需要插上电源适配器（充电器）再按电源键才能启动。这是正常现象，不是电脑坏了。

第一次开机，系统会引导你完成初始设置 `[查]`：

- 选择语言和区域（选"中文（简体，中国）"）
- 连接 Wi-Fi
- 登录微软账户（建议用微软账户，忘记 PIN 可以在线重置）
- 设置 PIN 码（纯数字密码，用于快速登录）
- 隐私设置：可以全部选"是"，也可以关闭部分选项

> **用户名建议用英文**（如 `ZhangSan`），避免部分老旧软件对中文路径支持不好。

设置完成后稍等几分钟，就会进入桌面。

> 此处是简单讲解。如需更详细的验机流程，建议搜索"新电脑验机"等关键词，或在视频平台查找视频教程跟着操作。注意查看教程发布时间，优先参考较新的内容。

### 关机

**正确关机方式**：点击"开始"按钮（屏幕左下角或任务栏中央的 Windows 图标，四个小方块组成的窗口图案）→ 点击电源图标 → 选择"关机"。电脑会自动关闭所有程序、保存系统状态、切断电源。

> 如果你找不到"开始"按钮，把鼠标移到任务栏中央（Windows 11）或屏幕最左下角（Windows 10）`[查]`，看到四个小方块组成的 Windows 图标就是。按键盘上带有同样图标的 Win 键也能打开它。

**正确关机的理由**：操作系统可能正在读写数据，强行断电可能导致文件损坏。当电脑完全死机（鼠标键盘都没反应）或系统严重卡顿无法恢复时，可以长按电源键 5-10 秒强制关机。

### 睡眠与重启

- **睡眠**：点击电源菜单里的"睡眠"。电脑进入低功耗状态，但内存继续供电以保持当前工作状态。按任意键或开盖会快速恢复到之前的状态。睡眠适合短时间离开（比如去吃饭、课间休息）。
- **重启**：点击"重启"。电脑会关闭所有程序、重新启动系统。当你安装了新软件或更新了系统，或者电脑出现卡顿异常时，重启往往能解决问题。重启是解决软件层面小问题的首选方法（硬件故障除外）。

### 合盖（笔记本专用）

合上盖子默认会让笔记本进入睡眠状态。可以在电源设置里修改为"不采取任何操作"（适合外接显示器使用）或"关机" `[查]`。

## 二、认识桌面

开机进入桌面后，你会看到以下部分：

### 桌面背景（壁纸）

占据整个屏幕的背景图片。右键点击桌面空白处 → "个性化" → "背景"可以更换。

### 桌面图标

桌面上的小图片，每个图标对应一个程序或文件。双击可打开：

- **此电脑**（或"计算机"）— 用来查看和管理硬盘、U盘、外接设备
- **回收站** — 删除的文件临时存放在这里，还没真正消失，可以恢复或彻底清空
- **控制面板** — 部分设置项需要在这里找

如果你桌面上没有"此电脑"，可以右键桌面空白处 → "个性化" → "主题" → "桌面图标设置" → 勾选"计算机" → 确定。

### 任务栏

屏幕底部那条长条，是 Windows 最重要的导航工具：

- **开始按钮**（Windows 图标）— 点击打开开始菜单。Windows 11 默认在任务栏居中，Windows 10 默认在左下角 `[查]`
- **搜索框/搜索图标** — 可以搜索电脑里的程序、文件、设置项。你可能还没注意到，按键盘上的 Win 键然后直接打字，效果一样
- **任务栏图标区** — 固定在这里的程序图标，以及正在运行的程序图标。点击图标可以打开或切换到对应程序。正在运行的程序图标下有一条小横线
- **托盘区**（也叫**系统托盘**，右下角）— 显示时间、音量、网络状态、电池（笔记本）等。部分后台运行的程序也会在这里显示图标。有些程序默认会把图标收起来，点击托盘区左侧的 **^（向上小箭头）** 就能看到被收起的图标。你也可以拖动图标，把它从折叠区拖出来让它一直显示，或者拖回去收起来
- **显示桌面**（任务栏最右侧一条细长的透明区域）— 点击一下可以让所有窗口最小化，直接看到桌面。再次点击恢复窗口

### 开始菜单

点击开始按钮打开开始菜单。里面有：最近使用的程序、所有程序的列表、以及一个齿轮图标——**设置**。

> **打开"设置"的方法：** 点击开始菜单里的齿轮图标，或直接按 `Win+I`。"设置"是后面很多章节都会用到的入口，记住这个快捷键很有用。

开始菜单打开后，左侧是常用程序列表，右侧是功能磁贴区域或固定区域 `[查]`。你可以在这里：

- 滚动或搜索找到任何已安装的程序
- 右键点击程序 →"固定到任务栏"或"固定到开始屏幕"
- 直接打字搜索（如上所述，按 Win 键后直接输入程序名）

## 三、窗口操作

在 Windows 中，每个打开的程序都显示在一个"窗口"里。学会窗口操作是使用电脑的基本功。

### 打开和关闭

- **打开**：双击桌面图标，或在开始菜单中点击程序
- **关闭**：点击窗口右上角的 ×，或者按 Alt+F4

### 移动窗口

把鼠标指针放在窗口最上面的标题栏（通常是一条横条，颜色因系统和程序而异），按住左键拖动，移到你想要的位置再松开。

### 调整窗口大小

- 把鼠标指针移到窗口的边缘或角落，指针会变成双向箭头，按住左键拖动即可。部分窗口（如某些设置弹窗）有固定大小，不支持手动拖拽调整
- 最大化（让窗口占满整个屏幕）：点击右上角的 □ 图标，或双击标题栏
- 还原（回到之前的大小）：再次点击 □ 图标
- 最小化（让窗口缩小到任务栏，不关闭程序）：点击右上角的 — 图标。任务栏上仍然有它的图标，点击可以恢复显示

### 快速分屏（有用的小技巧）

把窗口拖到屏幕的最左侧或最右侧边缘，直到鼠标碰到边缘，窗口会自动占满左半或右半屏幕。另一边可以放另一个窗口。这样你就能同时看到两个窗口的内容（比如一边看资料一边写文档），不需要来回切换。

Windows 11 还有更直观的分屏布局：把鼠标悬停在窗口右上角的 □ 图标上，会弹出几种布局方案。

### 多窗口切换

- Alt+Tab：按住 Alt 键不放，按一下 Tab 键会弹出一个缩略图列表，继续按 Tab 在程序间切换，松开 Alt 就切换到选中的程序
- Win+Tab：类似 Alt+Tab，但不会松开按键后立即切换，适合慢慢找
- 直接点击任务栏上的程序图标

### 注意：关闭窗口 ≠ 退出程序

点窗口右上角的 ×，有时候程序并没有真正关掉。

**常见的几种情况：**

- **计算器、记事本**：点 × 就是彻底退出，任务栏上的图标会消失
- **Word、Excel**（如果你装了 Office）：点 × 只关闭当前文档，程序还在运行。你还可以再新建一个文档
- **微信、QQ、音乐软件**：点 × 后窗口不见了，但程序躲到了屏幕右下角的托盘区（系统托盘），仍在后台运行（也就是"关了窗口不等于退出了程序"）。如果右下角没看到它，点一下托盘区左侧的 **^（向上小箭头）**，被收起的图标就会展开。这样下次打开速度快，也能继续收消息

**怎么判断程序是不是还在运行？**

两个基础简单的判断方法：

- **看托盘区**——窗口不见了，但右下角托盘区还有小图标，说明程序躲在后台运行（比如微信、音乐软件）。没找到就点一下 **^（向上小箭头）** 看看折叠区里有没有
- **看任务栏**——如果任务栏图标下面还有小横线，说明窗口只是被最小化了，点击图标就能恢复窗口

**想彻底退出怎么办？**

- Word 这类：点击"文件 → 退出"，或者关掉所有文档窗口
- 微信这类：在托盘区找到它的小图标，右键点击，选择"退出"
- 通用方法：在任务栏图标上右键 → "关闭窗口"

## 四、鼠标与键盘基础操作

### 鼠标

| 操作         | 作用                                              |
| ------------ | ------------------------------------------------- |
| 左键单击     | 选中某个对象（文件、按钮、链接）                  |
| 左键双击     | 打开文件或启动程序                                |
| 右键单击     | 弹出右键菜单（显示对该对象可进行的操作列表）      |
| 左键按住拖动 | 选中文字或文件、移动窗口、拖放文件到另一个位置    |
| 滚轮         | 向上/下滚动页面                                   |
| Ctrl+滚轮    | 放大/缩小视图（浏览器、Word、文件资源管理器等都支持） |

笔记本电脑的触摸板默认模拟这些操作：单指移动（移动鼠标光标）、单指点击（左键单击）、双指点击（右键单击）、双指滑动（滚轮）。

> **"指针"是什么？** 指针就是你在屏幕上看到的鼠标光标——那个白色小箭头（有时是手形、I形等其他形状）。移动鼠标或滑动触摸板时，指针跟着动。下文提到"指针"时，指的就是这个光标。

### 键盘布局速览

首先认识按键的大类：字母键（A-Z）、数字键（0-9）、符号键（如 , . / ; '）、功能键（F1-F12）、控制键（Ctrl、Alt、Shift、Win）、方向键、回车键 Enter、退格键 Backspace。不同键盘的布局可能略有差异（比如有些紧凑键盘没有数字小键盘，一些笔记本把 Fn 键放在左下角而有些则在 Ctrl 键外侧），但基本分区一致。

标准键盘主要分为几个区域：

- **主键盘区**（中间最大的一块）：字母、数字、标点符号
- **功能键区**（顶部 F1-F12）：不同软件中功能不同。F2（重命名文件）、F5（刷新页面）、F11（全屏）
- **控制键区**（主键盘右侧）：方向键（上下左右移动光标）、Home/End（跳到行首/行尾）、PageUp/PageDown（翻页）
- **数字小键盘**（最右侧）：方便输入大量数字。台式机键盘通常有，部分全尺寸笔记本也有，但许多笔记本没有（或通过 Fn 组合键复用字母键上的数字）。按一下 NumLock 键可以切换：灯亮时输入数字，灯灭时变成方向键

### 怎么按快捷键

**快捷键**（也叫**组合键**）是多键一起按来快速执行操作的方式。按法不是同时猛按，而是**依序按下、最后一起松开**：

1. 按住第一个键不放
2. 按住第二个键不放
3. 按一下第三个键（如果有）
4. 一起松开

例如按 `Ctrl + Shift + Esc`：先按住 `Ctrl`，再按住 `Shift`，按一下 `Esc`，最后全部松开。

### 先认识这些特殊键在哪

在记快捷键之前，先找到键盘上这几个常用的特殊键：

- **Win 键（Windows 键）**：键盘左下角，带有 Windows 图标（四个小方块）的按键。不同键盘布局可能略有不同，找带有 Windows 图标的键即可。按一下就会打开开始菜单。
- **Ctrl 键（控制键）**：键盘左下角，Win 键旁边，标有 "Ctrl"。左右各有一个，用哪个都可以。
- **Alt 键（交替键）**：Ctrl 键旁边，标有 "Alt"。左右各有一个。
- **Shift 键（上档键）**：键盘左右两侧，标有 "Shift"，带有向上的箭头。用于输入大写字母或配合其他键使用。
- **Delete 键（删除键）**：键盘右上角区域，标有 "Delete" 或 "Del"。
- **Esc 键（退出键）**：键盘左上角，标有 "Esc"。
- **F1-F12 键**：键盘最上方一排，标有 F1 到 F12。不同软件里功能不同。

> **笔记本注意**：有些笔记本的 F1-F12 键默认是调节亮度/音量，需要同时按 **Fn 键**（通常在键盘左下角）才能当普通功能键用。如果按某个快捷键没反应，试试加上 Fn 键。

### 建议记住的快捷键

这十几组快捷键几乎每天都会用到，值得花几分钟记住：

**基本编辑：**

- **Ctrl+C** — 复制（把选中的内容暂存起来）
- **Ctrl+X** — 剪切（复制并删除原内容）
- **Ctrl+V** — 粘贴（把刚才复制的内容放出来）
- **Ctrl+Z** — 撤销（撤销上一步操作）
- **Ctrl+Y** — 重做（取消撤销）

**文件管理：**

- **Ctrl+A** — 全选（选中当前窗口里所有内容）
- **Ctrl+S** — 保存（写文档、做图时随时按一下保存，养成习惯）
- **Ctrl+F** — 查找（在当前页面或文档中搜索关键词）

**系统操作：**

- **Win+E** — 打开文件资源管理器（随时管理文件）
- **Win+D** — 显示桌面（最小化所有窗口，再按一次恢复）
- **Win+L** — 锁定电脑（离开座位时按一下，保护隐私，回到锁屏界面需要重新输入密码）
- **Win+R** — 打开"运行"对话框（输入程序名可快速启动，后续章节会用到）
- **Alt+F4** — 关闭当前窗口/程序
- **Alt+Tab** — 切换窗口
- **F2** — 重命名选中的文件或文件夹
- **F5** — 刷新当前页面/文件夹内容
- **Delete** — 删除选中的文件到回收站
- **Shift+Delete** — 永久删除（跳过回收站，无法恢复，慎用）

> **不用一次全记住。** 这么多快捷键，先记住 Ctrl+C/V/Z/S 和 Win+E/L 就够日常用了。其他的用到时回来查，用多了自然就记住了。

### 中文输入法

Windows 自带微软拼音输入法，通常已经装好。使用方法：

- **切换输入法**：Win+空格键（切换所有已安装的输入法），或 Shift（在当前输入法里切换中英文）
- **中英文切换**：按 Shift 键
- **候选词翻页**：按 =（下一页）、-（上一页），或使用方括号 [ ]
- **输入特殊符号**：按 Win+. 或 Win+; 打开表情符号面板

> 如果你之前主要用手机打字，不太熟悉电脑键盘输入，可以查看[附录《打字基础》](../附录/打字基础.md)，了解最基本的电脑打字操作。
