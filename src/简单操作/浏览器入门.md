# 浏览器入门：上网的基本功

> 阅读前提：你已经会用鼠标键盘、会操作窗口（参见《基本操作》）。本文不绑定某款浏览器，而是告诉你浏览器里那些通用功能去哪找——换个浏览器你也能自己摸明白。

---

## 一、浏览器是什么

浏览器就是你上网用的那个软件。打开它，输入网址或者搜个东西，网页就显示出来了。

常见的浏览器有 **Edge**（Windows 自带，蓝色波浪图标）、**Chrome**、**Firefox**。它们长得不太一样，但核心东西都一样——地址栏、标签页、菜单、设置。这篇讲的东西，换哪个浏览器都管用。

> 对国内用户来说，Edge 是最方便的选择：系统自带，国内可以直接同步书签和密码。Chrome 需要额外配置才能稳定同步，新手建议先用 Edge。

---

## 二、标签页管理 —— 每天用得最多的操作

浏览器可以同时打开很多网页，每个网页占一个"标签"。所有标签排成一排，显示在浏览器窗口的顶部或左侧，地址栏的上方，这一排就叫**标签栏**。你当前看的网页对应的标签通常会高亮显示（颜色更深或带下划线）。

### 通用快捷键（所有浏览器都一样）

| 快捷键 | 干什么用的 | 不用快捷键怎么做 |
|-------|-----------|-------------------|
| Ctrl+T | 新开一个标签页 | 右键标签栏空白处 → "新建标签页"，或点击标签栏右侧的 + 按钮 |
| Ctrl+W | 关掉当前标签页 | 点击标签页上的 × 按钮，或右键标签页 → "关闭标签页" |
| Ctrl+Shift+T | 恢复刚关掉的标签（手滑关错，救命的） | 右键标签栏空白处 → "重新打开关闭的标签页" |
| Ctrl+Tab | 切到右边那个标签 | 直接用鼠标点击要切换的标签页 |
| Ctrl+Shift+Tab | 切到左边那个标签 | 直接用鼠标点击要切换的标签页 |
| Ctrl+1~8 | 直接切到第 1~8 个标签 | 直接用鼠标点击对应的标签页 |
| Ctrl+9 | 切到最后一个标签 | 直接用鼠标点击最后一个标签页 |

### 固定标签页 —— 把常用的钉在左边

有些网页你天天打开（邮箱、日历、工作后台），每次打开浏览器都要重新点一遍，挺烦的。可以把它**固定**住。

**怎么弄：** 右键点那个标签 → 菜单里找"固定标签页"或"固定"（不同浏览器叫法不一样，但都在右键菜单里）。固定之后，标签会缩成一个小图标挤在最左边，不占地方，下次打开浏览器它还在，也不怕手滑关掉。

### 标签分组 —— 把一堆标签装进一个组

开了一堆写作业的网页，又开了一堆买东西的网页，混在一起找半天找不到想要的那个。可以把同类标签放到一个组里。

**怎么弄：** 右键点一个标签 → 找"添加到新组"或"为标签添加分组"。可以给组起个名字、选个颜色，点一下组里任意标签就能展开或收起整组。

每个浏览器分组功能放的位置不太一样，但在右键菜单里都能找到。你右键点一下标签看看你的浏览器把分组放哪了。

## 三、地址栏不只是输网址的地方

浏览器最上面那条长条叫**地址栏**。你可能平时只往上面打网址，但它能干的不止这些。

### 直接当搜索框用

在地址栏里输个关键词（比如"今天天气"），按回车，浏览器就会用你设的搜索引擎来搜，不用先打开搜索引擎的首页。

### Ctrl+L —— 一下跳回地址栏

按 `Ctrl+L`，光标直接跳到地址栏，而且会选中里面的全部内容，你直接打字或者粘贴上去就行，不用拿鼠标去点。

### 看网址长什么样

打开一个网页，地址栏里会显示一串地址，比如 `https://www.baidu.com/something/page.html`。其中 **`baidu.com`** 这一段就是**域名**，相当于网站的门牌号。认准域名就能判断网站真假——骗子经常用跟真网站很像的域名来骗你，比如 `ba1du.com`（数字 1 冒充字母 i）假装是百度。输账号密码前，先看一眼地址栏里的域名对不对得上。

### 默认搜索引擎在哪改

浏览器默认用哪个搜东西（Edge 默认用必应，微软的搜索引擎），可以在设置里改。不同浏览器的设置入口不一样，但进去之后搜"搜索引擎"就能找到。

## 四、下载的文件跑哪去了

### Ctrl+J 打开下载记录

你在浏览器里点下载文件，文件不会凭空消失。按 `Ctrl+J` 打开下载记录，能看到正在下和已经下完的所有文件，双击可以直接打开文件所在的文件夹。

### 默认存在哪

浏览器下载的文件，默认存到 `C:\Users\你的用户名\Downloads`，也就是"下载"文件夹。下载完成后，Edge/Chrome 会在窗口底部或右上角出现下载提示，点击可以"打开文件"或"打开所在文件夹"。

要是想直接存到桌面或者其他盘，可以在浏览器设置里找"下载"相关的选项，改一下默认保存位置。

> 建议：下载完成后顺手把文件移到该放的位置（比如 D 盘的软件文件夹），不要一直堆在 Downloads 里。

### 留意浏览器的安全提醒

下载文件的时候，浏览器有时候会在底部或顶部弹出一条提示，说"此文件可能不安全"或者"此类型文件可能损害你的计算机"。如果你不确定这个文件靠不靠谱，先停下来看看，别直接点"保留"或"仍要下载"。

**特别常见的情况**：下载某些软件（尤其是编程工具、开源软件）时，Edge 可能会弹出"此文件可能有害"的提示，并阻止下载。这可能是因为这些软件没有向微软付费做数字签名认证，不代表它们真的有害。

**判断方法：**
- 如果你是从**官网**下载的（比如 python.org、code.visualstudio.com），可以放心使用
- 点击下载提示旁边的"..."或"保留"按钮，选择"保留"即可

> 不确定是不是官网？对照《日常使用》里讲的"识别官网"方法，检查网址域名。

### 下载后找不到文件

如果点了下载但找不到文件：

1. 按 `Ctrl+J` 打开下载记录，确认下载已完成
2. 点击"打开所在文件夹"定位文件
3. 如果文件名很长或带版本号，注意看扩展名（如 `.exe`、`.zip`、`.msi`）

## 五、书签 —— 把喜欢的网页存下来

### 存一个书签

打开一个以后还想看的网页，按 `Ctrl+D`。浏览器会弹出个小窗口让你确认名字和放哪，点"完成"或"确定"就行了。下次想看的时候，去书签菜单里点一下。

### 显示/隐藏书签栏

书签存多了，每次都去菜单里翻挺麻烦的。可以把书签栏显示出来——它就在地址栏下面那一条。按 `Ctrl+Shift+B` 就能显示或藏起来（所有浏览器都一样）。

不用把所有书签都放上书签栏，挑最常用的几个放上去就行。

### 书签怎么同步到其他设备

如果你在浏览器上登录了账号（Edge 登微软账户），并在设置里开启了同步，书签会自动同步到你其他设备上。在哪登录、怎么开同步，去浏览器设置里找"同步"或"账户"相关的选项。

### 要记得清理

书签这东西，很容易"存了再也不看"。隔几个月把没用的删一删，不然书签栏越堆越长反而不好找。

## 六、清理浏览痕迹

按 `Ctrl+Shift+Delete` 会弹出清除浏览数据的窗口（所有浏览器都一样），你可以选着清除：

- **浏览记录** —— 你去过哪些网页
- **缓存** —— 浏览器为了让你下次打开网页更快，提前存下来的网页内容
- **Cookie** —— 网站记住你登录状态的小记录（比如让你不用每次都输密码）
- **密码** —— 浏览器帮你存着的密码（这个别随便清）

平时不用折腾这些。只有浏览器变得很慢了、或者你要把电脑借给别人用一下的时候，才清一下浏览记录和缓存。

## 七、隐私模式

### 什么时候用得上

- 借别人电脑查自己的东西（用完关掉窗口，不留痕迹）
- 查病、买礼物之类的，不想被浏览器记住
- 同时登两个号（比如隐私窗口登个小号）

### 怎么打开

- Edge / Chrome：`Ctrl+Shift+N`
- Firefox：`Ctrl+Shift+P`

其他浏览器也能在菜单里找到"新建隐私窗口"或"无痕模式"之类的选项，叫法不一样，功能一样。

### 一个误区

隐私模式只是让你的浏览器**不在本地留记录**——关掉窗口之后，历史记录、Cookie、搜过的关键字都不会存下来。

但如果你在隐私窗口里登了微信或微博，**微信和微博照样知道你是谁**。隐私模式不是"隐身"，就是一个"走了不留下痕迹"的临时环境。

## 八、扩展程序（插件）

### 什么是扩展

扩展就是给浏览器加功能的小工具。比如：

- **广告拦截**（推荐 uBlock Origin，在扩展商店搜这个名字）—— 把网页广告挡掉
- **翻译** —— 翻译整页内容（有些浏览器自带）
- **密码管理** —— 帮你记住和填密码

### 从哪装

每个浏览器都有自己的官方扩展商店。在设置或菜单里找"扩展"或"加载项"，或者直接搜"你的浏览器名字 + 扩展商店"就能找到入口。

**只从浏览器的官方商店装**，别去陌生网站上点"下载插件"。

### 装扩展时的好习惯

官方商店的扩展基本是安全的，装的时候留意这几点就好：

- **看下载量和评价**——下载量大、评分高的扩展通常更靠谱
- **不用的可以随时在扩展管理页面关掉或卸载**，很方便

### 推荐试试：广告拦截

要是你觉得网页上广告太多、文章正中间插个大广告根本没法看，可以去扩展商店装个 uBlock Origin。装好之后大多数广告就没了，浏览体验好很多。

### 别装太多

扩展多了浏览器会变慢。只装真正用得上的，不用的可以关掉或删掉。

## 九、翻翻你的浏览器设置

每个浏览器都有个设置页，能在里面调各种行为。入口通常在右上角的**三个点**或**三条横线**菜单里，点进去找"设置"。

下面列几个常见的设置项，不用全记住，用到的时候知道去设置里找就行：

| 你想要的 | 去哪找 |
| -------- | -------- |
| 换默认搜索引擎 | 设置里搜"搜索引擎" |
| 改下载路径 | 设置里搜"下载" |
| 启动时打开指定页面 | 设置里搜"启动" |
| 管理存过的密码 | 设置里搜"密码" |
| 清除浏览数据 | 设置里搜"清除" |

不同浏览器的菜单位置不一样，但设置里的搜索框都是通用的——不知道某项功能在哪的时候，在设置页顶部的搜索框里搜关键词就行了。

## 十、自己试试看

1. **确认浏览器在自动更新**：设置或菜单里找"关于"，能看到版本号。保持最新版能修复安全问题，一般不用手动操心
2. **试试这几个快捷键**：新建标签（Ctrl+T）、关掉（Ctrl+W）、恢复（Ctrl+Shift+T）——这三个配合起来非常好用
3. **登个浏览器账号**：要是你还没登浏览器账号，去设置里看看"同步"或"账户"相关选项。Edge 用微软账户登录，登了之后书签、密码、设置都能跟着账号走，换电脑也不丢
4. **装个扩展玩玩**：去官方扩展商店搜"广告拦截"或"翻译"，装一个试试看
5. **翻翻设置页**：打开浏览器设置，从上到下扫一遍，不一定要改什么，知道浏览器能调什么就行

## 十一、总结

| 频率 | 你想干嘛 | 怎么弄 | 各浏览器通用吗 |
| -------- | -------- | -------- | ---------- |
| 每次都用 | 新开标签 | Ctrl+T | 都通用 |
| 每次都用 | 关掉当前标签 | Ctrl+W | 都通用 |
| 每次都用 | 恢复刚关的标签 | Ctrl+Shift+T | 都通用 |
| 每次都用 | 地址栏直接搜 | 打关键词按回车 | 都通用 |
| 天天用 | 看下过什么文件 | Ctrl+J | 都通用 |
| 每周 | 存个书签 | Ctrl+D | 都通用 |
| 每月 | 清浏览记录 | Ctrl+Shift+Delete | 都通用 |
| 偶尔 | 开隐私模式 | Ctrl+Shift+N 或 P | 键不一样 |
| 偶尔 | 装扩展 | 浏览器自己的扩展商店 | 各开各的店 |
| 偶尔 | 改设置 | 右上角菜单 → 设置 | 入口位置不同 |
