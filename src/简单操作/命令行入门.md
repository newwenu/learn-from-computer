# 命令行入门：学会和电脑"打字"交流

> 阅读前提：你已掌握基本的窗口操作和文件管理（参见《基本操作》和《日常使用》）。本文教你打开和使用 Windows 的命令提示符，这是理解其他文档中涉及命令行操作的预备知识。

---

## 一、命令行是什么

你平时用鼠标点击图标、拖动文件、点菜单选项——这种操作方式叫**图形界面**。

**命令行**（也叫命令提示符、cmd、黑窗口）是另一种和电脑交流的方式：**你输入文字命令，电脑执行并返回文字结果**。

```text
图形界面：点击"此电脑" → 双击 D 盘 → 看到文件
命令行：  输入  D: 回车 → 输入  dir 回车 → 看到文件列表
```

同样的事情，两种方式都能做。图形界面直观好记，命令行直接高效。有些操作（比如 ping、ipconfig）在图形界面上反而找不到入口，只能用命令行。

第一次看到命令行的感觉通常是"好古老、好可怕"——这很正常。你只需要记住：**命令行就是一台只能收发文字的聊天窗口，你在和操作系统"打字对话"**。它没有你想的那么神秘，也不需要背任何东西，跟着本文敲就行了。

### ⚠️ 安全提醒：不要在网上随便复制命令来跑

网上有很多"一键清理""加速电脑""修复系统"的文章或视频，里面经常贴一段命令行让你复制到黑窗口里执行。

**不建议这样做。** 命令行让你直接控制操作系统，一段你没看懂的代码（尤其是涉及 `del`、`rd`、`reg delete`、`format` 或递归操作的命令）可能在几秒内删光你的文件或破坏系统。

反过来，遇到你想搞清楚的命令，**拿去搜**——把那段命令复制到搜索引擎或 AI 工具里，问"这行命令是干什么的""跑它有什么风险"。这是最安全的学习方式，也是所有有经验的用户遇到看不懂的命令时都会做的事。

你不需要记住哪些命令是危险的——记住一条原则就够了：**看不懂的命令先查清楚再跑。** 如果某个教程让你复制一段代码但没解释每一行是干什么的，先拿去搜一下。本文里出现的每一条命令都说明了它的作用和场景，这是你判断"能不能跑"的标准。

## 二、打开命令提示符

Windows 自带的命令行程序叫**命令提示符**。以下三种方式随便用，记一个就行：

### 方式一：Win+R 法（最快）

1. 按 `Win+R`（弹出"运行"窗口）
2. 输入 `cmd`，按回车

### 方式二：搜索法

1. 点击任务栏的搜索图标（或按 `Win+S`）
2. 输入 `cmd` 或"命令提示符"
3. 点击搜索结果中的"命令提示符"

### 方式三：右键开始菜单

1. 右键点击开始按钮（或按 `Win+X`）
2. 选择"命令提示符"或"终端"

打开后会看到一个黑底白字的窗口，光标在闪烁等着你输入命令。这个窗口就是命令提示符。

> 本文所有操作都在 Windows 自带的命令提示符（cmd）中进行。日常操作 cmd 完全够用，想了解更多命令行工具可以去文末的「自行探索」看看。

## 三、基本操作

### 输命令，按回车

在黑窗口里输入一行文字，按回车，电脑就会执行这条命令。例如输入 `ping baidu.com` 然后回车，会看到网络测试的结果。

### cls —— 清屏

命令输出太多，屏幕太乱？输入 `cls` 回车，屏幕清空，回到顶部重新开始。

### exit —— 退出

输入 `exit` 回车，关闭命令提示符窗口。也可以直接点窗口右上角的 ×。

### ↑ 键——翻出刚才用过的命令

按键盘上方的 **↑** 方向键，会自动调出你刚才输入过的命令。多按几次可以翻更早的命令。如果翻多了，按 **↓** 往回翻。

这个操作非常实用——比如你输了一次 `ping baidu.com` 想再跑一次，按一下 ↑ 回车就行，不用重新打字。

### Tab 键——自动补全

输入路径或文件名时，按 **Tab** 键会自动补全。

举例：在当前目录下输入 `cd D`，按 Tab，如果当前目录里有一个以 D 开头的文件夹（比如 `Documents`），它会自动补全成 `cd Documents`。多个匹配项时连续按 Tab 切换。

## 四、以管理员身份运行

有些命令需要较高的权限才能执行（比如修改系统设置、重置网络）。普通方式打开的命令提示符执行它们会提示"拒绝访问"。

**什么时候需要"以管理员身份运行"：**
- 运行 `ipconfig /flushdns`（刷新 DNS 缓存）
- 运行 `netsh winsock reset`（重置网络）
- 修改系统文件或设置

**操作方法：**
1. 搜索 `cmd`
2. 在搜索结果中的"命令提示符"上右键
3. 选择"以管理员身份运行"
4. 弹出用户账户控制（UAC）窗口，点击"是"

你会看到窗口标题栏显示"管理员: 命令提示符"，说明你现在有管理员权限了。日常操作不需要这样做，只有用到上述少数命令时才需要。

> ⚠️ **特别注意**：在管理员模式下运行的命令拥有修改系统核心设置的权限，搞错了破坏力比普通模式大得多。安全提醒中说的"看不懂的命令不要跑"，在管理员模式下要更加严格——不确定的命令**先查清楚再跑**，查明白之前一个字符都不要敲。

## 五、最常用的几条命令

下面列出的命令足够应对大多数场景。你不需要背它们，记住有这些命令、知道它们是干什么的就行——用到时回来看。

### dir —— 看看当前文件夹里有什么

```cmd
dir
```

列出当前目录下的所有文件和子文件夹。效果和在文件管理器里打开一个文件夹看到的内容一样，只是以文字形式展示。

### cd —— 切换当前目录

```cmd
cd 文件夹名
```

进入指定的子文件夹。比如 `cd Documents` 进入文档文件夹。

**常用变体：**
- `cd ..` — 返回上一级文件夹（两个点）
- `cd \` — 直接回到当前盘的根目录（比如回到 C:\）

### ipconfig —— 查看本机网络配置

```cmd
ipconfig
```

列出电脑上所有网络适配器的信息，包括 IPv4 地址、子网掩码、默认网关。详见《理解你的网络》。

### ping —— 测试网络连通性

```cmd
ping baidu.com
```

向目标地址发送测试数据包，看它能不能通、响应快不快。详见《理解你的网络》。

### systeminfo —— 查看系统详细信息

```cmd
systeminfo
```

列出操作系统的完整信息，包括 Windows 版本、安装日期、内存大小、网卡信息等。比在设置里点来点去看得更全。

### 在命令行里退出正在运行的命令

如果你运行了 `ping baidu.com -t`（持续 ping），想让它停下来，按 **Ctrl+C**。这是通用的"中止当前命令"快捷键。

## 六、实用技巧

### 复制粘贴

命令提示符的复制粘贴和常规软件不太一样：

- **复制**：选中文字，按回车（或右键一下），文字就复制好了
- **粘贴**：在光标位置右键一下

现在的 Windows 终端也支持 Ctrl+Shift+C 复制、Ctrl+Shift+V 粘贴，和常规软件一致。

### 路径中有空格怎么办

如果文件夹名有空格（比如 `Program Files`），直接输入会报错。两种处理方法：

```cmd
cd "Program Files"    ← 用英文双引号包起来
cd Program Files      ← 或者打完 Program 后按 Tab 自动补全
```

按 Tab 自动补全最简单，Windows 会自动加好引号。

### 查看命令的帮助

在命令后面加 `/?`，会显示这个命令的用法说明：

```cmd
ping /?
```

会列出 ping 命令的所有参数和用法。英文的，但看到 `-t`、`-n`、`-l` 这些就是参数名。感兴趣可以查一下，日常用不着。

## 七、总结

| 操作/命令 | 作用 | 常用场景 |
|-----------|------|---------|
| `cls` | 清空屏幕 | 屏幕太乱时 |
| `exit` | 关闭命令提示符 | 用完了退出 |
| ↑ / ↓ | 翻历史命令 | 重复执行上一条命令 |
| Tab | 自动补全路径/文件名 | 不想打字时 |
| `dir` | 列出当前目录内容 | 看看文件夹里有什么 |
| `cd ..` | 返回上一级目录 | 目录切错了回退 |
| `cd \` | 回到盘符根目录 | 从深处直接跳回起点 |
| `ipconfig` | 查看本机网络配置 | 查 IP 地址、网关 |
| `ping 地址` | 测试网络连通性 | 判断能不能上网 |
| `systeminfo` | 查看系统详细信息 | 查 Windows 版本、安装日期 |
| Ctrl+C | 中止当前命令 | 停掉持续运行的命令 |
| `命令 /?` | 查看命令帮助 | 想了解一个命令的更多参数 |

---

自行探索：

1. **PowerShell** — Windows 自带的另一个命令行工具（蓝色背景），功能比 cmd 强大得多。从开始菜单搜索"Windows PowerShell"打开。试试输入 `Get-Process` 看看当前所有运行的进程——效果和任务管理器一样，但以文字列出来。

2. **命令的路径**——你输入的 `ping`、`ipconfig` 等命令其实对应的是 `C:\Windows\System32` 里的 `.exe` 文件（比如 `ping.exe`）。输入命令相当于让系统去这个目录找到对应的程序来执行。你可以用 `where 命令名` 来看命令对应的文件在哪，比如 `where ping`。

3. **文件名/目录名不区分大小写**——在 Windows 命令行里，`Documents` 和 `documents`、`PING.EXE` 和 `ping.exe` 是一样的，大小写随便写。

4. **如果你好奇更多命令**——在命令行输入 `help` 回车，会列出 cmd 支持的全部内部命令（`dir`、`cd`、`cls` 等）。但 `ping`、`ipconfig` 等外部命令不在这个列表里（它们是独立的 `.exe` 程序），不过同样可以用 `命令 /?` 查看用法。
