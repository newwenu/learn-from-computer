# 压缩与解压缩：怎么打开和创建压缩文件

> 阅读前提：你已经掌握了文件的基本操作（新建、复制、移动、删除），参见《日常使用》的"文件管理"部分。

---

## 一、什么是压缩文件

压缩文件是把一个或多个文件打包成一个文件，同时让体积变小，方便传输。

举个例子：你要把 10 个 Word 文档发给别人。一个个发很麻烦，打包成一个 .zip 文件，既省事又变小了。

常见的文件压缩（如 .zip、.7z）**不会损失文件内容**——解压缩后得到的是和原来一模一样的文件。它只是改变了文件的存储方式，就像把被子抽掉空气压缩进真空袋，拿出来抖一抖就恢复了。

常见的压缩格式：

| 格式 | 特点 | Windows 能否直接打开 |
| ---- | ---- | ---- |
| .zip | 最通用，所有系统都支持 | 能（无需额外软件） |
| .7z | 压缩率最高，免费无广告（开源软件，代码公开可查） | 新版 Windows 11 原生支持（旧版需安装 7-Zip），部分特殊压缩包也可能仍需要 7-Zip |
| .rar | 压缩率较高，WinRAR 专有格式（只有特定软件支持） | 不能 |

---

## 二、打开 .zip 文件

Windows 10/11 原生支持 .zip 格式，不需要安装任何软件。

### 方法一：直接查看内容

双击压缩文件，它会像文件夹一样在文件资源管理器中打开。你可以直接选中里面的文件复制出来。

### 方法二：提取全部文件

右键压缩文件 → 选择"全部提取"。在弹出的窗口中选择提取到的位置（保持默认即可），点击"提取"。系统会在压缩文件旁边创建一个同名文件夹，里面就是解压后的所有文件。

---

## 三、创建压缩文件

选中你想打包的文件或文件夹 → 右键 →"压缩到"→"zip文件" `[查]`。系统会在同一位置生成一个 .zip 文件，名字默认是其中某个文件的名称，你可以按 F2 重命名。

创建压缩包后，原始文件还在原位置。如果你确认压缩包没问题，可以删除原始文件。

---

## 四、右键菜单里的 7-Zip 选项，到底用哪个

安装 7-Zip 后，右键菜单会多出好几个选项，新手常常不知道该点哪个。下面是日常最常用的三个：

| 选项 | 作用 | 什么时候用 |
| ---- | ---- | ---------- |
| **7-Zip → 提取到当前文件夹** | 把压缩包里的文件直接解压到压缩包所在的位置 | 压缩包里只有几个文件，你想让它们和压缩包待在一起 |
| **7-Zip → 提取到 "文件夹名\\"** | 自动创建一个和压缩包同名的文件夹，把文件解压进去 | **最推荐**，文件不会散落一地，找起来方便 |
| **7-Zip → 提取文件...** | 弹出一个窗口，让你手动选择解压到哪里 | 你想指定一个特定的位置 |

**简单原则**：不知道选哪个时，就选**"提取到 '文件夹名\\'"**，这样解压出来的文件会整整齐齐地放在一个文件夹里，不会和桌面上其他东西混在一起。

### 创建压缩包时的选项

右键文件 → 7-Zip → "添加到压缩包"，会弹出一个设置窗口：

- **压缩格式**：选 .zip（通用，对方不需要装任何软件就能打开）或 .7z（压缩率更高）
- **压缩等级**：日常用"正常"即可，"极限"压缩耗时更长但体积减少有限
- **加密/密码**：在右侧"加密"区域输入密码，重要文件建议用 .7z 格式加密

---

## 五、其他格式与 7-Zip

当你拿到 .rar 格式的文件时，Windows 无法直接打开，需要安装第三方软件。.7z 格式在较新的 Windows 11 中已原生支持，但如果遇到解压失败或需要设密码等高级功能，同样需要 7-Zip。推荐 **[7-Zip](https://7-zip.org)**（免费、开源、无广告，安装时不会附带其他软件，使用过程中也没有弹窗打扰）。7-Zip 支持解压 .rar，但无法创建 .rar 文件（RAR 是专有格式），日常不需要——发给别人时优先用 .zip 或 .7z。

### 双击打开

安装 7-Zip 后，双击 .rar、.7z 文件会自动用 7-Zip 打开，不需要额外设置。

### 什么时候需要设密码

如果你有一些文件不想让别人随便看，可以在创建压缩包时设置密码。右键文件 → 7-Zip → "添加到压缩包" → 在右侧"加密"区域输入密码。注意：.zip 格式的加密方式因工具而异，为确保安全，重要文件建议用 .7z 格式加密。

## 六、收到分卷压缩包怎么办

有时候你会看到一组文件名很像的压缩包，名字后面带着编号：

```text
资料.001
资料.002
资料.003
```

或者类似 `资料_part1`、`资料_part2` 这样的命名。

这叫**分卷压缩**——因为文件太大或为了方便传输，压缩时被拆成了好几份。解压时，你只需要做一件事：**把所有分卷放在同一个文件夹里，然后右键点击第一个分卷（编号最小的那个）选择解压**。解压软件会自动找到同文件夹里的其他分卷，把它们拼成完整的文件。

**常见的问题**：

- **只下载了部分分卷**：从网盘下载时容易漏掉某一个，解压时会提示"缺少分卷"。回到下载页面检查是否全部下载完了，尤其是最后一个分卷不要漏掉

## 七、压缩包损坏或提示密码错误怎么办

### 压缩包损坏

解压时如果提示"压缩包已损坏"或"文件校验错误"，常见原因和解决办法：

- **下载不完整**：重新下载一遍，尤其是大文件容易下载中断
- **传输过程中损坏**：让对方重新发一次，或换种方式传输（如用网盘代替聊天软件直接发）
- **尝试修复**：部分压缩软件带有修复功能，但成功率不高。最靠谱的办法还是重新下载或让对方重新发送

### 提示密码错误

- 先确认密码是否输入正确，注意**大小写**（Password 和 password 是不一样的）
- 确认没有多打或少打空格
- 如果密码是对方给的，确认对方有没有发错密码或发错压缩包
- 仍然解不开，大概率是压缩包本身有问题，联系发送方重新提供

## 八、压缩文件的使用场景

| 场景 | 怎么做 |
| ---- | ------ |
| 把多个文件发给别人 | 选中文件 → 右键 → 发送到 → 压缩文件夹，然后把 .zip 文件发过去 |
| 收到别人发来的 .zip | 右键 → 全部提取 |
| 收到 .rar 或 .7z | 安装 7-Zip 后双击即可 |
| 打包文件节省空间 | 压缩为 .7z 格式（压缩率最高），但发给别人时优先用 .zip（兼容性最好） |
| 给文件加密码保护 | 用 7-Zip 创建压缩包时设置密码 |

### 试一试

1. 在桌面新建一个文件夹，里面放几个文件（比如复制几张图片或一个文本文件进去）
2. 右键这个文件夹 →"发送到"→"压缩(zipped)文件夹"，创建一个 .zip 文件
3. 把 .zip 文件复制到桌面另一个位置，右键 →"全部提取"，看看解压出来的文件是否和原来一样
4. 如果你有 7-Zip，尝试用 7-Zip 创建一个带密码的 .7z 压缩包，再试着解压它

## 九、收到陌生压缩包怎么办

### 场景：同学/好友发给你一个压缩包，说"帮我看看这个文件"

**不太建议的做法：**

- **不建议** 直接双击打开压缩包查看内容（双击可能直接触发恶意文件运行）
- **不建议** 解压后直接运行里面的 .exe 文件（.exe 是可执行程序，运行后它能在你的电脑上做任何事）
- **不建议** 因为"是同学发的"就放松警惕（同学的电脑可能已经中毒，病毒自动群发压缩包）

**建议的做法：**

1. 先问清楚这个压缩包是什么内容，确认对方确实发过
2. 右键压缩包 → 用杀毒软件扫描一下（Windows Defender 右键菜单里通常有"扫描"选项）
3. 确认安全后，解压到一个单独的文件夹，先查看里面有什么文件
4. 如果里面有 .exe 文件但你预期的是文档或图片，提高警惕，先不要急着运行
5. 不确定时，把压缩包上传到 [VirusTotal](https://www.virustotal.com) 在线扫描，让多个杀毒引擎一起检查

**一个简单的判断原则：** 正常的资料分享通常是 .pdf、.docx、.jpg、.mp4 这些常见格式。如果压缩包里突然出现 .exe、.bat、.scr 这些可执行文件，而对方说是"照片"或"文档"，那多半有问题。

## 十、注意事项

- **压缩不是加密**。设了密码才需要密码解压，没设密码的压缩包谁都能打开。压缩包里的敏感文件在发送前应单独考虑加密方式
- **解压后有乱码？** 部分压缩包创建时用的字符集和当前系统不一致，文件名会显示为乱码（技术上也叫"编码"问题，感兴趣可以搜索"字符编码"了解）。不影响文件内容，尝试用 7-Zip 解压通常能解决

> **编码小知识**：编码是电脑存储文字时用的"翻译规则"。同一个中文字，用 UTF-8 编码保存，用不同的编码打开，就会显示成乱码或问号（如 `å•Šå“ˆ`）。
