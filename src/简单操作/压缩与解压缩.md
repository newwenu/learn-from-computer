# 压缩与解压缩：怎么打开和创建压缩文件

> 阅读前提：你已经掌握了文件的基本操作（新建、复制、移动、删除），参见《日常使用》的「文件管理」部分。

---

## 一、什么是压缩文件

压缩文件是把一个或多个文件打包成一个文件，同时让体积变小，方便传输。

举个例子：你要把 10 个 Word 文档发给别人。一个个发很麻烦，打包成一个 .zip 文件，既省事又变小了。

压缩**不会损失文件内容**——解压缩后得到的是和原来一模一样的文件。它只是改变了文件的存储方式，就像把被子抽掉空气压缩进真空袋，拿出来抖一抖就恢复了。

常见的压缩格式：

| 格式 | 特点 | Windows 能否直接打开 |
| ---- | ---- | ---- |
| .zip | 最通用，所有系统都支持 | 能（无需额外软件） |
| .7z | 压缩率最高，开源免费 | 新版 Windows 11 原生支持（旧版需安装 7-Zip），部分特殊压缩包也可能仍需要 7-Zip |
| .rar | 压缩率较高，WinRAR 专有格式 | 不能 |

---

## 二、打开 .zip 文件

Windows 10/11 原生支持 .zip 格式，不需要安装任何软件。

### 方法一：直接查看内容

双击压缩文件，它会像文件夹一样在文件管理器中打开。你可以直接选中里面的文件复制出来。

### 方法二：提取全部文件

右键压缩文件 → 选择「全部提取」。在弹出的窗口中选择提取到的位置（保持默认即可），点击「提取」。系统会在压缩文件旁边创建一个同名文件夹，里面就是解压后的所有文件。

---

## 三、创建压缩文件

选中你想打包的文件或文件夹 → 右键 →「发送到」→「压缩(zipped)文件夹」。系统会在同一位置生成一个 .zip 文件，名字默认是其中某个文件的名称，你可以按 F2 重命名。

创建压缩包后，原始文件还在原位置。如果你确认压缩包没问题，可以删除原始文件。

---

## 四、其他格式与 7-Zip

当你拿到 .rar 格式的文件时，Windows 无法直接打开，需要安装第三方软件。.7z 格式在较新的 Windows 11 中已原生支持，但如果遇到解压失败或需要设密码等高级功能，同样需要 7-Zip。推荐 **[7-Zip](https://7-zip.org)**（免费、开源、无广告，与其他同类软件相比没有捆绑或者弹窗问题）。7-Zip 支持解压 .rar，但无法创建 .rar 文件（RAR 是专有格式），日常不需要——发给别人时优先用 .zip 或 .7z。

### 安装后

右键菜单会新增几个选项：

- **7-Zip → 提取到当前文件夹**（把压缩包里的文件提取到当前位置）
- **7-Zip → 提取到 "文件夹名\\"**（提取到一个同名子文件夹里，推荐）
- **7-Zip → 添加到压缩包**（创建压缩文件，可以选择格式、设置密码）

安装 7-Zip 后，双击 .rar、.7z 文件也会自动用 7-Zip 打开，不需要额外设置。

### 什么时候需要设密码

右键 → 7-Zip → 添加到压缩包 → 在「加密」一栏输入密码。注意：.zip 格式的加密安全性较弱，如果文件比较重要，建议用 .7z 格式加密。

---

## 五、压缩文件的使用场景

| 场景 | 怎么做 |
| ---- | ------ |
| 把多个文件发给别人 | 选中文件 → 右键 → 发送到 → 压缩文件夹，然后把 .zip 文件发过去 |
| 收到别人发来的 .zip | 右键 → 全部提取 |
| 收到 .rar 或 .7z | 安装 7-Zip 后双击即可 |
| 打包文件节省空间 | 压缩为 .7z 格式（压缩率最高），但发给别人时优先用 .zip（兼容性最好） |
| 给文件加密码保护 | 用 7-Zip 创建压缩包时设置密码 |

## 六、收到陌生压缩包怎么办

### 场景：同学/好友发给你一个压缩包，说"帮我看看这个文件"

**不太建议的做法：**

- ❌ 直接双击打开压缩包查看内容（双击会触发预览，某些恶意文件可能利用预览漏洞）
- ❌ 解压后直接运行里面的 .exe 文件（.exe 是可执行程序，运行后它能在你的电脑上做任何事）
- ❌ 因为"是同学发的"就放松警惕（同学的电脑可能已经中毒，病毒自动群发压缩包）

**建议的做法：**

1. 先问清楚这个压缩包是什么内容，确认对方确实发过
2. 右键压缩包 → 用杀毒软件扫描一下（Windows Defender 右键菜单里通常有"扫描"选项）
3. 确认安全后，解压到一个单独的文件夹，先查看里面有什么文件
4. 如果里面有 .exe 文件但你预期的是文档或图片，提高警惕，先不要运行
5. 不确定时，把压缩包上传到 [VirusTotal](https://www.virustotal.com) 在线扫描，让多个杀毒引擎一起检查

**一个简单的判断原则：** 正常的资料分享通常是 .pdf、.docx、.jpg、.mp4 这些常见格式。如果压缩包里突然出现 .exe、.bat、.scr 这些可执行文件，而对方说是"照片"或"文档"，那多半有问题。

---

## 七、注意事项

- **压缩不是加密**。设了密码才需要密码解压，没设密码的压缩包谁都能打开。压缩包里的敏感文件在发送前应单独考虑加密方式
- **解压后有乱码？** 部分压缩包使用的编码与当前系统不一致，文件名会显示为乱码。不影响文件内容，尝试用 7-Zip 解压通常能解决
