# 文件资源管理器：管理文件的"总部"

> 阅读前提：你已经了解了文件和文件夹的基本概念（参见《日常使用》的"文件管理"部分）。

## 一、打开文件资源管理器

- 按 **Win+E**
- 或双击桌面上的"此电脑"图标
- 或点击任务栏上的文件夹图标 [查]

打开后，左侧是导航栏（快速访问、此电脑、桌面、文档、下载等），右侧是文件列表。

## 二、基本操作

> 复制、移动、删除、选中多个文件、新建文件夹等操作，参见《日常使用》的"文件管理"部分。这里重点讲文件资源管理器特有的功能和背后的原理。

### 查看文件

- **切换视图**：点击右上角的图标，可以在"详细信息"（显示文件名、大小、修改日期）和"大图标/平铺"（看图方便）之间切换
- **排序**：点击列标题（如"名称""修改日期"）可以按该列排序，再点一次反向排序
- **预览**：选中文件后，右侧或下方可能显示预览（图片、文档内容等）

## 三、地址栏与路径

地址栏显示你当前所在的位置，如 `此电脑 > 本地磁盘 (C:) > Users > 用户名 > Documents`。点击地址栏中的某一级可以快速跳转到该位置。

你也可以直接在地址栏输入路径跳转，比如输入 `%temp%` 回车，直接打开临时文件夹。

## 四、显示隐藏文件和扩展名

默认情况下，Windows 隐藏了系统文件和已知文件类型的扩展名。建议打开显示：

- **显示扩展名**：参见《日常使用》的"显示文件扩展名"部分
- **显示隐藏文件**：文件资源管理器顶部"查看" → 勾选"隐藏的项目" [查]

### 为什么显示扩展名很重要

扩展名是文件名中最后一个点号后面的部分，比如 `报告.docx` 中的 `docx`。它告诉 Windows 这个文件应该用什么软件打开。

**但更重要的是——显示扩展名能帮你识别伪装成文档的病毒。**

想象一下这个场景：你收到一个文件，名字叫"高数速成秘籍.doc"，图标也是 Word 文档的样子。如果不显示扩展名，它看起来就是一个普通的 Word 文档。

但如果你打开了扩展名显示，可能会发现它的真实名字是：

```
高数速成秘籍.doc.exe
```

`.exe` 表示这是一个可执行程序（软件），不是文档。有人故意把图标做成 Word 的样子来骗你双击运行。一旦运行，它就能在你的电脑上做任何事——偷密码、锁文件、装病毒。

**所以，显示扩展名是一个很重要的安全习惯。** 当你看到 `.exe`、`.bat`、`.scr` 这些可执行扩展名时，即使图标看起来像文档，也要提高警惕。

### 扩展名不能随便改

如果你把一个 `报告.docx` 改成 `报告.txt`，系统就会用记事本去打开它——结果当然是一堆乱码。因为扩展名告诉系统"用什么工具打开"，改了扩展名，系统就换工具了。

如果你确实想改（比如把 `jpg` 图片改成 `png`），需要用专门的转换工具，而不是直接重命名改扩展名。

## 五、快速访问

左侧导航栏的"快速访问"会自动显示你最近打开和经常使用的文件夹。你可以把常用的文件夹固定到这里：右键文件夹 →"固定到快速访问"。

## 六、搜索文件

在文件资源管理器右上角的搜索框输入文件名或关键词，可以在当前文件夹及其子文件夹中搜索。如果搜索慢，可以限定在当前文件夹（不搜子文件夹）或按文件类型筛选（如 `*.pdf`）。

## 七、路径是什么

地址栏里那一串 `此电脑 > 本地磁盘 (C:) > Users > 用户名 > Documents`，就是**路径**——它告诉你"我现在在哪个位置"。

你可以把电脑里的文件夹想象成一棵倒着长的树：

- **树根**在最上面，是"此电脑"
- **树干**是 C 盘、D 盘这些分区
- **树枝**是一层一层的文件夹
- **树叶**是具体的文件

这种组织方式在计算机里叫做"目录树"，感兴趣可以搜索了解。比如这个路径：

```
D:\学习资料\高数\第三章.docx
```

表示：在 D 盘（树干）→ "学习资料"文件夹（树枝）→ "高数"文件夹（小树枝）→ 里面有个叫"第三章.docx"的文件（树叶）。

`\` 是分隔符，就像地址里的"省/市/区"一样，一级一级往下找。

### 路径的两种写法

**绝对路径**：从盘符开始写完整路径，比如 `C:\Users\小明\Desktop\报告.docx`。不管你在哪里，这个路径指向的都是同一个文件。

**相对路径**：相对于你当前所在的位置。比如你现在在 `C:\Users\小明` 这个文件夹里，那么 `Desktop\报告.docx` 就表示"当前文件夹下的 Desktop 文件夹里的报告.docx"。

> 日常使用中，你不需要手写路径，但理解路径的概念能帮你：看懂地址栏里的位置、理解"为什么这个文件找不到了"（可能路径变了）、使用命令行时更准确。想了解更多可以搜索"绝对路径 相对路径"自行探索。

## 八、快捷方式不是软件本身

桌面上那些软件图标，大部分都不是软件本身，而是**快捷方式**——一个指向软件真实位置的"路标"。

### 怎么区分快捷方式和真实文件

快捷方式的图标左下角通常有一个**小箭头**（↗）。右键它 →"属性"，你会看到"目标"一栏，里面写的是软件真正的位置，比如：

```
C:\Program Files\网易云音乐\cloudmusic.exe
```

### 重要：删除快捷方式 ≠ 卸载软件

桌面上的图标只是"路标"，不是软件本身——删掉图标不等于卸载软件。删掉快捷方式只是删掉了"路标"，软件还好好地躺在 `C:\Program Files` 里。

要真正卸载软件，应该去"设置 → 应用 → 已安装的应用"里卸载。

### 自己创建快捷方式

如果你经常打开某个文件夹或文件，可以给它创建快捷方式放到桌面：

右键文件/文件夹 →"发送到" →"桌面快捷方式"

这样你就不用每次都去层层文件夹里找了。

## 九、文件的打开方式

每个类型的文件都有默认的"打开方式"——双击它时，系统用哪个软件来打开。

| 扩展名 | 默认打开方式 | 说明 |
|--------|-------------|------|
| .txt | 记事本 | 纯文本 |
| .docx | Word / WPS | Word 文档 |
| .xlsx | Excel / WPS | 电子表格 |
| .pptx | PowerPoint / WPS | 演示文稿 |
| .pdf | Edge 浏览器 / PDF 阅读器 | 电子文档（不可编辑） |
| .jpg / .jpeg / .png | 照片应用 | 图片 |
| .gif | 照片应用 / 浏览器 | 动态图片 |
| .mp4 / .avi / .mkv | 电影和电视 / 播放器 | 视频 |
| .mp3 / .wav / .flac | 媒体播放器 | 音频 |
| .zip / .rar / .7z | 压缩软件 | 压缩包 |
| .exe | 直接运行 | 软件程序 |
| .msi | 直接运行 | 软件安装包 |
| .html / .htm | 浏览器 | 网页文件 |
| .md | 记事本 / 专用编辑器 | Markdown 文档 |

### 临时换软件打开

有时候你想用其他软件打开一个文件（比如用画图而不是照片应用打开图片）：

右键文件 →"打开方式" → 选择你要用的软件。

如果列表里没有，选"选择其他应用"，然后在列表里找到你要用的软件。

**注意**：如果勾选"始终使用此应用打开"，系统就会记住这个选择，以后双击都用这个软件打开。

### 打开方式突然变了怎么办

有时候 Windows 更新后，你的 PDF 突然用 Edge 打开了，而不是原来的阅读器。这是因为系统的打开方式被修改了（技术上也叫"文件关联表"被修改，感兴趣可以搜索了解）。

修复方法：右键文件 →"打开方式" →"选择其他应用" → 选回原来的软件 → 勾选"始终使用此应用打开"。

## 十、管理好你的文件

> 关于文件命名建议和文件夹分类结构的基本思路，参见《日常使用》的"文件管理"部分。这里从"为什么这样更好"的角度补充一些原理。

### 为什么桌面不适合长期存放文件

桌面看起来方便，但有几个隐患：

1. **桌面文件实际在 C 盘**（`C:\Users\你的用户名\Desktop`），堆满桌面等于堆满 C 盘
2. **桌面文件太多会拖慢开机速度**，因为系统启动时需要加载更多内容
3. **重装系统时 C 盘会被清空**，桌面文件如果没有备份就会丢失

所以桌面只放"最近几天正在处理"的文件，处理完就移到正式位置。

### 为什么建议按"用途"而不是"文件类型"分类

有些人喜欢这样分类：

```
D:\
  ├── 文档/
  ├── 图片/
  ├── 视频/
  └── 音乐/
```

这样分的问题在于：你的一份"课程作业"可能包含 Word 文档、截图、录屏，它们会被拆到三个不同的地方。

更好的方式是按"用途/项目"分类，每个项目内部再按类型细分：

```
D:\
  ├── 学习/
  │   ├── 高数/
  │   │   ├── 笔记.docx
  │   │   ├── 老师课件.pdf
  │   │   └── 错题截图/
  │   └── 英语/
  └── 工作/
      └── 项目A/
          ├── 方案.docx
          ├── 预算.xlsx
          └── 会议录音/
```

这样当你想找"高数相关的一切"时，只需要打开"高数"文件夹。

### C 盘还是 D 盘——背后的原理

C 盘和 D 盘可能是同一块硬盘的不同分区，也可能是两块不同的硬盘。

- **同一块硬盘分区**：C 盘满了不会影响 D 盘的数据安全，但整个硬盘的物理寿命是共享的
- **两块不同硬盘**：如果 C 盘是 SSD（固态硬盘），D 盘是 HDD（机械硬盘），那么 C 盘速度更快但价格更贵，D 盘容量大但速度慢。把常用软件放 C 盘、大文件（电影、备份）放 D 盘是合理的选择。想了解更多硬盘类型的区别，可以搜索"SSD HDD 区别"自行探索。

如果你不确定自己的 C 盘和 D 盘是不是同一块硬盘，可以打开"任务管理器"→"性能"→"磁盘"查看 [查]。
