# 文件资源管理器：管理文件的"总部"

> 阅读前提：你已经了解了文件和文件夹的基本概念（参见《日常使用》的"文件管理"部分）。

文件资源管理器是 Windows 里用来管理文件和文件夹的工具。你平时复制文件、打开图片、整理资料，经常通过它完成。

它由几个主要区域组成：

- **左侧**：导航栏，快速跳转到常用位置
- **右侧**：文件列表，显示当前文件夹里的内容
- **顶部**：地址栏和工具栏，显示你现在在哪，也提供各种操作按钮
- **右上角**：搜索框，用来找文件

**怎么打开**：
- 按 **Win+E**
- 或双击桌面上的"此电脑"图标
- 或点击任务栏上的文件夹图标 [查]

刚开始看到这些区域可能会觉得复杂，不用担心——**多使用就会熟悉**。下面介绍几个重要的功能和概念。

---

## 一、认识界面

### 标签页

和浏览器一样，Win11 文件资源管理器也支持**标签页**。你可以在一个窗口里打开多个文件夹标签，不用开一堆窗口。

**常用操作**：
- **新建标签**：点标签栏右侧的 **+** 按钮
- **关闭标签**：点标签上的 **×**
- **切换标签**：直接点要看的标签

> 快捷键：**Ctrl+T** 新建标签、**Ctrl+W** 关闭标签、**Ctrl+Tab** 切换标签

### 地址栏和搜索框

地址栏显示你当前的位置，比如 `此电脑 > 本地磁盘 (C:) > Users > 用户名 > Documents`。地址栏右侧是**搜索框**，输入文件名就能搜索。

**地址栏的用法**：
- **点某一级**：快速跳转到该位置
- **点地址栏空白处**：地址栏变成可输入状态，直接输路径跳转
- **按 F4**：地址栏变成可输入状态（和点击空白处效果一样）

**地址栏左侧的导航按钮**：
- **← →**：后退和前进，回到你之前浏览过的位置
- **↑**：向上，返回上一级文件夹
- **↻**：刷新，重新加载当前文件夹的内容（如果文件有变化但没显示，点一下刷新）


### 工具栏

地址栏下方有一条工具栏，包含一排常用操作按钮（剪切、复制、粘贴、重命名、删除、新建文件夹等），还有"查看"等菜单。

**常用功能**：
- **基本操作**：选中文件后，点工具栏上的按钮进行复制、粘贴、重命名、删除等操作
- **新建文件夹**：点"新建文件夹"按钮，快速创建新文件夹
- **切换视图**：点"查看"菜单，选不同的显示方式：
  - **详细信息**：显示文件名、大小、修改日期等，适合整理文件
  - **大图标/平铺**：看图方便，适合浏览图片
- **显示/隐藏**：点"查看"→"显示"，可以开关预览窗格、文件扩展名、隐藏的项目

> **提示**：工具栏上的按钮有限，更多操作可以**右键文件**，在右键菜单里找到。

### 左侧导航栏

导航栏是你快速跳转位置的"地图"：

- **快速访问**：显示最近打开和经常使用的文件夹。右键文件夹 →"固定到快速访问"，常用文件夹会固定在这里。不需要了可以右键 →"从快速访问取消固定"
- **此电脑**：显示所有盘符（C盘、D盘等）
- **主文件夹**：显示你的个人文件夹（桌面、文档、下载、图片等）
- **网络**：局域网里的其他电脑或设备

### 右侧文件列表

显示当前文件夹里的内容，是你最常用的操作区域：

- **打开文件/文件夹**：双击打开
- **选中多个**：按住 **Ctrl** 点选多个**分开的**文件，或按住 **Shift** 选中**相邻的**多个文件
- **排序**：点列标题（如"名称""修改日期""大小"）按该列排序，再点一次反向排
- **切换视图**：点"查看"菜单，选不同的图标大小
- **右键菜单**：右键文件，可以看到复制、剪切、删除、重命名、属性等常用操作
- **拖拽整理**：把文件拖到左侧导航栏的文件夹里，可以整理文件位置（同一盘符内是移动，不同盘符是复制）

## 二、路径：文件的"地址"

地址栏里那一串位置信息，就是**路径**——它告诉你"我现在在哪"，也用来定位一个文件或文件夹。

Windows 路径用反斜杠 `\`（向左倾斜的线）分隔每一级：

```
D:\学习资料\高数\第三章.docx
```

表示：D盘 → "学习资料"文件夹 → "高数"文件夹 → "第三章.docx"文件。

> **注意**：Windows 用反斜杠 `\`，网址（URL）用正斜杠 `/`，两者方向不同，别混用。

理解路径的概念能帮你：看懂地址栏里的位置、明白"为什么这个文件找不到了"（可能路径变了）。

## 三、显示隐藏文件和扩展名

默认情况下，Windows 隐藏了系统文件和已知文件类型的扩展名。建议打开显示：

- **显示扩展名**：参见《日常使用》的"显示文件扩展名"部分
- **显示隐藏文件**：点"查看"→"显示"→"隐藏的项目" [查]

### 为什么显示扩展名很重要

**显示扩展名能帮你识别伪装成文档的病毒。**

比如收到一个叫"高数速成秘籍.doc"的文件，图标也是 Word 的样子。如果不显示扩展名，它看起来就是普通文档。但如果显示了扩展名，你可能会发现它实际是：

```
高数速成秘籍.doc.exe
```

`.exe` 表示这是软件程序，双击就会运行，不是文档。有人故意把图标做成 Word 的样子来骗你双击运行。**所以看到 `.exe` 结尾的文件，即使图标像文档，也不要轻易打开。**

### 扩展名不能随便改

如果你把一个 `报告.docx` 改成 `报告.txt`，系统就会用记事本去打开它——结果当然是一堆乱码。因为扩展名告诉系统"用什么工具打开"，改了扩展名，系统就换工具了。

如果你确实想改（比如把 `jpg` 图片改成 `png`），需要用专门的转换工具，而不是直接重命名改扩展名。

## 四、快捷方式不是软件本身

桌面上**很多**软件图标，其实是**快捷方式**——一个指向软件真实位置的"路标"。

### 怎么区分快捷方式和真实文件

快捷方式的图标左下角通常有一个**小箭头**。右键快捷方式图标 →"属性"，你会看到"目标"一栏，里面写的是软件真正的位置，比如：

```
C:\Program Files\网易云音乐\cloudmusic.exe
```

### 重要：删除快捷方式 ≠ 卸载软件

桌面上的图标只是"路标"，不是软件本身——删掉图标不等于卸载软件。删掉快捷方式只是删掉了"路标"，软件还好好地躺在 `C:\Program Files` 里。

要真正卸载软件，应该去"设置 → 应用 → 已安装的应用"里卸载。

### 自己创建快捷方式

如果你经常打开某个文件夹或文件，可以给它创建快捷方式放到桌面：

右键文件/文件夹 →"显示更多选项"→"发送到"→"桌面快捷方式"（如果右键菜单直接显示"发送到"，可以直接点）

这样你就不用每次都去层层文件夹里找了。

## 五、搜索文件

点右上角的搜索框，输入文件名或关键词，即可在当前文件夹及其子文件夹中搜索。

**如果搜索慢**：
- Windows 自带的搜索功能不算快，尤其是文件夹内容多的时候
- 如果经常需要搜索文件，推荐使用专门的搜索工具（参见[《常用工具软件推荐》的"文件搜索"部分](../附录/常用工具软件推荐.md#文件搜索everything)）

## 六、管理好你的文件

> 关于文件命名建议和文件夹分类结构，参见《日常使用》的"文件管理"部分。这里补充几个原理。

### 为什么桌面不适合长期存放文件

桌面看起来方便，但有几个隐患：

1. **桌面文件实际在 C 盘**（`C:\Users\你的用户名\Desktop`），堆满桌面等于堆满 C 盘
2. **桌面文件太多会让桌面操作变卡**，返回桌面、刷新图标或加载缩略图时会变慢
3. **重装系统时 C 盘会被清空**，桌面文件如果没有备份就会丢失

所以桌面只放"最近几天正在处理"的文件，处理完就移到正式位置。

### 系统盘还是其他盘

C 盘通常是**系统盘**（安装 Windows 的盘），里面放着系统文件和大部分软件。如果你有其他盘（如 D 盘、E 盘），建议把**大文件（电影、备份）放在其他盘**，给 C 盘留出空间，这样系统运行更顺畅。

## 快捷键汇总

| 快捷键          | 作用                 | 场景         |
| --------------- | -------------------- | ------------ |
| Win+E           | 打开文件资源管理器   | 随时         |
| F2              | 重命名选中文件       | 选中文件后   |
| F4              | 地址栏变为可输入状态 | 需要输路径时 |
| Ctrl+C / Ctrl+V | 复制 / 粘贴          | 选中文件后   |
| Ctrl+X / Ctrl+V | 剪切 / 移动          | 选中文件后   |
| Delete          | 移到回收站           | 选中文件后   |
| Shift+Delete    | 彻底删除             | 选中文件后   |
| Ctrl+A          | 全选                 | 在文件列表中 |
| Ctrl+Shift+N    | 新建文件夹           | 在文件列表空白处 |
| Alt+↑           | 返回上一级文件夹     | 导航时       |
| Alt+← / Alt+→   | 后退 / 前进          | 导航时       |

> 部分快捷键在《日常使用》已提过，这里作为汇总。
