# 常见故障排查思路：出问题时怎么办

> 阅读前提：本文部分排查方法会用到任务管理器，如不熟悉可先阅读《理解你的操作系统》的"任务管理器"部分。

---

## 一、通用原则：先试这三步

遇到任何电脑问题，先按顺序做这三件事，大部分问题在前两步就解决了：

1. **重启电脑** — 这不是敷衍。很多临时性错误（程序卡住、网络连不上、U 盘不识别）在重启后自动消失。重启是最安全、最有效、最值得尝试的第一步。
2. **搜一下错误信息** — 把报错窗口上的文字完整复制下来，粘贴到搜索引擎或 AI 工具里。几乎可以肯定有人遇到过并给出了解决办法。
3. **判断是否需要求助** — 如果搜到的答案看不懂（涉及注册表、命令行、修改系统文件），不要硬试。**看不懂的操作不要碰，这是最重要的原则。**

需要求助时，找对人和说清问题同样重要：

- **懂电脑的朋友或同事** — 描述你做了什么操作、看到了什么报错
- **如果身边没有懂的人** — 把错误信息完整复制或拍照，去搜索引擎或 AI 工具搜索；或在靠谱的电脑论坛/社区发帖，附上错误截图和你的操作步骤

## 二、程序卡死（无响应）怎么办

程序突然卡住，鼠标点不动，窗口标题栏出现"无响应"。

### 尝试等待

先等 30 秒到 1 分钟。有些程序只是临时忙（比如处理大文件、加载网页），等一会儿就恢复了。

### 强制结束程序

如果等了之后仍然完全没反应：

1. 按 **Ctrl+Shift+Esc** 打开任务管理器
2. 在"进程"列表里找到卡住的程序（通常 CPU 或内存占用异常高）
3. 选中它，点击右下角的"结束任务"

程序会强制关闭。没有保存的数据会丢失，所以这个操作只应该在程序确定死机时用。

### 什么时候避免结束任务

- 系统进程（名称看起来像系统服务、以 .exe 结尾但你从没手动打开过的）——结束它们可能导致系统不稳定，重启电脑比乱关系统进程安全
- 只是"有点慢"但没有显示"无响应"的程序——再等一下

如果某个程序频繁卡死，可能是软件本身的问题（版本太旧、不兼容），可以在搜索引擎中搜"程序名 + 卡死"看看是否有人有同样的反馈。

## 三、常见场景排查清单

### 场景 1：电脑开机后黑屏，但风扇在转

1. 检查显示器电源灯是否亮着
2. 检查显示器和主机之间的视频线（HDMI/DP/VGA）是否插紧
3. 按显示器上的菜单键，手动切换输入源（有些显示器有多个接口，需要选到正确的那个）
4. 以上都正常？可能是显卡或内存接触不良，需要找人帮忙

### 场景 2：Wi-Fi 显示已连接，但打不开网页

先判断是"某个网站打不开"还是"所有网站打不开"：
- 某个网站打不开 → 可能是网站本身问题，过会儿再试
- 所有网站打不开 → 继续排查

**快速排查（不需要命令行）：**

1. 试试微信或 QQ 能不能登录
   - 能登录微信，但打不开网页 → 通常是 DNS 问题，见下方"修改 DNS"
   - 微信也登录不了 → 网络本身有问题，重启路由器或联系运营商

2. 重启路由器（拔掉电源等 10 秒再插回）

**如果确定是 DNS 问题，修改 DNS 服务器：**

1. 打开"设置"→"网络和 Internet"→"高级网络设置"→"更多网络适配器选项"
2. 右键你正在使用的网络（Wi-Fi 或以太网）→"属性"
3. 双击"Internet 协议版本 4 (TCP/IPv4)"
4. 选择"使用下面的 DNS 服务器地址"：
   - 首选：`223.5.5.5`（阿里）或 `119.29.29.29`（腾讯）
   - 备用：`114.114.114.114`（国内通用）或 `223.6.6.6`（阿里备用）
5. 点击"确定"，重启浏览器试试

> `[查]` 不同 Windows 版本设置路径可能不同，建议结合搜索最新教程操作。

**会用命令行？可以进一步定位：**

按 `Win` + `R` 输入 `cmd` 打开命令行：

1. 输入 `ipconfig` 查看网络信息
   - 找到"默认网关"，后面的数字就是路由器 IP（通常是 `192.168.1.1` 或 `192.168.0.1`）
   - 如果"IPv4 地址"以 `169.254` 开头 → 没拿到 IP，检查网线/Wi-Fi 或重启路由器

2. 输入 `ping` 加上刚才看到的默认网关（如 `ping 192.168.1.1`）
   - 显示"回复" → 到路由器连接正常，问题在路由器之外
   - 显示"请求超时" → 电脑到路由器有问题，重启路由器

3. 如果到路由器正常，输入 `ping 114.114.114.114`
   - 显示"回复" → 网络是通的，可能是 DNS 问题
   - 显示"请求超时" → 路由器到互联网断了，联系运营商

### 场景 3：软件安装后打不开

1. 右键软件图标 →"属性" →"兼容性"，看看是否需要以兼容模式运行
2. 右键软件图标 →"以管理员身份运行"试试
3. 检查软件是否支持你当前的系统类型（32 位软件可以在 64 位系统运行，反之不行）
4. 以上都不行？搜索"软件名 + 打不开 + 错误提示"

### 场景 4：电脑突然变得很慢

1. 按 Ctrl+Shift+Esc 打开任务管理器
2. 看"进程"选项卡，点击"CPU"列头排序，找出占用最高的程序
3. 如果是你认识的程序（如浏览器开了 50 个标签页）→ 关闭一些标签页或窗口
4. 如果是你不认识的程序名 → 先不要急着结束它，把名字复制到搜索引擎查一下是什么
5. 如果"磁盘"列显示 100% 且持续不降 → 可能是硬盘老化或某个程序在疯狂读写，重启试试

## 四、什么情况该重启，什么情况该找人

**这些情况，重启通常能解决：**

某个程序没反应 —— 按上面说的结束任务，或者直接重启。

新装软件/更新后出问题 —— 先重启，重启后问题依旧就卸载刚装的软件或更新。

U 盘/外设插上没反应 —— 换个 USB 口试试，或者重启电脑。

**这些情况，重启后还需要额外操作：**

连不上网（Wi-Fi 图标有黄色感叹号）—— 重启电脑。不行就重启路由器。还不行就是宽带问题，联系网络服务商。

电脑突然变慢 —— 打开任务管理器看哪个程序占用高，关闭或重启该程序。

**这些情况，建议找人帮忙：**

蓝屏（屏幕变蓝、显示白色文字）—— 按提示的二维码或错误代码去搜索。**如果重启后能正常进入桌面，通常不需要额外处理**。

出现看不懂的英文错误 —— 把文字复制下或拍照来搜索或问AI。

> **谨慎操作**：如果搜到的解决方案涉及删除系统文件、修改注册表、运行不熟悉的命令行、下载第三方修复工具——这些操作有风险，不建议自行尝试。把错误信息拍照或复制，问懂电脑的朋友，或去搜一下有没有更安全的替代方案。

## 五、总结：一个简单的分流思路

```text
遇到问题
  ├─ 先重启 → 好了？→ 大概率不用管了
  │
  └─ 重启没好
       ├─ 搜索错误信息
       │    ├─ 搜到的答案看得懂且安全 → 按步骤做
       │    └─ 搜到的答案看不懂或要动系统文件 → 问懂电脑的朋友，或把错误信息发到论坛/AI 工具
       │
       └─ 没有错误信息（比如只是慢、卡）
            ├─ 打开任务管理器看看哪个程序占用高
            ├─ 搜"软件名 + 卡顿"看看是不是通病
            └─ 还不行就暂时不管，下次更新版本可能就好了
```

电脑出问题很常见，多数情况重启就能解决。**最重要的不是你会修，而是你知道什么时候该停手——不动自己不懂的东西，比盲目尝试更安全。**

## 六、如果以上方法都无效，如何求助

当你需要向他人求助时，提供以下信息能让对方更快帮你：

### 准备这些信息

1. **问题现象**：具体发生了什么？（如"开机黑屏，风扇在转""软件闪退，没有报错"）
2. **已尝试的方法**：你试过哪些解决步骤？（如"重启过两次，卸载重装过软件"）
3. **系统环境**：Windows 10 还是 11？笔记本还是台式机？什么品牌？
4. **截图**：如果有错误提示窗口，按 `Win+Shift+S` 截图保存
5. **隐私保护**：截图前打码个人信息（姓名、地址、手机号、账号密码）

### 去哪里求助

| 渠道                 | 适合场景                                    |
| -------------------- | ------------------------------------------- |
| **品牌官方客服**     | 笔记本/电脑硬件问题，在保修期内             |
| **搜索引擎/AI 工具** | 把错误信息完整粘贴，通常能找到解决方案      |
| **技术论坛**         | 如 V2EX、知乎、百度贴吧，附上截图和操作步骤 |
| **身边懂电脑的朋友** | 描述你做了什么操作、看到了什么报错          |

### 提问的技巧

- **不好的提问**："我的电脑坏了，怎么办？"
- **好的提问**："我的笔记本（联想小新 Pro14，Win11）今天开机后 Wi-Fi 图标消失了，重启两次没用，设备管理器里网卡显示黄色感叹号，请问该怎么排查？"

**说清问题比说"急"更有用。**
