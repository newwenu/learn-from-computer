# 数据备份与恢复：保护你的数字资产

> 阅读前提：你已经了解了文件管理的基本操作（参见[《日常使用》](../日常使用.md)的"文件管理"部分）。
>
> 如果你还没读过[《防患于未然：日常怎么保护电脑和账号》](../安全意识/日常安全习惯.md)的第六节"重要文件定期备份"，建议先去看一眼——那里讲了"为什么要备份"，本文讲"具体怎么做"。

---

## 一、先搞清楚：同步 ≠ 备份

"文件放在 OneDrive 里就是备份了"——这个想法可能有隐患。

### 同步是什么

同步（sync）的意思是：你在电脑上改了一个文件，云盘上的副本也会跟着改；你删了电脑上的文件，云盘上的副本也会被删掉。

**同步像一面镜子**。镜子里映出你现在的样子——你变了镜子里的你也变了，你消失了镜子里也看不到了。同步的好处是方便：多台设备之间保持一致。但同步不能替代备份——如果文件被误删、被病毒加密、被覆盖成错误版本，同步会让所有副本一起遭殃。

### 备份是什么

备份（backup）的意思是：在某一个时间点，把文件复制一份存到别的地方。之后如果你不去主动更新它，备份的那份就保持原样不变。

**备份像一张快照**。像拍照一样，按下快门的那一刻被永久定格。即使后来人变了、场景变了，照片里还是当初的样子。

### 为什么这个区别很重要

举个例子：

- 你不小心把一份重要文档删了，而这份文档也在 OneDrive 里同步
- OneDrive 发现本地文件没了，于是把云端的那份也删了
- 或者你打开文档改错了内容并保存，OneDrive 把错误版本同步到了云端，之前的正确版本就找不回来了

如果用的是真正的备份（比如定期复制到移动硬盘），硬盘里的旧版本还在——因为除非你主动更新它，备份就保持原样不变。

> **简单记忆法**：同步是为了"方便"，备份是为了"安全"。两者各有用途，但不能互相替代。

## 二、备份的核心原则：3-2-1

备份领域有一个广泛认可的原则，叫 **3-2-1 规则**：

- **3** 份数据副本（原始文件 + 2 个备份）
- **2** 种不同介质（比如电脑硬盘 + 移动硬盘，或电脑硬盘 + 云盘）
- **1** 份异地保存（防止火灾、盗窃等物理灾害同时毁坏所有本地副本）

听起来要求很高？对普通人来说，不必一开始就做到完美，可以从简化版开始。简化版是：

**重要的文件至少存在两个不同的地方。** 比如，你的论文同时存在电脑 D 盘和一个 U 盘里——这就是两个不同的地方。

什么叫"不同的地方"？以下通常算：

- 电脑硬盘 + U 盘 / 移动硬盘
- 电脑硬盘 + 云盘（OneDrive、百度网盘等）
- 电脑硬盘 + 另一台电脑（如果不在同一房间）

以下通常不算"不同的地方"：

- C 盘和 D 盘——如果它们在同一块物理硬盘上，硬盘故障时两个分区可能同时受影响

## 三、Windows 自带的备份工具

Windows 自带了一些备份工具，不需要额外安装软件就能使用。下面介绍两个最常用的。

### 3.1 文件历史记录

**这是什么**：自动跟踪你选中的文件夹，当你修改或删除文件时，它会把旧版本保存下来。类似于文档编辑软件里的"撤销"功能，但针对整个文件夹。

**适合场景**：误删了文件、覆盖了错误版本、需要找回几天前的某个版本。

**怎么开启**：

1. 插上一个 U 盘或移动硬盘（文件历史记录需要一个外部存储位置，不建议用 C 盘）
2. 打开文件历史记录：在开始菜单搜索"文件历史记录"并打开 `[查]`（也可以通过控制面板进入：开始菜单搜索"控制面板"→"系统和安全"→"文件历史记录"）
3. 点击"添加驱动器"，选择你的 U 盘或移动硬盘
4. 默认会备份"桌面""文档""图片""音乐""视频"这些文件夹
5. 点击"更多选项"可以调整备份频率（默认每小时）和保留文件的时长

开启后，它会安静地在后台工作，你几乎感觉不到它的存在。只要 U 盘或移动硬盘插着，它就会按设定的频率自动保存文件的旧版本。

**怎么恢复文件**：右键点击文件（或它所在的文件夹）→"属性"→"以前的版本" `[查]`，在弹出的列表中选择一个时间点的版本，点击"打开"查看内容，或点击"复制"保存到其他位置。

> 如果"以前的版本"列表为空，可能是文件历史记录未开启，或者这个文件不在受保护的文件夹范围内。可以在开始菜单搜索"文件历史记录"确认是否已开启。

**局限性**：

- 只能恢复文件，不能恢复系统设置或已安装的程序
- 需要一个始终连接的外部存储设备（U 盘拔了就停止工作）
- 如果文件在被备份之前就被删除，无法恢复

> **试一试**：如果你手边有 U 盘，现在就可以开启文件历史记录试试。插上 U 盘 → 在开始菜单搜索"文件历史记录"并打开 → 选择驱动器 → 开启。开启后不需要做任何额外操作，它会自动帮你保存文件的旧版本。

### 3.2 系统还原点

**这是什么**：给系统的关键部分拍一张"快照"。当装了某个软件或驱动之后电脑开始出问题时，可以把系统还原到之前正常的状态。

**适合场景**：装了可疑软件后电脑变慢或报错、更新驱动后蓝屏、某些系统设置被改乱了。

**怎么创建还原点**：

1. 开始菜单搜索"创建还原点"并打开 `[查]`（也可以右键"此电脑"→"属性"→"高级系统设置"→"系统保护"）
2. 在打开的"系统属性"窗口中，确认已切换到"系统保护"选项卡
3. 选择你要保护的磁盘（通常是 C 盘）
4. 点击"创建"，输入一个描述名称（如"装 Office 之前"）
5. 等待创建完成（通常几分钟）

**什么时候值得创建还原点**：在安装新软件、更新驱动、修改系统设置之前，顺手创建一个。这样万一出问题，可以回退到操作之前的状态。

> **试一试：** 现在创建一个还原点，给它起个容易记住的名字（比如"正常状态"）。以后遇到系统问题时，可以回退到这里。

**怎么使用还原点恢复**：

1. 开始菜单搜索"创建还原点"并打开 `[查]`（也可以右键"此电脑"→"属性"→"高级系统设置"→"系统保护"）
2. 点击"系统还原"
3. 选择一个还原点（列表会显示创建时间和描述）
4. 按提示确认，电脑会重启并恢复

**重要提醒**：

- 系统还原**主要影响系统和程序**，一般不动你的个人文件（文档、照片等）。所以还原后个人文件通常不会丢失
- 还原点是临时的——如果 C 盘空间紧张，Windows 可能会自动删除旧的还原点腾出空间
- 不是所有问题都能靠还原解决。如果问题是硬件损坏或硬盘故障，还原点帮不上忙

> 系统还原是系统级操作。如果步骤看不懂，先停手，搜索图文教程确认后再操作，或请懂的人帮忙。

## 四、你应该备份什么

不是所有文件都值得花时间备份。按优先级排列：

### 强烈建议备份（丢了无法替代）

- 正在写的论文、报告、作业
- 工作文档（简历、合同、项目文件）
- 个人照片和视频（尤其是家人合影、旅行回忆）
- 重要证书扫描件（身份证、学位证、护照）

### 建议备份（丢了很麻烦）

- 浏览器书签（你收藏的网页地址）
- 软件配置文件（如果你花了很多时间自定义设置）
- 聊天记录（如果对你有价值）

### 通常不需要专门备份（可以重新获取，但可能需要时间和网络）

- 操作系统本身（可以从微软官网重新下载安装包）
- 可以重新下载的软件安装包
- 音乐、电影（如果你是从正规平台购买的，通常可以重新下载）
- 浏览器缓存和临时文件

### 特殊情况：如果要重置或重装系统

如果你打算重置电脑或重装系统（参见[《常见故障排查思路》](./常见故障排查思路.md)的"什么情况该重启，什么情况该找人"），在动手前建议先确认以下内容是否已经备份：

- 桌面上的所有文件（桌面文件实际存在 C 盘，重装后会丢失）
- "文档""下载""图片"文件夹里的重要内容
- 浏览器书签（在浏览器设置中导出为文件）
- 软件的许可证密钥（也叫序列号、激活码，用于证明你购买了正版软件）——如果有购买的付费软件才需要
- Wi-Fi 密码（可以在"设置 → 网络和 Internet → WLAN → 管理已知网络"中点击网络名，选择"查看 Wi-Fi 安全密钥" `[查]`；或直接截图保存）
- 确认云盘同步已完成（等 OneDrive 或其他云盘的同步图标显示"已同步"）

## 五、云盘的使用注意事项

OneDrive、百度网盘等云盘服务可以作为一种备份手段，但要注意它们的局限。

### OneDrive（Windows 自带）

- 免费空间通常 5GB（微软账户赠送）
- 与 Windows 深度集成，设置简单
- 开启后会在文件资源管理器左侧显示
- **注意**：它是同步工具，不是纯备份工具（参见第一节的区别）
- 建议：用它来同步少量重要文件（如正在写的论文），但最好不要把它当作唯一备份

### 国内网盘（百度网盘等）

- 免费空间较大（通常数百 GB 到 TB 级）
- 适合存储大文件（照片、视频）
- 局限：上传下载速度可能受限（免费用户）；长期不登录可能清理文件；服务商本身的安全性和持续性无法保证

### 使用云盘的安全建议

1. **开启双重验证**（登录时除了密码还需要手机验证码）：就像邮箱和支付平台一样，给你的云盘账户也加上第二道防线（参见[《防患于未然：日常怎么保护电脑和账号》](../安全意识/日常安全习惯.md)的"密码基本原则"）
2. **敏感文件最好先加密再上传**：身份证照片、银行卡号、密码本等，直接放在云盘上有账号被盗的风险
3. **云盘不适合作为唯一备份**：云盘服务商也可能出问题（服务器故障、服务商倒闭）。按照 3-2-1 原则，最好还有离线备份

## 六、误删文件后第一时间该做什么

发现文件不见了，先别慌。按以下顺序来。

### 先检查最明显的地方

1. **回收站**：双击桌面"回收站"图标，看看文件在不在。如果在，右键→"还原"即可。回收站的基本操作详见[《日常使用》](../日常使用.md)的"删除文件"部分。
2. **搜索文件名**：有时候文件只是被误移动到了别的文件夹。按 `Win+S`（也可以点击任务栏上的搜索图标）输入文件名搜索一下。

### 如果回收站也没有

如果你之前开启了文件历史记录（见上文 3.1），或者系统还原点功能处于开启状态，可以试试：

- 右键点击文件原来所在的文件夹→"属性"→"以前的版本" `[查]`
- 如果列表中有历史版本，可以选择一个时间点的版本，点击"打开"查看内容，或点击"复制"把它保存到桌面等其他位置 `[查]`，确认是要找回的版本后再使用

如果"以前的版本"里为空，或者找不到需要的时间点——**这时候建议先停下来，深呼吸**。先确认一下：是不是记错了文件名或存放位置？这个文件对你重要吗？想清楚这些问题后，如果确实需要找回，再继续往下看。需要说明的是，如果没有提前备份，误删的文件通常很难完整恢复，继续操作可能让找回的机会更小。

### 先停下来：这几件事暂时别做

误删文件后，如果继续在电脑上存新文件、安装软件，可能会占用被删文件原本占用的位置，导致恢复更加困难。所以建议：

- 尽量少往误删文件所在的盘存新东西
- 避免在那个盘安装任何软件（包括数据恢复软件）
- 如果文件很重要，可以考虑尽快找人帮忙

### 什么时候该找专业帮助

- 文件确实重要且无法替代（论文、照片、工作文档）→ 找专业数据恢复服务
- 硬盘发出异响（咔咔声、摩擦声）→ 建议尽快关机，避免继续通电，找专业人员处理
- 自己检查了回收站和文件历史记录都找不到 → 停下来，找人帮忙

专业数据恢复服务通常价格不低 `[查]`，成功率也受多种因素影响。如果文件确实重要，可以咨询专业机构评估一下，做好心理准备，了解具体情况后再决定。

> **越早越好**：误删后尽快停止往那个盘存东西，然后找人帮忙。虽然现代固态硬盘（SSD）有自动清理机制，但及时停下来总比继续操作更稳妥。

## 七、一个好习惯：定期检查备份是否有效

备份不是一次性的工作。每隔几个月，花 10 分钟确认一下：

- 外接硬盘还能正常读取吗？（插上看看能不能打开文件）
- 云盘同步还在正常运行吗？（上传一个小测试文件看看能不能同步）
- 最近的重要文件有没有漏掉？（翻一下文档和图片文件夹）
- U 盘 / 硬盘有没有出现异常？（复制一个大文件试试速度正不正常）

如果发现某个备份出了问题，及时修复比事后处理更省心。

## 总结

1. **同步 ≠ 备份**：同步是镜子（实时反映变化），备份是快照（定格某一时刻）
2. **3-2-1 原则简化版**：重要文件至少存在两个不同的地方
3. **用好 Windows 自带工具**：文件历史记录（保存文件旧版本）、系统还原点（保存系统快照）
4. **误删后先止损**：检查回收站和文件历史记录，找不到就停下来——避免让情况更难处理，找人帮忙
5. **定期检查备份有效性**：备份不是设好就不用管了

备份不需要做到完美，只要养成习惯，关键时刻能帮到你。
