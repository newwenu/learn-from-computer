# Windows 11 设置导航：设置里有什么、怎么快速找到它

> 阅读前提：你已经掌握了基本操作（参见《基本操作》），知道任务栏、开始菜单和窗口的基本用法。

Windows 11 的"设置"是现代 Windows 的管理中心。从调节屏幕亮度到添加输入法，从管理开机启动项到检查系统更新，大部分日常配置都可以在这里完成。本文帮你建立对设置应用的整体认知，并补充正文各章节未涉及的分类。

---

## 一、怎么打开设置

打开设置的方式有好几种，不需要刻意去记，挑一个顺手的用就行：

**方式一（最快）：** 按键盘上的 `Win+I`。`Win` 键就是键盘左下角带有 Windows 图标（四个小方块）的按键。

**方式二：** 点击任务栏上的开始按钮（Windows 图标）打开开始菜单，直接输入"设置"两个字，搜索结果里会出现"设置"应用，点击它即可。不需要用鼠标去翻找图标。

**方式三：** 右键点击任务栏上的开始按钮（或者按 `Win+X`），在弹出的菜单中选择"设置"。这个菜单里还有其他常用工具，比如"文件资源管理器""任务管理器"等。

**方式四：** 点击任务栏右下角的网络、音量或电池图标（三个图标挨在一起），会弹出"快速设置"面板，面板右下角有一个齿轮形状的设置图标，点击它也能打开设置。

打开后，你会看到一个窗口，左侧是一列分类名称，右侧是当前选中分类的具体内容。点击左侧的任意分类，右侧就会切换显示对应的内容。

> Windows 10 的设置界面布局和 Windows 11 略有不同，左侧分类列表在 Win10 中可能显示为图标或折叠状态 `[查]`。

---

## 二、用搜索快速找到设置项

设置窗口顶部有一个搜索框，输入关键词就能快速找到对应的设置项，不需要逐个分类翻找。

**搜索技巧**：
- 输入功能名称（如"分辨率""夜间模式""启动"）
- 输入问题描述（如"屏幕太亮""连不上网"）
- 如果搜索结果为空，尝试换几个关键词再搜

搜索是找到设置项最高效的方式。当你不确定某个功能在哪个分类下时，优先使用搜索。

## 三、左侧导航栏一览

Windows 11 设置的左侧有十几个分类。下面按实际顺序逐个介绍，告诉你每个分类主要管什么。正文各章节已经详细讲过的，这里只列一句话并标注参见位置；正文没怎么讲的，会适当展开。

### 系统

管的是和电脑核心功能相关的一切：显示、声音、通知、电源、存储、多任务、投影、剪贴板、系统信息等。

- **显示**：分辨率、缩放比例、多显示器排列、夜间模式 → 详见《连接外设》"外接显示器与投影"一节
- **声音**：输出设备选择、音量调节 → 详见《连接外设》"有线音频设备"一节
- **存储**：存储感知、临时文件清理、查看各类型文件占用 → 详见《日常使用》"C 盘满了怎么办"和《Windows 存储进阶》
- **多任务**：分屏行为、Alt+Tab 切换范围 → 详见《虚拟桌面与多任务》
- **电源和电池**：电源模式、屏幕关闭时间、睡眠设置
- **剪贴板**：剪贴板历史记录（按 `Win+V` 可调出）
- **系统信息**：Windows 版本号、设备名称、处理器和内存信息

### 蓝牙和其他设备

管的是所有连到电脑上的外设：蓝牙设备、鼠标、键盘、触控板、笔、USB、自动播放等。

- **蓝牙开关和设备配对** → 详见《连接外设》"蓝牙设备"一节
- **鼠标和触控板**：指针速度、滚动方向、触摸板手势 `[查]`

### 网络和 Internet

管的是所有和网络相关的内容：Wi-Fi、以太网（有线网络）、VPN、代理、飞行模式、高级网络设置等。

- **Wi-Fi 连接和管理** → 详见《日常使用》"连接网络"一节
- **网络疑难解答**：网络连不上时，可以让 Windows 自动诊断修复

### 个性化

管的是电脑"长什么样"：桌面背景、颜色、主题、锁屏、开始菜单、任务栏、字体等。

- **背景**：更换桌面壁纸
- **主题**：切换系统配色方案，管理桌面图标（如显示"此电脑"）
- **任务栏**：任务栏对齐方式（居中或靠左）、任务栏图标显示/隐藏 → 详见《基本操作》"认识桌面"一节
- **开始菜单**：开始菜单上显示的文件夹、推荐内容开关

### 应用

管的是电脑上安装的软件：查看已安装应用、设置默认应用、管理可选功能、离线地图、启动项等。

- **已安装的应用**：卸载软件 → 详见《日常使用》"卸载软件"一节
- **默认应用**：设置用哪个程序打开哪种文件 → 详见《日常使用》"安装与卸载软件"一节
- **启动**：管理开机自动启动的程序 → 详见《理解你的操作系统》"任务管理器"一节

### 账户

管的是登录这台电脑的用户：账户信息、登录选项（密码/PIN/指纹/面部识别）、其他用户、访问工作单位或学校等。

- **登录选项**：设置 PIN、密码、指纹或面部识别 → 详见《日常使用》"账户与安全设置"一节
- **账户类型**：查看当前账户是管理员还是标准用户 → 详见《用户账户控制与权限》

### 时间和语言

管的是日期时间、语言和区域、输入法、语音等。

这个分类在正文中涉及不多，但日常使用频率不低：

**语言和区域**：添加或删除输入法、设置系统显示语言、调整日期和时间格式。如果你需要切换中英文输入法之外的第三种语言（如日语、韩语），或者想修改日期显示格式（如把"2026/7/1"改成"2026年7月1日"），都在这里设置。

**输入**：管理输入法的行为，比如是否显示候选框、是否开启自动纠错等。不同输入法的详细设置可能需要打开输入法自身的设置界面。

**语音**：语音 recognition（语音识别）和文字转语音的相关设置。

### 游戏

管的是和游戏相关的功能：Xbox Game Bar、游戏模式、捕获、控制器等。

- **Xbox Game Bar**：屏幕录制和截图 → 详见《屏幕截图与录屏》"Game Bar"一节
- **游戏模式**：开启后系统会优先分配资源给游戏，减少后台干扰

### 辅助功能

管的是帮助有不同需求的用户更好地使用电脑的功能：视觉、听力、交互三大类。

这个分类在正文中几乎没有涉及，但部分功能对普通用户也有用：

**视觉类**：
- **文本大小**：单独调大文字，不改变整体界面缩放
- **颜色滤镜**：把屏幕变成灰度、反色，或针对色盲用户优化颜色显示
- **高对比度**：用强烈的颜色对比替代默认配色，适合视力较弱或在强光环境下使用
- **放大镜**：放大屏幕局部区域，按 `Win+加号` 开启

**听力类**：
- **单声道音频**：把左右声道合并成一个，适合单耳听力受损的用户
- **音频通知的视觉提示**：收到通知时屏幕闪烁代替声音提示

**交互类**：
- **键盘**：粘滞键（按组合键不需要同时按）、筛选键（忽略重复按键）、切换键（按 Caps Lock 时发出提示音）
- **鼠标**：鼠标键（用数字小键盘控制鼠标指针）、指针大小和颜色

这些功能默认都是关闭的。如果你或身边的人有视力、听力或操作上的困难，可以在这里探索适合自己的选项。

### 隐私和安全性

管的是你的数据安全和隐私权限：Windows 安全中心、应用权限、搜索权限、诊断和反馈、活动历史记录等。

这个分类在正文中涉及不多，但值得了解：

**Windows 安全中心**：查看 Microsoft Defender 的状态、运行快速扫描、管理防火墙规则。日常保持开启即可，不需要额外操作。

**应用权限**：控制哪些应用可以访问你的摄像头、麦克风、位置、通讯录、照片等。如果你发现某个应用突然请求使用摄像头，或者想禁止某个应用访问你的位置信息，可以在这里查看和调整。

**搜索权限**：控制 Windows 搜索是否包含网页结果、是否记录搜索历史。如果你希望搜索只查找本地文件而不显示网页内容，可以在这里关闭"搜索 highlights"和"云内容搜索"。

**活动历史记录**：Windows 会记录你打开过的文件和访问过的网站，用于时间线功能和跨设备同步。如果你不希望被记录，可以在这里关闭。

**诊断和反馈**：控制 Windows 向微软发送哪些使用数据和错误报告。

> 隐私设置没有"绝对正确"的配置，取决于你对便利性和隐私的偏好。如果你不确定某个选项的作用，保持默认通常是安全的选择。

### Windows 更新

管的是系统更新：检查更新、查看更新历史记录、设置活跃时段、高级选项、传递优化、系统激活等。

- **检查更新和设置活跃时段** → 详见《理解你的操作系统》"为什么系统总提示更新"一节和《日常安全习惯》"保持系统更新"一节
- **系统激活**：查看 Windows 是否已激活（预装 Windows 的电脑通常出厂即激活）

## 四、设置里找不到的，去控制面板

Windows 11 把大部分日常设置都搬到了"设置"里，但仍有少数功能保留在旧版的"控制面板"中。如果你在设置里翻遍了也找不到某个选项，可以试试控制面板。

**打开控制面板的方法**：
- 按 `Win+R` 打开运行对话框，输入 `control` 回车
- 或在开始菜单搜索"控制面板"

正文中提到需要去控制面板操作的场景：
- **电源选项的合盖行为**：控制面板 → 硬件和声音 → 电源选项 → 选择关闭盖子的功能 → 详见《连接外设》"合盖使用外接显示器"一节
- **文件历史记录**：控制面板 → 系统和安全 → 文件历史记录 → 详见《数据备份与恢复》

微软正在逐步把控制面板的功能迁移到"设置"中，所以不同版本的 Windows 11，某些选项的位置可能略有差异 `[查]`。

## 五、快速打开某个设置页

如果你已经知道要去哪个分类，但想更快到达，可以用"运行"命令直接跳转。

**操作方法**：按 `Win+R` 打开运行对话框，输入对应的命令，回车。

以下是几个常用的命令：

| 命令 | 打开的页面 |
| ---- | ---------- |
| `ms-settings:display` | 显示设置 |
| `ms-settings:sound` | 声音设置 |
| `ms-settings:storage` | 存储设置 |
| `ms-settings:bluetooth` | 蓝牙和其他设备 |
| `ms-settings:network` | 网络和 Internet |
| `ms-settings:personalization` | 个性化 |
| `ms-settings:appsfeatures` | 已安装的应用 |
| `ms-settings:privacy` | 隐私和安全性 |
| `ms-settings:windowsupdate` | Windows 更新 |
| `ms-settings:system` | 系统信息 |

> 这些命令在《命令行入门》中提到的 `Win+R` 运行对话框里使用。如果你还不熟悉 `Win+R`，可以回顾《基本操作》中关于运行对话框的说明。

## 六、总结

Windows 11 的设置应用把系统配置集中到了一处。左侧十几个分类覆盖了从硬件连接到隐私安全的方方面面。日常使用时，记住这三件事：

- 打开设置最快的方式是按 `Win+I`
- 不确定某个功能在哪时，用顶部的搜索框
- 设置里找不到的，去控制面板看看

设置应用本身也在不断更新——不同版本的 Windows 11，某些选项的位置或名称可能略有变化 `[查]`。如果你发现某条路径和实际界面不符，用搜索功能定位通常是最快的解决办法。
