# 常用工具软件推荐

> 阅读前提：你已经掌握了软件安装的基本操作（参见《日常使用》）。本文解决"用什么软件好"的问题。所有推荐均为免费或开源软件，无广告、无捆绑。如需看图界面，可参考《你缺失的那门计算机课》的[工具类软件推荐](https://www.criwits.top/missing/tools-software.html)。

---

## 为什么需要这份推荐

Windows 自带了很多工具，但功能往往不够用：
- 记事本缺少语法高亮和行号
- 自带的视频播放器支持格式有限
- 搜索文件慢得像蜗牛
- 截图功能太基础

本文按**使用场景**分类，每个场景推荐**一个**靠谱的免费选择。不需要全部安装，遇到对应问题时再来查。

> **怎么访问官网**：每个软件下方都列出了官网地址。你可以直接点击地址访问，也可以手动输入到浏览器地址栏。点击前建议先看一下网址是不是软件的官方地址（比如 7-zip.org 是 7-Zip 的官网），而不是某个下载站的地址——这和《日常使用》里讲的"识别官网"是一个道理。

## 文本编辑：Notepad3

**解决什么问题**

Windows 自带的"记事本"在 Windows 11 中已经大幅改进，可以打开几百 MB 的文件，也支持基础的换行符识别。但它仍然缺少一些实用功能：不支持语法高亮（看代码时关键字不会变色）、不能显示行号、没有代码折叠。此外，部分用户反馈新版记事本偶尔会出现下划线显示异常的 bug。

**为什么推荐它**

- 免费、开源、无广告
- 启动速度极快（和记事本差不多）
- 自动检测文件编码，解决打开中文乱码问题
- 支持语法高亮（看代码时关键字会变色）
- 可以替换系统自带的记事本

**官网**

[rizonesoft.com/downloads/notepad3](https://www.rizonesoft.com/downloads/notepad3/)（页面为英文，找到 Download 按钮即可）

**不足之处**

- 界面比较朴素，没有现代软件的精美设计
- 插件生态不如 VS Code 丰富
- 主要面向 Windows，macOS/Linux 用户需要找其他替代品

> 如果你只是偶尔写点纯文字，系统记事本够用。但如果你想看代码、编辑配置文件、或者打开别人发来的各种文本文件，Notepad3 会省心很多。

## PDF 阅读：SumatraPDF

**解决什么问题**

Adobe Acrobat Reader 体积庞大、启动慢、经常弹更新；用 WPS 打开 PDF 有广告且功能冗余。

**为什么推荐它**

- 极速启动（秒开）
- 极简界面，只专注阅读
- 支持 PDF、ePub、MOBI、CBZ 等多种格式
- 单文件便携版，无需安装

**官网**

[sumatrapdfreader.org/download-free-pdf-viewer](https://www.sumatrapdfreader.org/download-free-pdf-viewer)（官网为英文，软件自带中文）

**不足之处**

- **不能编辑 PDF**（填表、签名、批注都不行）
- 界面过于简单，没有阅读进度同步、书签管理等功能
- 对复杂排版（如杂志、画册）的渲染偶尔有问题

> 如果你需要**编辑** PDF，SumatraPDF 做不到。可以试试 LibreOffice Draw（免费）或 PDF-XChange Editor（免费版功能有限）。

## 视频播放：VLC

**解决什么问题**

Windows 自带的"电影和电视"播放器支持格式有限，很多下载的视频打不开，或者只有画面没有声音。

**为什么推荐它**

- 支持几乎所有视频格式（MP4、MKV、AVI、FLV、WMV……）
- 支持几乎所有音频格式
- 无需额外安装解码器
- 无广告、无间谍软件
- 跨平台（Windows、macOS、Linux、手机都有）

**官网**

[videolan.org/vlc](https://www.videolan.org/vlc)（官网支持中文，软件自带中文）

**不足之处**

- 界面设计比较陈旧，不够现代美观
- 媒体库管理功能较弱（不如专业播放器）
- 默认设置下，某些高清视频可能出现画面撕裂，需要手动调整设置

> VLC 的界面不算好看，但"能播"是它的核心优势。如果你追求精美界面，可以考虑 PotPlayer（免费，但注意从官网下载避免捆绑）。

## 文件搜索：Everything

**解决什么问题**

Windows 自带的文件搜索很慢，尤其是在大硬盘上，搜一个文件要等几十秒甚至几分钟。

**为什么推荐它**

- **秒搜**：首次启动时建立索引，之后搜索几乎是瞬时的
- 极小的内存占用
- 支持通配符和正则表达式高级搜索
- 可以通过 HTTP 在局域网内分享搜索结果

**官网**

[voidtools.com/zh-cn](https://www.voidtools.com/zh-cn/)（中文官网，软件自带中文）

**不足之处**

- **只搜索文件名**，不搜索文件内容（搜不到"合同里写了多少钱"）
- 首次建立索引需要几分钟，期间硬盘占用较高
- 界面比较简陋，没有文件预览功能
- 不支持按文件内容类型筛选（比如"找所有包含表格的文件"）

**使用技巧**

- 第一次打开时，它会扫描全盘建立索引，可能需要几分钟
- 搜索时直接打字，结果实时出现
- 用 `*.jpg` 搜所有图片，`size:>100mb` 搜大文件

> 如果你需要全文搜索（比如找"合同里写了多少钱"），可以用 AnyTXT Searcher（免费）。

## 磁盘空间分析：WizTree

**解决什么问题**

C 盘莫名其妙满了，不知道什么占了空间；Windows 自带工具分析速度慢、不够直观。

**为什么推荐它**

- 比系统自带工具快 100 倍以上（直接读取 NTFS 主文件表）
- 可视化矩形树图，一眼看出哪个文件夹最占空间
- 可以按文件大小排序，快速定位"空间杀手"
- 免费版功能完整

**官网**

[diskanalyzer.com/download](https://www.diskanalyzer.com/download)（官网为英文，左侧 DOWNLOAD INSTALLER 是安装版，右侧 DOWNLOAD PORTABLE 是便携版）

**不足之处**

- 只能分析 NTFS 格式的分区（对 FAT32/exFAT 无效）
- 界面设计比较传统，不够美观
- 没有文件清理功能（只能看，不能一键删，删除需要手动操作）
- 免费版仅供个人使用，商用需要购买专业版

**和 SpaceSniffer 的区别**

两者功能类似，WizTree 速度更快，SpaceSniffer 的界面更酷（动态矩形图）。新手建议先用 WizTree。

> 关于 C 盘清理的详细方法，参见《Windows 存储进阶》。

## 下载工具：Motrix

**解决什么问题**

浏览器下载大文件容易失败、不支持断点续传（断网后从头下载）、不支持 BT/磁力链下载。

**为什么推荐它**

- 开源、免费、界面简洁美观
- 支持 HTTP/FTP/BT/磁力链/迅雷链接
- 支持断点续传
- 可以设置下载限速，避免拖慢网络

**官网**

[motrix.app/download](https://motrix.app/download)（页面为英文，选择一个即可）

**不足之处**

- 更新频率较低，部分功能（如浏览器扩展）不够完善
- BT 下载时，冷门资源的速度可能不如专业 BT 客户端（如 qBittorrent）
- 需要手动设置浏览器扩展才能捕获下载链接
- 项目维护不太活跃，未来更新不确定

**替代选择**

[Free Download Manager](https://www.freedownloadmanager.org/zh/download.htm)—— 功能类似，但界面偏传统，更新更频繁。两者选一个即可。

> 如果你只是偶尔下载小文件，浏览器自带下载够用。下载工具主要解决"大文件、容易断、资源冷门"的场景。

## 截图工具：PixPin

**解决什么问题**

微信/QQ 截图需要登录软件才能用；系统自带截图（Win+Shift+S）功能太基础，不能滚动截图、不能 OCR 文字识别。

**为什么推荐它**

- 免费、国产、中文界面友好
- 支持滚动截图（截长网页、长聊天记录）
- 支持 OCR 文字识别（截图里提取文字）
- 支持贴图（把截图钉在屏幕上，方便对照）
- 支持 GIF/视频录制

**官网**

[pixpin.cn](https://pixpin.cn)（页面为中文，直接下载即可）

**不足之处**

- 部分高级功能（如长截图的自动拼接）偶尔会出现识别错误
- OCR 对复杂排版或手写文字的识别率有限
- 目前只有 Windows 版，macOS/Linux 用户无法使用
- 云同步和跨设备功能不如 Snipaste 成熟

**替代选择**

[ShareX](https://getsharex.com)（官网为英文，软件自带中文；GitHub 备用：github.com/ShareX/ShareX）—— 功能更强大（自动上传图床、丰富的后处理），但界面复杂、学习成本高。新手建议先用 PixPin。

## 微软官方工具集：PowerToys

**解决什么问题**

Windows 缺少一些高级用户功能：窗口置顶、批量重命名图片、颜色拾取、快速预览文件内容……

**为什么推荐它**

- 微软官方出品，免费、安全、持续更新
- 模块化设计，需要什么功能开什么
- 常用功能包括：
  - **PowerRename**：批量重命名文件（比如把 100 张图片从 IMG_001.jpg 改成 北京旅游_001.jpg）
  - **Color Picker**：按快捷键取屏幕上任意位置的颜色值
  - **Always on Top**：让某个窗口始终置顶（看视频时记笔记很有用）
  - **File Locksmith**：查看哪个程序正在占用某个文件（删不掉文件时很有用）
  - **Peek**：按空格键快速预览文件内容（类似 macOS 的 Quick Look）

**获取方式**

1. Microsoft Store 搜索"Microsoft PowerToys"一键安装
2. 或 [GitHub 发布页](https://github.com/microsoft/PowerToys/releases/latest)下载

**不足之处**

- 功能繁多但学习成本较高，很多功能新手用不上
- 部分功能（如 FancyZones 窗口布局）和 Windows 自带功能有重叠
- 偶尔会和系统更新或其他软件产生冲突
- 内存占用比单独的小工具高（因为是一套集成工具）

> PowerToys 面向"想提高效率"的用户，不是必需品。可以先记住这个名字，遇到具体问题时再安装对应功能。

## 压缩软件：7-Zip

**解决什么问题**

Windows 自带压缩功能只支持 ZIP，打不开 RAR、7z 等格式；很多国产压缩软件有广告。

**为什么推荐它**

- 开源、免费、无广告
- 支持格式最全（ZIP、RAR、7z、TAR、GZ 等）
- 7z 格式的压缩率比 ZIP 高 30% 左右
- 可以集成到右键菜单，操作方便

**官网**

[7-zip.org](https://www.7-zip.org)（官网为英文，软件界面是中文）

**不足之处**

- 界面设计非常朴素，甚至可以说"简陋"
- 没有压缩包修复功能（损坏的压缩包无法恢复）
- 不支持压缩包注释和加密文件名（Bandizip 支持）
- 更新频率较低，新功能添加慢

> 关于压缩格式的详细说明，参见《压缩与解压缩》。

## 浏览器：Microsoft Edge

**解决什么问题**

IE 已淘汰，旧版 Edge 不好用，需要一款现代浏览器。

**为什么推荐它（作为默认选择）**

- Windows 自带，无需安装
- 基于 Chromium 内核（和 Chrome 相同），兼容性好
- 内存占用比 Chrome 略低
- 内置垂直标签页、网页截图、数学求解器等实用功能
- 国内同步服务可用（Chrome 的同步在国内不稳定）

**不足之处**

- 和 Chrome 一样存在"Chromium 内核垄断"的争议（网站只适配 Chromium，其他内核浏览器体验差）
- 部分用户反馈偶尔会出现兼容性问题（如某些网银只认 IE）
- 微软会推送一些功能（如 Copilot、购物比价）并非所有用户都需要
- 扩展商店不如 Chrome 丰富，部分扩展需要手动安装

**什么时候考虑 Chrome/Firefox**

- Chrome：如果你已经在 Google 生态里（Gmail、Google 文档），或者需要丰富的扩展
- Firefox：如果你注重隐私保护，不想用 Chromium 内核的浏览器

> 关于浏览器的详细选择建议，参见《浏览器入门》和《你缺失的那门计算机课》的[浏览器章节](https://www.criwits.top/missing/browsers-and-how-to-choose.html)。
