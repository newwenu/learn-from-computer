## 一、PC 是什么

PC 是 Personal Computer（个人计算机）的缩写。它能够接收你的指令（通过键盘、鼠标、触摸板），进行处理，然后把结果呈现给你（通过屏幕、扬声器）。

从本质上说，PC 是一个**信息处理工具**。你给它输入信息（打字、点击、语音），它按照预先写好的程序去处理这些信息，然后输出结果（显示文字、播放声音、保存文件）。就这么简单。

计算机并不神秘。它不具备人类的"思考"能力，它只是在极快地执行极其简单的指令——快到你感觉它是在"思考"。

## 二、PC 的常见形态

你见到的 PC 主要有三种形态，它们的核心部件是一样的，只是布局和便携性不同。

### 台式机（Desktop）

主机和显示器是分开的。主机是一个独立的箱子（机箱），里面装着所有核心部件。优点是性能强、散热好、方便升级和维修，缺点是占地方、不能搬着走。

台式机适合固定场所使用——家里、办公室、实验室。

### 笔记本电脑（Laptop/Notebook）

把主机、显示器、键盘、触摸板、电池、扬声器全部整合在一个折叠机身里。优点是便携、自带电池（不插电也能用几小时），缺点是不太容易升级硬件（多数部件焊死在主板上），同价位性能通常不如台式机。

笔记本适合需要移动使用的场景——教室、图书馆、会议室、咖啡厅。

> **电池保养**：笔记本插电使用是常态，不必"充满就拔"。建议开启品牌自带的"养护模式"（联想叫"养护模式"、华硕叫"电池健康充电"、华为叫"智能充电"），让电量保持在 80% 左右，延长电池寿命。如果找不到设置，长期插电也没关系，现代电池都有过充保护，但满电状态会略微加速电池老化，这是正常损耗。

### 二合一平板（2-in-1 / Tablet PC）

屏幕可以拆下来或翻转过去当平板用，比如 Microsoft Surface Pro 系列。它本质上是一台笔记本，但多了触控和手写笔功能。优点是形态灵活，缺点是同价位性能弱于传统笔记本。

## 三、PC 的核心硬件

无论什么形态，一台 PC 里面都有下面这几样核心部件。理解它们的作用，是理解计算机的基础。

### 1. CPU（中央处理器）— 执行指令的核心

CPU 是计算机里最重要的芯片，负责执行程序中的指令。它一秒钟能执行几十亿条简单指令，靠这种极快的速度完成复杂任务——比如显示网页、播放视频。

CPU 安装在主板的中央位置，上面覆盖着一个金属散热片和风扇（合称散热器），用来给它降温。如果你打开台式机的机箱，看到主板中央那个被风扇压着的方形芯片就是 CPU。笔记本的 CPU 也藏在机身内部，同样连着散热铜管和风扇。

CPU 的性能由频率、核心数、架构等多个因素决定。目前主流品牌是 Intel（酷睿 i3/i5/i7/i9，新机型已改为 Core 3/5/7/9 和 Core Ultra 5/7/9 系列）和 AMD（锐龙 R3/R5/R7/R9）。对大多数人来说，知道 i5/R5 是常见选择就够了。

CPU 可以理解为"计算机的大脑"，但更准确的比喻是"一个极其勤奋、执行力极强、但完全不懂思考的工人"。它严格按指令做事，不会自己决定做什么。

### 2. 内存（RAM）— 临时存放正在处理的数据

内存是一块临时存储区域，CPU 直接从这里读写数据。

- 内存的特点是**速度快**（比硬盘快得多），但**断电后数据全部丢失**。
- 内存只存放"当前正在用"的数据。比如你打开一个 Word 文档，这个文档的内容就从硬盘加载到内存里了。你编辑的内容都临时存在内存里，等你点了"保存"，才写回硬盘。
- 内存的容量单位是 GB。16GB 是现在的基础配置，32GB 适合同时开很多大型软件。

一个常用的比喻：内存是你的"工作桌"，硬盘是"文件柜"。你从文件柜（硬盘）里拿出文件放在桌上（内存）来处理。桌子越大（内存越大），你能同时摊开处理的东西就越多。

### 3. 硬盘（存储设备）— 长期保存数据的地方

硬盘负责存东西——操作系统、软件、文档、照片、电影，关机后数据不会丢。

目前有两种硬盘：

**机械硬盘（HDD）**：里面有旋转盘片和磁头，容量大、便宜，但速度慢、怕震动。适合存不常用的大文件。

**固态硬盘（SSD）**：里面是闪存芯片，没有机械部件，速度快、安静、抗震。现在的新电脑几乎都标配 SSD。

SSD 比 HDD 快很多：开机通常十几秒 vs 几十秒到一分钟，打开软件几乎是秒开 vs 等几秒。如果你的电脑还是机械硬盘，换成 SSD 是提升体验最明显的升级。

> **存储单位**：电脑数据的最小单位是**比特（bit）**，8 个比特 = 1 个**字节（Byte，简写 B）**。日常说的文件大小用字节计算：1024 B = 1 KB，1024 KB = 1 MB，1024 MB = 1 GB。网速的"Mbps"是比特，下载速度的"MB/s"是字节，差 8 倍。
>
> 另外，Windows 按 1024 进制算容量，但硬盘厂商按 1000 进制标称。所以你买的"1 TB"硬盘在电脑上显示约 931 GB，这是正常的，不是质量问题。

### 4. 主板（Motherboard）— 连接所有部件的骨架

主板是一块大电路板，上面有各种插槽和接口，CPU、内存、硬盘、显卡等都插在上面，通过主板上的线路互相通信。

不同 CPU 需要不同的主板（插槽不通用，Intel 和 AMD 的主板不能互换）。笔记本的主板按机型定制，无法像台式机一样自由更换。

### 5. 显卡（GPU）— 负责把画面送到显示器

显卡的核心是 GPU（图形处理器），它专门负责计算图像数据——把操作系统界面、游戏画面、视频内容转换成显示器能显示的信号。

- **集成显卡**：集成在 CPU 内部，和 CPU 共享内存。性能够用，适合办公、看视频、轻度游戏。
- **独立显卡**：有自己的 GPU 芯片和显存，性能更强，适合玩游戏、3D 设计、视频剪辑。不玩游戏不需要额外购买。

（显卡型号含义和选购建议，详见后文《电脑配置详解》。）

### 6. 电源（Power Supply）— 提供能源

电源把墙壁插座里的交流电转换成电脑各部件需要的直流电。

台式机的电源功率用"瓦特（W）"衡量，选购时要留一定余量。笔记本通过外接的"电源适配器"供电，随机器附带，不需要自己选配。

### 7. 散热器 — 把热量带走

CPU 和显卡工作时会产生大量热量，如果不及时散热，温度过高会自动变慢来保护硬件，严重时可能损坏。散热器就是负责把这些热量带走。

- **风冷**：一块金属散热片贴在 CPU 上，风扇对着散热片吹风把热量带走。简单可靠便宜。
- 绝大多数日常使用场景下，风冷完全够用。

## 四、外设：人与电脑交互的工具

**外设**（外部设备）是指连接在电脑主机外面的设备，用来和电脑交互。台式机需要单独连接显示器、键盘、鼠标等外设才能使用。而笔记本把这些都集成在了机身里——自带了屏幕、键盘、触控板、扬声器、摄像头和麦克风，开盖即用，不需要额外连接。

### 显示器

显示器的核心参数是**分辨率**（屏幕上像素点的数量，1920×1080 是目前主流）和**刷新率**（每秒刷新次数，60Hz 日常够用）。分辨率越高画面越细腻，但对显卡要求也越高。

### 键盘与鼠标

键盘和鼠标是主要的输入设备。笔记本自带键盘和触摸板，台式机需要单独购买。日常办公用普通键盘鼠标就够了，有特殊需求（如游戏、长时间打字）时再考虑机械键盘或人体工学鼠标。

## 五、软件：让硬件有用起来的东西

硬件本身不会思考，需要软件告诉它该做什么。软件分为两大类。

### 操作系统（OS）

操作系统是电脑最核心的软件之一。它负责管理所有硬件资源、提供操作界面、让其他软件能正常运转。

常见的操作系统：

- **Windows**：微软出品，是个人电脑上最普及的系统（桌面端市场占有率最高）。兼容性最好，绝大多数软件和游戏都支持 Windows。
- **macOS**：苹果公司出品，只用在苹果电脑（MacBook、iMac）上。界面简洁，适合开发者和创意工作者。
- **Linux**：开源免费的系统，有很多不同的版本（Ubuntu、Fedora、Debian 等）。服务器领域占主导地位，桌面端使用人群较少，主要是程序员和爱好者。

对于第一次接触电脑的人来说，Windows 是最容易上手的选择。

### 应用软件

安装在操作系统之上的"工具"，用来完成具体的任务：

- **浏览器**（Chrome、Edge、Firefox）— 上网浏览
- **办公套件**（Microsoft Office、WPS Office）— 写文档、做表格、做演示文稿
- **媒体播放器** — 播放音乐和视频
- **图像编辑**（Photoshop、GIMP）— 处理照片和图片
- **通讯软件**（微信、QQ、钉钉）— 聊天和协作

## 六、它们是怎么协同工作的

以"你在 Word 里打字，然后保存文档"这个最简单的操作为例，看看整个系统是怎么配合的：

1. 你按下键盘上的字母 A → 键盘把这个按键信号发送给主板（笔记本通过内部排线，台式机通过 USB 接口）
2. 主板把信号传给 CPU → CPU 读出操作系统里"处理键盘输入"的那段代码
3. CPU 执行这段代码 → 把字母 A 对应的数据写入内存（Word 软件的编辑区域中）
4. CPU 告诉显卡"屏幕上这个位置需要显示一个字母 A" → 显卡把画面输出到显示器
5. 你看屏幕上出现了一个 A ← 这是你用眼睛感受到的结果
6. 你点了"保存"按钮 → CPU 把内存中 Word 软件的全部内容读取出来，写入 SSD/硬盘

整个过程在毫秒到微秒级别完成，所以你感觉是即时发生的。

## 七、总结：对 PC 的正确认识

| 部件   | 作用                         | 一句话理解   |
| ------ | ---------------------------- | ------------ |
| CPU    | 执行指令，进行计算和逻辑判断 | 干活的人     |
| 内存   | 临时存放当前正在处理的数据   | 工作桌       |
| 硬盘   | 永久保存数据和软件           | 文件柜       |
| 主板   | 连接所有部件，传递数据       | 骨架和公路网 |
| 显卡   | 计算图像，输出到显示器       | 画画的人     |
| 电源   | 提供稳定的电能               | 食物来源     |
| 散热器 | 带走废热                     | 风扇+散热片  |

计算机的本质：**输入 → 处理 → 输出**。

你通过键盘鼠标（输入设备）给计算机输入数据，CPU 按照程序（软件）的指令处理这些数据，然后把结果通过显示器/扬声器（输出设备）呈现给你。硬件提供了物理基础，软件决定了硬件做什么以及怎么做。

## 八、怎么查看自己电脑的配置

知道硬件是什么之后，你可能想知道自己电脑里具体装的是什么。Windows 自带了查看工具，不需要安装任何软件。

### 最快的方式——任务管理器

按 **Ctrl+Shift+Esc** 打开任务管理器，点击顶部的"性能"选项卡：

| 项目 | 你能看到的信息                                |
| ---- | --------------------------------------------- |
| CPU  | 处理器型号（如"Intel Core i5-13400"）、核心数 |
| 内存 | 总容量（如 16.0 GB）                          |
| 磁盘 | 硬盘型号、总容量、类型（SSD 或 HDD）          |
| GPU  | 显卡型号、专用显存                            |

### 看 Windows 版本和系统类型

右键点击"开始"按钮（屏幕左下角或任务栏中央的 Windows 图标，四个小方块组成的窗口图案）→ 选择"系统" → 找到"关于"：

| 信息项          | 说明               |
| --------------- | ------------------ |
| 处理器          | CPU 型号和基础频率 |
| 安装内存（RAM） | 内存总容量         |
| 系统类型        | 64 位还是 32 位    |

### 什么时候需要更详细的配置信息

上面的方法足够回答"你这电脑什么配置"这个问题。当你需要更详细的信息时（如买游戏前对照最低配置、想升级内存或硬盘），可以阅读[《电脑配置详解》](../电脑硬件/电脑配置详解.md)。
