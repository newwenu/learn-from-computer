# 查看电脑配置：了解你电脑里装了什么

> 阅读前提：你已经了解了 PC 有哪些硬件部件（参见《认识PC》）。本文教你用 Windows 自带工具查看你电脑的 CPU 型号、内存大小、硬盘容量、显卡型号等信息。

---

## 一、为什么要学会看配置

学会查看电脑配置，很多场景用得上：

- 别人问"你这电脑什么配置"，你能说出 CPU 和内存
- 想给电脑加内存条或换硬盘，需要先确认当前型号和规格
- 买个新游戏或软件，需要对照它的"最低要求"看自己电脑带不带得动
- 电脑变慢了，你能先看看是 CPU 满载还是内存不够，而不是直接找人修

Windows 自带了好几种查看配置的工具，不需要装任何第三方软件。它们分散在不同地方，展示的信息各有侧重。本文按使用频率排序，从最常用的讲起。

---

## 二、最快的查看方式——任务管理器

**打开方法：** 按 `Ctrl+Shift+Esc`，或按 `Ctrl+Alt+Delete` 然后选"任务管理器"。

如果看到的是简略列表，点击底部的"更多详细信息"展开完整视图。

### 切换到"性能"选项卡

在左侧列表中点击 **性能**（或第二栏的"性能"标签），会看到几个分组，从左到右依次列出各硬件：

| 项目 | 你能看到的信息 |
| ------ | ----------- |
| CPU | 处理器型号（如"Intel Core i5-13400"）、**基础频率**（单位 GHz）、**核心数**与**逻辑处理器数** |
| 内存 | **总容量**（如 16.0 GB）、**速度**（如 3200 MHz）、已用和可用 |
| 磁盘 0 / 磁盘 1 | 硬盘型号、**总容量**（如 512GB）、**类型**（SSD 或 HDD）、读写速度 |
| GPU 0 / GPU 1 | 显卡型号（如"NVIDIA GeForce RTX 4060"）、**专用显存**、使用率 |

每个分组的右下角有小图表显示实时占用率。点击某个分组，右侧会展示更详细的信息窗口。

**这种方式最适合快速查看**（几秒就能看到），但它不展示 Windows 版本和系统类型（32/64 位）等信息。那些信息在下面会讲到。

> 提示：任务管理器"性能"选项卡的展示形式在 Windows 10 和 11 上略有不同，但可以看到的信息是一样的。把鼠标停在某个参数上，通常会弹出提示说明；也可以把参数名称复制到搜索引擎中查找含义。

---

## 三、系统信息全览——"关于"页面

**打开方法：** 右键点击"开始"按钮 → 选择"系统" → 在窗口左侧或底部找到"关于"，或直接在设置搜索框里输入"关于"。路径是：**设置 → 系统 → 关于**。

这里把设备和系统信息汇总在一个页面，分成两大块：

### 设备规格

| 信息项 | 示例 | 说明 |
| ------ | ------ | ------ |
| 处理器 | Intel Core i5-13400 2.50 GHz | CPU 型号和基础频率 |
| 安装内存（RAM） | 16.0 GB | 内存总容量 |
| 系统类型 | 64 位操作系统，基于 x64 的处理器 | 你的系统是 64 位还是 32 位（参考《理解你的操作系统》中"32 位和 64 位"一节） |
| 笔和触控 | 不支持触控输入 | 屏幕是否支持触控 |

### Windows 规格

| 信息项 | 说明 |
| ------ | ------ |
| 版本（如 Windows 11 专业版） | 你用的哪个 Windows 版本 |
| 版本号 | 如 23H2，表示功能更新的版本批次 |
| 操作系统版本 | 如 22631.xxxx，更精确的构建编号 |
| 体验 | Windows 体验包版本 |

如果只看 CPU 型号和内存大小，"关于"页面是最直观的：打开就能看到，不需要找图标。另外，如果你想知道自己的 Windows 大版本号是多少（比如是不是最新的），也是在这里看。

> 自行探索：按 `Win+R` → 输入 `winver` → 回车。会弹出一个专门的窗口显示 Windows 版本和操作系统版本号，比"关于"页面更简洁。

## 四、硬盘容量和使用情况——"此电脑"

**打开方法：** 按 `Win+E` 打开文件资源管理器，左侧点击"此电脑"，或在桌面双击"此电脑"图标。

你会看到各硬盘分区（C 盘、D 盘等）和对应的容量条。每个盘符下面显示"xxx GB 可用，共 xxx GB"。

这里也能看到插上的 U 盘和移动硬盘。

如果想看更详细的硬盘信息（具体型号、是 SSD 还是 HDD），返回任务管理器的"性能"选项卡，点击"磁盘 0"查看。

---

## 五、显卡和显示器详细信息——DirectX 诊断工具

当你需要确认显卡的**具体型号、显存大小、驱动版本**时，上面的工具给的信息可能不够详细。这时可以用 DirectX 诊断工具：

**打开方法：** 按 `Win+R` → 输入 `dxdiag` → 回车 → 在弹出的窗口中点击"是"（检查驱动程序签名）。

### 主要看这两个选项卡

**系统选项卡**（打开时默认显示）：

- 操作系统名称和版本
- 处理器（CPU 型号和频率）
- 内存（RAM 总容量）
- DirectX 版本（如 DirectX 12）

**显示选项卡**（如果有独立显卡，会有多个显示选项卡）：

- 显卡名称和制造商
- 显存（显示内存）
- 当前显示模式（分辨率、刷新率）
- 驱动程序版本和日期

这个工具的信息最全，适合装机后确认所有硬件是否正常识别到，或排查游戏、图形相关问题时查看。

> 自行探索：dxdiag 的"输入"选项卡可以看到你连接的键盘、鼠标、手柄等信息。"声音"选项卡可以看到声卡和音频输出设备。这些在日常使用中较少用到，感兴趣的话可以自己点开看看。

## 六、几个自行探索的方向

Windows 自带的查看工具已经足够应对大多数场景。如果还想了解更详细的信息，这里给出几个可以自行探索的方向：

1. **第三方硬件信息工具** — 如 CPU-Z、GPU-Z、Speccy。它们能展示更详细的参数（比如内存的通道数、显卡的核心频率、主板型号）。搜索"CPU-Z 怎么看配置"可以找到教程。**建议先用熟本文的自带工具，再尝试第三方工具。**

2. **命令行的 systeminfo** — 按 `Win+R` → 输入 `cmd` 回车 → 在黑色窗口中输入 `systeminfo` 回车。会列出操作系统的详细安装信息、网络配置、已安装的修补程序列表等。在找不到图形界面入口时，这是一个补充手段。

3. **PowerShell 的 Get-ComputerInfo** — 在 PowerShell 中输入 `Get-ComputerInfo` 会输出极其详尽的系统信息（比 systeminfo 更全）。还可以只查某项，比如 `Get-ComputerInfo -Property "Os*"` 只查操作系统相关信息。有兴趣的话可以自行搜索了解，日常不必掌握。

4. **BIOS/UEFI 中查看硬件信息** — 开机时按特定键（Del、F2、F10 等，不同主板不同）可以进入系统固件设置界面，里面也能看到 CPU、内存、硬盘信息。

**不管用哪种方法，如果遇到看不懂的参数项，最简单的做法就是：把那个词复制到搜索引擎里搜索。** 比如看到"逻辑处理器"不明白，搜索"逻辑处理器是什么意思"，比死记硬背强得多。

---

## 七、总结：什么时候用哪个

| 你想看什么 | 用哪个工具 | 怎么打开 |
| ------ | ------ | ------ |
| CPU 型号、内存大小、显卡型号（快速看） | 任务管理器 → 性能 | Ctrl+Shift+Esc |
| Windows 版本、系统类型（32/64 位） | 设置 → 关于 | 右键开始 → 系统 → 关于 |
| 硬盘还剩多少空间 | 此电脑 | Win+E |
| 显卡驱动版本、显存大小、DX 版本 | DirectX 诊断工具 | Win+R → dxdiag |
| 只看 Windows 版本号 | winver | Win+R → winver |

你看得越多，就越能把自己在《认识PC》里学到的硬件知识和实物对应起来——"哦，我电脑里装的就是这个型号的 CPU，这个大小的内存"，这种对应本身就是最好的学习。
