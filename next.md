# 选题

## 建议1

| 选题 | 说明 | 与现有内容的关系 |
| ------ | ------ | ----------- |
| 1 | 查看电脑配置 | 教读者如何查看自己电脑的 CPU、内存、硬盘、显卡、系统类型等信息（任务管理器、「关于」页面、DXDiag 等） 配合《认识PC》，学了硬件后知道怎么查自己的配置 |
| 2 | 屏幕截图与录屏 | Win+Shift+S（截图工具）、Win+G（录屏/游戏栏）、Print Screen、Win+Print Screen 保存截图 全新的实用技能，每天都可能用到 |
| 3 | 系统备份与还原 | 文件历史记录、系统还原点、重置此电脑、「设置 → 恢复」中的选项 补充《日常使用》的「好习惯」，讲出事后的恢复手段 |
| 4 | 虚拟桌面与多任务 | Win+Tab、Win+Ctrl+D 创建虚拟桌面、多桌面切换、分屏布局进阶 补充《基本操作》的窗口操作，进阶多任务 |
| 5 | 常见故障排查思路 | 按「软件→驱动→硬件→网络」分层排查，蓝屏代码怎么看、日志查看器（事件查看器）入门 与《理解你的操作系统》的任务管理器互补，从诊断到解决 |
| 6 | 蓝牙与外设连接 | 蓝牙配对、打印机/耳机/鼠标连接、安全弹出硬件 补充《日常使用》的网络和设备部分 |
| 7 | 多显示器与投影 | Win+P 投影模式、扩展/复制屏幕、分辨率与排列设置 笔记本外接显示器的常见场景 |
| 8 | 系统清理 | 磁盘清理（cleanmgr）、存储感知、清理临时文件 区别于「别用第三方优化软件」，教用系统自带的 |
