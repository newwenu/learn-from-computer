# 用户操作时间线

模拟一个普通用户从拿到电脑开始，在不同时间尺度下可能执行的操作，按频率由高到低排列。

---

## 每次使用（每会话）

开机 → 解锁（PIN/密码/指纹）→ 打开浏览器/聊天软件/文档 → 多窗口切换 → 分屏 → 地址栏搜索 → 新开/关标签 → 保存文件 → 关机/合盖

## 每天

- **浏览器**：标签页管理、地址栏搜索、下载文件、存书签、装扩展
- **文件管理**：下载文件夹整理、U盘插拔传文件
- 截图
- 微信/QQ/邮件：收发文件、打开收到的压缩包
- 遇到小问题：程序卡了、连不上网
- 充电/电源管理

## 每周

- 清理 C 盘空间
- 安装软件/系统更新
- 压缩文件发给别人
- 传输文件（U盘/网盘/微信文件）

## 每月

- Windows 系统更新
- 查电脑配置（加内存前确认规格、买游戏前看最低要求）
- 外接显示器/投影仪
- 整理桌面和下载文件夹
- 清理浏览器缓存和历史
- 改系统设置（默认程序、音量、壁纸、屏幕亮度）

## 每半年到一年

- 磁盘分区调整（C 盘扩容、新硬盘初始化）
- 重装系统 / 重置此电脑
- 数据备份
- 更换硬件（加内存、换硬盘、换电脑）
- 误删文件想找回
- 多用户管理（给别人建账号）

## 偶尔（特定场景）

- 打印文档（添加打印机、设置默认打印机）
- 参加视频会议（摄像头/麦克风设置）
- 用蓝牙连耳机/音箱
- 设闹钟/计时器（自带闹钟应用）
- 连接游戏手柄等外设
- 打开扫描仪扫描文件
- 接入扩展坞
- 用隐私模式浏览
