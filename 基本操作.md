# 基本操作：拿到一台新电脑后怎么上手

> 阅读前提：你已经了解了 PC 有哪些部件（参见《认识PC》）。本文假设你面前有一台装好 Windows 的电脑。

## 一、开机与关机

### 开机

找到电源键。台式机通常在机箱前面板（正面），笔记本通常在键盘上方或机身侧面 C 面边缘。按一下，听到风扇转动、屏幕亮起，说明开机了。

第一次开机，系统会引导你完成初始设置（不同品牌、不同 Windows 版本略有差异，实际以屏幕提示为准）：

- 选择语言和区域（选"中文（简体，中国）"）
- 连接 Wi-Fi（如果有有线网，插上网线会自动连接）
- 登录微软账户。如果你没有微软账户，可以用手机或另一台电脑提前到 account.microsoft.com 注册，也可以在这步选"创建账户"现场注册。部分 Windows 版本会强烈推荐登录微软账户，本地账户入口被隐藏得较深（可能需要在未联网时才会出现），如果找不到可以搜索当前版本的解决方案。登录微软账户的好处是设置可以同步到其他电脑，以及方便使用 OneDrive 等微软服务。
- 设置 PIN 码（一组纯数字密码，用于快速登录，比输入完整密码方便）。部分版本或特定系统配置可能不要求设置 PIN，按屏幕提示操作即可
- 隐私设置：可以全部选"是"，也可以根据需要关闭部分选项（如定位、广告 ID）

以上每一步的具体选项可能因 Windows 版本更新而变化，遇到不清楚的选项可以在搜索引擎或 AI 工具中搜一下具体含义，但要注意甄别——AI 的回答有时会出现错误，尽量交叉验证。设置完成后稍等几分钟，就会进入桌面。

### 关机

**正确关机方式**：点击屏幕左下角的"开始"按钮（Windows 图标）→ 点击电源图标 → 选择"关机"。电脑会自动关闭所有程序、保存系统状态、切断电源。

**为什么不能直接拔电源或长按电源键强行关机**：因为操作系统正在读写数据，强行断电可能导致文件损坏或系统异常。只有在电脑完全死机（鼠标键盘都没反应，持续几分钟以上）时，才长按电源键 5-10 秒强制关机。

### 睡眠与重启

- **睡眠**：点击电源菜单里的"睡眠"。电脑进入低功耗状态，但内存继续供电以保持当前工作状态。按任意键或开盖会快速恢复到之前的状态。睡眠适合短时间离开（比如去吃饭、课间休息）。
- **重启**：点击"重启"。电脑会关闭所有程序、重新启动系统。当你安装了新软件或更新了系统，或者电脑出现卡顿异常时，重启往往能解决问题。重启是解决各种小问题的首选方法。

### 合盖（笔记本专用）

合上盖子默认会让笔记本进入睡眠状态。可以在"设置→系统→电源→合上盖子后的操作"里修改为"不采取任何操作"（适合外接显示器使用）或"关机"。

## 二、认识桌面

开机进入桌面后，你会看到以下部分：

### 桌面背景（壁纸）

占据整个屏幕的背景图片。右键点击桌面空白处 → "个性化" → "背景"可以更换。

### 桌面图标

桌面上的小图片，每个图标对应一个程序或文件。双击可打开：

- **此电脑**（或"计算机"）— 用来查看和管理硬盘、U盘、外接设备
- **回收站** — 删除的文件临时存放在这里，还没真正消失，可以恢复或彻底清空
- **控制面板**（旧版系统设置，部分设置项需要在这里找）

如果你桌面上没有"此电脑"，可以右键桌面空白处 → "个性化" → "主题" → "桌面图标设置" → 勾选"计算机" → 确定。

### 任务栏

屏幕底部那条长条，是 Windows 最重要的导航工具：

- **开始按钮**（Windows 图标）— 点击打开开始菜单。Windows 10 默认在左下角，Windows 11 默认在任务栏居中
- **搜索框/搜索图标** — 可以搜索电脑里的程序、文件、设置项。很多人不知道，按键盘上的 Win 键然后直接打字，效果一样
- **任务栏图标区** — 固定在这里的程序图标，以及正在运行的程序图标。点击图标可以打开或切换到对应程序。正在运行的程序图标下有一条小横线
- **托盘区**（右下角）— 显示时间、音量、网络状态、电池（笔记本）等。部分后台运行的程序也会在这里显示图标
- **显示桌面**（任务栏最右侧一条细长的透明区域）— 点击一下可以让所有窗口最小化，直接看到桌面。再次点击恢复窗口

### 开始菜单

点击开始按钮后弹出。左侧是常用程序列表，右侧是功能磁贴区域（Windows 10）或固定区域（Windows 11）。你可以在这里：

- 滚动或搜索找到任何已安装的程序
- 右键点击程序 →"固定到任务栏"或"固定到开始屏幕"
- 直接打字搜索（如上所述，按 Win 键后直接输入程序名）

## 三、窗口操作

在 Windows 中，每个打开的程序都显示在一个"窗口"里。学会窗口操作是使用电脑的基本功。

### 打开和关闭

- **打开**：双击桌面图标，或在开始菜单中点击程序
- **关闭**：点击窗口右上角的 ×，或者按 Alt+F4

### 移动窗口

把鼠标指针放在窗口最上面的标题栏（通常是一条横条，颜色因系统和程序而异），按住左键拖动，移到你想要的位置再松开。

### 调整窗口大小

- 把鼠标指针移到窗口的边缘或角落，指针会变成双向箭头，按住左键拖动即可。部分窗口（如某些设置弹窗）有固定大小，不支持手动拖拽调整
- 最大化（让窗口占满整个屏幕）：点击右上角的 □ 图标，或双击标题栏
- 还原（回到之前的大小）：再次点击 □ 图标
- 最小化（让窗口缩小到任务栏，不关闭程序）：点击右上角的 — 图标。任务栏上仍然有它的图标，点击可以恢复显示

### 快速分屏（有用的小技巧）

把窗口拖到屏幕的最左侧或最右侧边缘，直到鼠标碰到边缘，窗口会自动占满左半或右半屏幕。另一边可以放另一个窗口。这样你就能同时看到两个窗口的内容（比如一边看资料一边写文档），不需要来回切换。

Windows 11 还有更直观的分屏布局：把鼠标悬停在窗口右上角的 □ 图标上，会弹出几种布局方案。

### 多窗口切换

- Alt+Tab：按住 Alt 键不放，按一下 Tab 键会弹出一个缩略图列表，继续按 Tab 在程序间切换，松开 Alt 就切换到选中的程序
- Win+Tab：类似 Alt+Tab，但不会松开按键后立即切换，适合慢慢找
- 直接点击任务栏上的程序图标

### 注意：关闭窗口 ≠ 退出程序

点击 × 关闭 Word 的一个文档窗口，如果还有其他文档打开，程序仍在运行（任务栏上仍有图标）。只有关闭所有文档（或点击"文件→退出"）程序才会完全退出。这个区别对初学者不太重要，但你可能会疑惑为什么点了 × 程序还在任务栏上。

## 四、鼠标与键盘基础操作

### 鼠标

| 操作 | 作用 |
| ------ | ------ |
| 左键单击 | 选中某个对象（文件、按钮、链接） |
| 左键双击 | 打开文件或启动程序 |
| 右键单击 | 弹出上下文菜单（对该对象可进行的操作列表） |
| 左键按住拖动 | 移动窗口、选中多个文件、拖放文件到另一个位置 |
| 滚轮 | 向上/下滚动页面 |
| Ctrl+滚轮 | 放大/缩小视图（浏览器、Word、文件管理器等都支持） |

笔记本电脑的触摸板默认模拟这些操作：单指移动（移动鼠标光标）、单指点击（左键单击）、双指点击（右键单击）、双指滑动（滚轮）。

### 键盘布局速览

首先认识按键的大类：字母键（A-Z）、数字键（0-9）、符号键（如 , . / ; '）、功能键（F1-F12）、控制键（Ctrl、Alt、Shift、Win）、方向键、回车键 Enter、退格键 Backspace。不同键盘的布局可能略有差异（比如部分笔记本把 Fn 键放在左下角而非 F1 旁边，有些紧凑键盘没有数字小键盘），但基本分区一致。

标准键盘主要分为几个区域：

- **主键盘区**（中间最大的一块）：字母、数字、标点符号
- **功能键区**（顶部 F1-F12）：不同软件中功能不同。F2（重命名文件）、F5（刷新页面）、F11（全屏）
- **控制键区**（主键盘右侧）：方向键（上下左右移动光标）、Home/End（跳到行首/行尾）、PageUp/PageDown（翻页）
- **数字小键盘**（最右侧）：方便输入大量数字。台式机键盘通常有，部分全尺寸笔记本也有，但许多笔记本没有（或通过 Fn 组合键复用字母键上的数字），需要 NumLock 开启才生效

### 建议记住的快捷键

这十几组快捷键几乎每天都会用到，值得花几分钟记住：

**基本编辑：**

- **Ctrl+C** — 复制（复制选中的内容到剪贴板）
- **Ctrl+X** — 剪切（复制并删除原内容）
- **Ctrl+V** — 粘贴（把剪贴板的内容粘贴出来）
- **Ctrl+Z** — 撤销（撤销上一步操作）
- **Ctrl+Y** — 重做（取消撤销）

**文件管理：**

- **Ctrl+A** — 全选（选中当前窗口里所有内容）
- **Ctrl+S** — 保存（写文档、做图时随时按一下保存，养成习惯）
- **Ctrl+F** — 查找（在当前页面或文档中搜索关键词）

**系统操作：**

- **Win+E** — 打开文件资源管理器（随时管理文件）
- **Win+D** — 显示桌面（最小化所有窗口，再按一次恢复）
- **Win+L** — 锁定电脑（离开座位时按一下，保护隐私，回到锁屏界面需要重新输入密码）
- **Alt+F4** — 关闭当前窗口/程序
- **Alt+Tab** — 切换窗口
- **F2** — 重命名选中的文件或文件夹
- **F5** — 刷新当前页面/文件夹内容
- **Delete** — 删除选中的文件到回收站
- **Shift+Delete** — 永久删除（跳过回收站，谨慎使用）

### 中文输入法

Windows 自带微软拼音输入法，通常已经装好。使用方法：

- **切换输入法**：Win+空格键（切换所有已安装的输入法），或 Shift（仅在中英文之间切换，如果你只用一个中文输入法）
- **中英文切换**：按 Shift 键
- **候选词翻页**：按 =（下一页）、-（上一页），或使用方括号 [ ]
- **输入中文标点**：中文模式下，句号是 。，逗号是 ，，问号是 ？等
- **输入特殊符号**：按 v+数字（比如 v1 会出现各种符号），或右键输入法状态栏选择符号面板
